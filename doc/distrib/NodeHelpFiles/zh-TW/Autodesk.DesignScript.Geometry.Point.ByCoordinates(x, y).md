## 深入資訊
Point.ByCoordinates (x, y) 需要兩個數字輸入，並在 Dynamo 基準平面上產生一個點。在第一個範例中，我們使用兩個滑棒動態控制點在 Z 平面上的 X 和 Y 位置。在第二個範例中，我們使用兩個數字滑棒和一個 Code Block 範圍決定格線的最大範圍和點之間的間距。透過將 Points.ByCoordinates 的交織設定為「交互參考」，我們可以沿 X 軸和 Y 軸建立多維點集。
