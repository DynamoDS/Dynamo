## 深入資訊
NurbsCurve.ByControlPoints 輸入要用作控制點的點清單，並輸出 Nurbs 曲線。此範例使用 Code Block 產生兩個清單，分別對應於一系列點的 X 和 Y 位置。使用這些點繪製 Nurbs 曲線。
