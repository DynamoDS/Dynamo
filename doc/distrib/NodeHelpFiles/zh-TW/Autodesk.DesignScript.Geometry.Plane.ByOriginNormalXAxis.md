## 深入資訊
Plane.ByOriginNormalXAxis 與 Plane.ByOriginNormal 類似，但能讓我們輸入 XAxis 的方向控制平面的方位。如果輸入 xAxis 向量不在 origin 和 normal 輸入所定義的平面中，則會先投影到平面以決定平面 XAxis。在以下範例中，我們使用世界 YAxis 作為法線向量，並使用兩個 Code Block 定義原點和輸入 xAxis。
