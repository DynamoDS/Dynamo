## 深入資訊
Color Range 會在一組輸入顏色之間建立漸層，並允許輸入值清單選取該漸層中的顏色。第一個輸入 colors 是漸層中要使用的顏色清單。第二個輸入 indices 決定漸層中輸入顏色的相對位置。此清單應與顏色清單相對應，每個值都在 0 到 1 的範圍內。確切值不重要，只需要值的相對位置。與最小值對應的顏色將位於漸層左側，與最大值對應的顏色將位於漸層右側。最後的 value 輸入可讓使用者選取漸層中 0 到 1 範圍內的點來輸出。在以下範例中，我們先建立兩種顏色: 紅色和藍色。這些顏色在漸層中的順序由我們使用 Code Block 建立的清單決定。使用第三個 Code Block 建立 0 到 1 之間的數字範圍，該範圍將決定漸層的輸出顏色。沿 x 軸產生一組立方體，最後使用 Display.ByGeometryColor 節點根據漸層將這些立方體著色。
