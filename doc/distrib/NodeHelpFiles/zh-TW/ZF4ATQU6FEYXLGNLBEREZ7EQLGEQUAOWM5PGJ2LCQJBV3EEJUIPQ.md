## 深入資訊
`CoordinateSystem.Rotate (coordinateSystem, plane, degrees)` 會傳回繞平面旋轉給定度數的新 CoordinateSystem。

在以下範例中，座標系統繞設定在 0, 0, 0 插入點的基準 YZ 平面旋轉 135 度。
