## 深入資訊
`List.Chop` 會根據輸入整數長度清單，將給定清單分成較小清單。第一個巢狀清單包含 `lengths` 輸入中第一個數字指定的元素數。第二個巢狀清單包含 lengths 輸入中第二個數字指定的元素數，依此類推。`List.Chop` 會重複 `lengths` 輸入中的最後一個數字，直到輸入清單的所有元素都分完為止。

在以下範例中，我們使用 Code Block 產生一個從 0 到 5，以 1 遞增的數字範圍。此清單有 6 個元素。我們使用第二個 Code Block 建立一個要將第一個清單切斷的長度清單。此清單的第一個數字是 1，`List.Chop` 使用此數字建立一個包含 1 個項目的巢狀清單。第二個數字是 3，因此建立一個包含 3 個項目的巢狀清單。由於沒有再指定更多長度，因此 `List.Chop` 就將剩餘所有項目放入第三個也是最後一個巢狀清單。
