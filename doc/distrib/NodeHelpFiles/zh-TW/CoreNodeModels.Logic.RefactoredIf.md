## 深入資訊
If 的運作方式是當成一個條件控制節點。「test」輸入接受布林值，而「true」和「false」輸入可接受任何資料類型。如果 test 值為「true」，節點將傳回「true」輸入的項目，如果 test 為「false」，節點將傳回「false」輸入的項目。在以下範例中，我們先產生一個 0 到 99 的隨機數字清單。清單中的項目數由一個整數滑棒控制。我們使用 Code Block 的公式「x%a==0」測試是否能被第二個數字滑棒決定的第二個數字整除。這會產生一個布林值清單，對應隨機清單中的項目是否可被第二個整數滑棒決定的數字整除。使用此布林值清單作為 If 節點的「test」輸入。我們使用預設圓球作為「true」輸入，使用預設立方體作為「false」輸入。If 節點的結果是一個包含圓球或立方體的清單。最後，我們使用 Translate 節點將幾何圖形清單散開。

If 會對「好像設為最短」的所有節點重複進行。您可以在附加的範例中發現原因，在對公式節點作用「最長」卻看到通過「短的」條件分支的結果時尤其明顯。這些改變也是為了在使用單一布林輸入或布林清單時能夠預測行為。
