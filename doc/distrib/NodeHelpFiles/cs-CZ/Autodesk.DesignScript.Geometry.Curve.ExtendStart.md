## Podrobnosti
Uzel Extend Start prodlouží vstupní křivku o zadanou vstupní vzdálenost počínaje počátečním bodem křivky. V níže uvedeném příkladu nejprve vytvoříme křivku Nurbs pomocí uzlu ByControlPoints, přičemž jako vstup se použije sada náhodně generovaných bodů. Číselný posuvník nám umožňuje řídit vzdálenost prodloužení.
