## Podrobnosti
Uzel Start Point vrátí počáteční bod vstupní křivky. V níže uvedeném příkladu nejprve vytvoříme křivku Nurbs pomocí uzlu ByControlPoints, přičemž jako vstup se použije sada náhodně generovaných bodů. Počáteční bod této křivky můžeme najít pomocí uzlu StartPoint.
