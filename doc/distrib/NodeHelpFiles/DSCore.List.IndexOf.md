## In Depth
`List.IndexOf` returns the index location of a given element in a list. If the given element isnâ€™t found in the list or its data type does not match the list element (for example, the input is a double and the list element is a string), `List.IndexOf` returns -1. 

Note that this node returns only the first index, even if the element appears multiple times on the list. To return multiple indices, use `List.AllIndicesOf` instead.
