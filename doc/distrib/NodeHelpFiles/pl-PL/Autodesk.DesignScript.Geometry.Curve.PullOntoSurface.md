## Informacje szczegółowe
Węzeł Pull Onto Surface tworzy nową krzywą przez rzutowanie krzywej wejściowej (curve) na powierzchnię wejściową (surface), używając wektorów normalnych tej powierzchni jako kierunków rzutowania. W poniższym przykładzie najpierw tworzymy powierzchnię za pomocą węzła Surface.BySweep, który używa krzywych wygenerowanych zgodnie z krzywą sinusoidy. Jest to powierzchnia bazowa dla węzła PullOntoSurface. Rolę krzywej pełni okrąg tworzony za pomocą węzła Code Block określającego współrzędne punktu środkowego oraz suwaka Number Slider sterującego promieniem okręgu. Wynikiem jest rzut okręgu na powierzchnię.
