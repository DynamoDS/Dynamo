## Informacje szczegółowe
Węzeł `CoordinateSystem.Rotate (coordinateSystem, plane, degrees)`zwraca nowy układ CoordinateSystem obrócony wokół płaszczyzny (plane) o podaną liczbę stopni (degrees).

W poniższym przykładzie układ współrzędnych zostaje obrócony wokół płaszczyzny bazowej YZ ustawionej w punkcie wstawienia 0, 0, 0 o 135 stopni.
