## Informacje szczegółowe
Węzeł Nurbs Curve By Control Points pobiera listę punktów (points), które mają być punktami sterującymi, i zwraca krzywą Nurbs. W tym przykładzie za pomocą węzła Code Block generowane są dwie listy odpowiadające położeniom X i Y serii punktów. Punkty są używane do narysowania krzywej Nurbs.
