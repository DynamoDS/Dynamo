## Informacje szczegółowe
Węzeł `>` jest operatorem „większe niż”. Jeśli wartość wejściowa `x` jest większa niż wartość wejściowa `y`, zwraca wartość True (Prawda). Jeśli wartość `x` jest mniejsza niż `y` lub jeśli obie wartości są równe, zwraca wartość False (Fałsz). W poniższym przykładzie węzeł `>` jest używany do określenia, czy wartość wejściowa `x` jest większa niż wartość `y`. Do sterowania danymi wejściowymi operatora `>` służą dwa węzły Number Slider.
