## Informacje szczegółowe
Węzeł `<` jest operatorem „mniejsze niż”. Jeśli wartość wejściowa `x` jest mniejsza niż wartość wejściowa `y`, węzeł zwraca wartość True (Prawda). Jeśli wartość `x` jest większa niż `y` lub jeśli te wartości są równe, ten operator zwraca wartość False (Fałsz). Ten element istnieje w słowniku 1.x; edytowana nazwa wykresu: LessThan.dyn Y

W poniższym przykładzie węzeł `<` jest używany do określenia, czy wartość wejściowa `x` jest mniejsza niż wartość wejściowa `y`. Do sterowania danymi wejściowymi operatora `<` są używane dwa węzły Number Slider.
