## In profondità
Il nodo `||' è l'operatore Or. Utilizza due valori booleani come input e restituisce True se uno o entrambi i valori sono True. Se entrambi i valori sono False, restituisce False. Nell'esempio seguente, tre nodi `||` vengono utilizzati per determinare se almeno uno dei valori booleani di input è True. Il primo nodo ha un input True e False, pertanto restituisce True. Il secondo nodo ha due input True, pertanto restituisce True. Il terzo nodo ha due nodi False, quindi restituisce False.
