## In profondità
Il nodo `<` è l'operatore Less Than. Se l'input `x` è minore dell'input `y`, restituisce True. Se `x` è maggiore di `y` o se i due valori sono uguali, questo operatore restituisce False. Esiste nel dizionario 1.x; nome del grafico modificato: LessThan.dyn Y.

Nell'esempio seguente, viene utilizzato un nodo `<` per determinare se l'input `x` è minore dell'input `y`. Vengono utilizzati due dispositivi di scorrimento numerici per controllare gli input dell'operatore `<`.
