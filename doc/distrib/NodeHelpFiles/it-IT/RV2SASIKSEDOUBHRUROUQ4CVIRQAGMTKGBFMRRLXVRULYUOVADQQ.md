## In profondità
Crea una NurbsSurface che soddisfa una raccolta di caratteristiche di superficie diverse. È il metodo più avanzato per l'adattamento di superfici. La superficie risultante passa attraverso tutti i punti. Il numero di tangenti deve essere uguale al numero di punti nella direzione corrispondente. La superficie risultante è di grado 3 in entrambe le direzioni U e V. Le derivate degli angoli devono essere di secondo ordine (dP/dUdV) e devono essere fornite nel seguente ordine [ lowU, lowV ], [ highU, lowV ], [ lowU, highV ], [ highU, highV ].
