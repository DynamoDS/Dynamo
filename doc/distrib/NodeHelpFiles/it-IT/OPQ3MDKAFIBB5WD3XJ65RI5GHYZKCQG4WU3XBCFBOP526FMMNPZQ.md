## In profondità
Il nodo Rectangle.ByCornerPoints utilizza quattro punti di input e crea un rettangolo da essi. I punti di input devono riflettere la forma di un rettangolo. In questo esempio, vengono utilizzati due Number Slider per creare Point.ByCoordinates in corrispondenza dei vertici del rettangolo. Un nodo Rectangle.ByCornerPoints simile ma distinto utilizza un elenco di punti come input anziché ogni input separatamente. Entrambi i nodi vengono mostrati in questo esempio.
