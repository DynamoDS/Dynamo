## In profondità
Il nodo `>` è l'operatore Greater Than. Se l'input `x` è maggiore dell'input `y`, restituisce True. Se `x` è minore di `y` o se i due valori sono uguali, restituisce False. Nell'esempio seguente, viene utilizzato un nodo `>` per determinare se l'input `x` è maggiore dell'input `y`. Vengono utilizzati due dispositivi di scorrimento numerici per controllare gli input dell'operatore `>`.
