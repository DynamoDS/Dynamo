## In profondità
`Geometry.ExportToSAT` esporta la geometria da Dynamo in un tipo di file SAT. Questo nodo è senza unità. Per specificare un'unità, utilizzare `ExportToSAT`.

Nell'esempio seguente, una PolySurface proveniente dall'intersezione di tre sfere può essere esportata in un file SAT una volta selezionato un percorso di file.
