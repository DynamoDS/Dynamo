## In profondità
`Mesh.ImportFile` utilizza il nome di un file come input dal percorso del file definito. Il file può essere dei seguenti tipi:
- .obj - Wavefront OBJ
- .mix - Meshmixer
- .stl - STL
- .ply - Polygon File Format
