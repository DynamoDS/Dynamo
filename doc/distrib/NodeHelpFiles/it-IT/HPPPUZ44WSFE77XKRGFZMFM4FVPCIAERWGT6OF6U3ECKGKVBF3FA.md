## In profondità
Nell'esempio seguente, una superficie T-Spline semplice viene convertita in formato TSM stringa con un nodo `TSplineSurface.SerializeAsTSM` ed esportata come file di testo. Questo file può essere successivamente ricostruito utilizzando un nodo `TSplineSurface.DeserializeFromTSM`.
