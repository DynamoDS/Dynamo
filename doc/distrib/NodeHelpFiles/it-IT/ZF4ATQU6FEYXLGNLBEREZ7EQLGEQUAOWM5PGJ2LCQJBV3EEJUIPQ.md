## In profondit√†
`CoordinateSystem.Rotate (coordinateSystem, plane, degrees)` restituisce un nuovo CoordinateSystem ruotato attorno ad un piano in base al numero specificato di gradi.

Nell'esempio seguente, un sistema di coordinate viene ruotato attorno ad un piano YZ di base impostato in corrispondenza del punto di inserimento 0, 0, 0 di 135 gradi.
