## Em profundidade
O nó `<` é o operador Menor que. Se a entrada `x` for menor que a entrada `y`, será retornado True. Se `x` for maior que `y` ou se os dois valores forem iguais, esse operador retornará False. Existe no dicionário 1.x; nome do Gráfico editado: LessThan.dyn Y

No exemplo abaixo, um nó `<` é usado para determinar se a entrada `x` é menor que a entrada `y`. Usamos dois controles deslizantes de números para controlar as entradas para o operador `<`.
