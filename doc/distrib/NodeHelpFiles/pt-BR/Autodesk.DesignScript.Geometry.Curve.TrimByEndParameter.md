## Em profundidade
Trim By End Parameter removerá o final de uma curva de entrada aparando a curva no parâmetro especificado. No exemplo abaixo, primeiro criaremos uma curva Nurbs usando um nó ByControlPoints, com um conjunto de pontos gerados aleatoriamente como entrada. Um controle deslizante de número definido para o intervalo de 0 a 1 é usado para controlar a entrada de parâmetro para um nó TrimByEndParameter.
