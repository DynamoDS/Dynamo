## Em profundidade
O nó `>=` é o operador Maior que ou Igual a. Se a entrada `x` for maior que a entrada `y` ou se os dois valores forem iguais, existirá no dicionário 1.x; nome do Gráfico editado: GreaterThanOrEqualTo.dyn Y

será retornado True. Se `x` for menor que `y`, esse operador retornará False. No exemplo abaixo, um nó `>=` é usado para determinar se a entrada `x` é maior ou igual à entrada `y`. Usamos dois controles deslizantes de números para controlar as entradas para o operador `>=`.
