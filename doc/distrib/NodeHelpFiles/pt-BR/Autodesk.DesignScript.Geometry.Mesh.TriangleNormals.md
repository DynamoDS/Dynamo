## Em profundidade
No exemplo abaixo, 'Mesh.TriangleNormals' é usado em uma malha importada e o nó 'Line.ByStartPointDirectionLength' é usado para visualizar os vetores normais. O nó 'Mesh.TriangleCentroids' é usado para auxiliar o posicionamento das linhas no centro de cada triângulo.
