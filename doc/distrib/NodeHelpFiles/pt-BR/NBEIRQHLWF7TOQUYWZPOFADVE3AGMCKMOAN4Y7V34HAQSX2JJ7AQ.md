## Em profundidade
O nó `*` é o operador de multiplicação. Ele retorna o produto dos dois números de entrada. No exemplo abaixo, um nó `*` é usado para multiplicar x vezes y. Usamos dois controles deslizantes de números para controlar as entradas para o operador `*`.
