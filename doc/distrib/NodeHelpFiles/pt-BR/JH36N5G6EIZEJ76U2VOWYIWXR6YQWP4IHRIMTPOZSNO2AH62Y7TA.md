## Em profundidade
`Geometry.ExportToSAT` exporta geometria do Dynamo para um tipo de arquivo SAT. Esse nó não tem unidade; para especificar uma unidade, use `ExportToSAT`.

No exemplo abaixo, é possível exportar uma PolySurface da intersecção de três esferas para um arquivo SAT assim que um caminho de arquivo é selecionado.
