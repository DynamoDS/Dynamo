## Em profundidade
O nó `>` é o operador Maior que. Se a entrada `x` for maior que a entrada `y`, será retornado True. Se `x` for menor que `y` ou se os dois valores forem iguais, será retornado False. No exemplo abaixo, um nó `>` é usado para determinar se a entrada `x` é maior que a entrada `y`. Usamos dois controles deslizantes de números para controlar as entradas para o operador `>`.
