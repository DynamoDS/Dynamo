## Im Detail
Parameter At Segment Length gibt den Parameter eines Punkts zurück, der um eine bestimmte Länge entlang der Kurve vom Startpunkt entfernt liegt. Im folgenden Beispiel wird zunächst eine NURBS-Kurve mit einem ByControlPoints-Block erstellt, wobei eine Reihe zufällig generierter Punkte als Eingabe verwendet wird. Ein Zahlen-Schieberegler wird verwendet, um die Segmentlänge zu steuern, an der ein Parameter zu finden ist. Wenn die eingegebene Segmentlänge länger ist als die Kurve, gibt dieser Block den Parameterwert des Endpunkts der Kurve zurück.
