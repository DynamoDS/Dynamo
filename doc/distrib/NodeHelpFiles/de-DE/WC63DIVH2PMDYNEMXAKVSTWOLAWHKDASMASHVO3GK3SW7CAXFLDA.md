## Im Detail
`CoordinateSystem.Translate (coordinateSystem, xTranslation, yTranslation, zTranslation)` gibt ein neues CoordinateSystem-Objekt zurück. Dessen Position basiert auf den Verschiebungsstrecken in X-, Y- und Z-Richtung.

Im folgenden Beispiel wird anhand von Verschiebungsstrecken von 3.8 auf der X-Achse, 2.7 auf der Y-Achse und 0.8 auf der Z-Achse ein neues CoordinateSystem-Objekt zurückgegeben. Das neue CoordinateSystem-Objekt wird mit einem darauf erstellten Zylinder identifiziert.
