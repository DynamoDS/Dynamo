## Im Detail
Im folgenden Beispiel werden zwei Scheitelpunkte einer T-Spline-Quaderoberfläche mithilfe des Blocks `TSplineSurface.WeldVertices` verbunden. Eine neue Position für die verschweißten Scheitelpunkte wird als `newPosition`-Eingabe bereitgestellt. Darüber hinaus steuert die Eingabe `keepSubdCreases`, ob die Kanten entlang der Schweißnaht geknickt werden.
