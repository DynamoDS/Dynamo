## Im Detail
`Geometry.Rotate (origin, axis, degrees)` dreht eine eingegebene Geometrie um einen definierten Gradwert um eine Basisebene.

Im folgenden Beispiel wird ein Kegel mit einem Zahlen-Schieberegler um die XY-Ebene gedreht.
