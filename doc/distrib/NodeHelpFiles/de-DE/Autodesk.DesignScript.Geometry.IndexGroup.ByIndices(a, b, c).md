## Im Detail
ByIndices (a, b, c) (a, b, c) gibt eine IndexGroup mit drei Indizes zurück. Im folgenden Beispiel werden drei Indizes für ein dreiseitiges Netz definiert, das mit Mesh.ByPointsFacesIndices erstellt wurde.
