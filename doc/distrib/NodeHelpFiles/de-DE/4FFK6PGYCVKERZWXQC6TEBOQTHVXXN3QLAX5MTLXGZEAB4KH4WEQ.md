## Im Detail
Wandelt eine Oberfläche durch Extrusion in Richtung der Oberflächennormalen in einen Volumenkörper um. Wenn der Parameter both_sides den Wert True hat, wird die Oberfläche auf beiden Seiten verstärkt.
