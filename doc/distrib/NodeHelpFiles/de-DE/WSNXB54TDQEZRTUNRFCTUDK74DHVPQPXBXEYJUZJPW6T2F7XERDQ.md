## Im Detail
Im folgenden Beispiel wird `TSplineTopology.VertexByIndex` verwendet, um eine Liste von Scheitelpunkten auf einer planaren T-Spline-Oberfläche anzuzeigen.
Die Oberfläche stellt extrudierte Scheitelpunkte dar, von denen einige dann mithilfe des Blocks `TSplineSurface.FlattenVertices(vertices)` vereinfacht werden.
Die Blöcke `TSplineVertex.UVNFrame` und `TSplineUVNFrame.Position` werden verwendet, um die zu vereinfachenden Scheitelpunkte hervorzuheben.
