## Im Detail
Im folgenden Beispiel wird eine einzelne Fläche einer Quader-T-Spline-Oberfläche mit dem Block `TSplineTopology.FaceByIndex` ausgewählt. Die Fläche wird dann entlang einer bereitgestellten `curve`-Eingabe mit dem Block `TSplineSurface.ExtrudeFacesAlongCurve` extrudiert. Das Ergebnis ist eine Form mit vier extrudierten Flächen - dies wird durch die auf die Form angewendete anfängliche Symmetrie erklärt.
