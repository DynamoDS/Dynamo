## Im Detail
`CoordinateSystem.Scale (coordinateSystem, amount)` gibt ein neues CoordinateSystem-Objekt zurück, das um einen Faktor skaliert ist.

Im folgenden Beispiel wird ein CoordinateSystem-Objekt am Ursprung mit einem Faktor von 3.5 skaliert. Das resultierende CoordinateSystem-Objekt besitzt Vektorlängen und Skalierfaktoren von 3.5. Passen Sie den Schieberegler für verschiedene Skalierfaktoren an.
