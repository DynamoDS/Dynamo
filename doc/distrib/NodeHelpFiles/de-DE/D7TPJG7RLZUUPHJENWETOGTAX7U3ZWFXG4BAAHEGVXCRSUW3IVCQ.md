## Im Detail
`CoordinateSystem.Translate (coordinateSystem, direction, distance)` gibt ein neues CoordinateSystem-Objekt zurück, das um einen Eingabevektor und einen Abstand verschoben wurde.

Im folgenden Beispiel wird ein CoordinateSystem-Objekt am Ursprung um den Eingabevektor (3.1, 4.1, 0) und einen Abstand von 3 in eine Richtung verschoben. Die Eingabe `distance` überschreibt die Länge der Eingabe `vector`. Das neue CoordinateSystem-Objekt wird durch einen darauf erstellten Zylinder identifiziert.
