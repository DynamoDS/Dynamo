## Im Detail
Der Block `<` ist der Kleiner-als-Operator. Wenn die Eingabe für `x` kleiner ist als die Eingabe für `y`, wird True zurückgegeben. Wenn `x` größer als `y` ist oder die beiden Werte gleich sind, gibt dieser Operator False zurück. Ist im 1.x-Wörterbuch enthalten; Name des bearbeiteten Diagramms: LessThan.dyn Y

Im folgenden Beispiel wird mit einem `<`-Block bestimmt, ob die Eingabe für `x` kleiner als die Eingabe für `y` ist. Die Eingaben für den Operator `<` werden mit zwei Zahlen-Schiebereglern gesteuert.
