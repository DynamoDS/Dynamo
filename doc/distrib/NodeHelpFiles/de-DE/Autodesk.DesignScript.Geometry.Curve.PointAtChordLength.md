## Im Detail
Point At Chord Length gibt den Punkt zurück, der um eine angegebene Sehnenlänge von einem angegebenen Startparameter entfernt liegt. Im folgenden Beispiel erstellen Sie zunächst eine NURBS-Kurve mit einem ByControlPoints-Block, wobei eine Reihe zufällig generierter Punkte als Eingabe verwendet wird. Der geradlinige Sehnenabstand, an dem der Parameter zu finden ist, wird mit einem Zahlen-Schieberegler gesteuert. Ein zweiter auf den Bereich 0 bis 1 eingestellter Zahlen-Schieberegler wird verwendet, um den Startpunkt entlang der Kurve zu steuern, von der aus die Sehnenlänge gemessen wird. Schließlich wird ein boolescher Schalter verwendet, um zu bestimmen, von welcher Richtung aus die Sehnenlänge gemessen wird.
