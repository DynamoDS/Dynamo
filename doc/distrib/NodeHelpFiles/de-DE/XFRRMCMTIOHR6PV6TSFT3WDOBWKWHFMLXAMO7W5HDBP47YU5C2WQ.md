## Im Detail
`Geometry.Rotate (basePlane, degrees)` dreht eine eingegebene Geometrie um einen angegebenen Gradwert um einen Ursprungspunkt und einen Achsenvektor.

Im folgenden Beispiel wird ein Kegel um seinen Endpunkt um den Y-Achsenvektor gedreht.
