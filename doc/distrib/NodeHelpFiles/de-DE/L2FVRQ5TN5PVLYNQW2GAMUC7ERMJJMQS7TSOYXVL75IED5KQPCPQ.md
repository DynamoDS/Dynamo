## Im Detail
`Solid.ByLoft (crossSections, guideCurves)` ermöglicht es, eine Liste mit Führungskurven einzugeben, um verschiedene Profile entlang des Umfangs eines erhobenen Volumenkörpers zu erstellen.

Im folgenden Beispiel werden zwei versetzte Rechtecke als `crossSections` verwendet. Um den Umfang werden zwei verschiedene Führungskurven erstellt, eine basierend auf einer Kosinusfunktion und die andere basierend auf einer einfachen Linie.
