## Im Detail
`Vector.Scale (vector, scale_factor)` gibt einen neuen Vektor zurück, der um einen Faktor skaliert wurde.

Im folgenden Beispiel wird ein Vektor der Länge 1.732 von einem Faktor von 7.2 auf 12.471 skaliert.
