## Im Detail
Mit `CoordinateSystem.Scale (coordinateSystem, plane, xamount, yamount, zamount)` wird ein Koordinatensystem ungleichmäßig um eine bestimmte Ebene skaliert. Das Koordinatensystem wird an seiner Position skaliert, indem der Ursprung des Basiskoordinatensystems für die Ebeneneingabe verwendet wird.
