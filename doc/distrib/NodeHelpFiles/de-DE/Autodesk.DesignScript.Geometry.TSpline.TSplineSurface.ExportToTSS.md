## Im Detail
In the example below, a T-Spline surface is created using `TSplineSurface.ByRevolve`, with a NURBS curve as a profile.
Bei Angabe eines Dateipfads wird die Oberfläche dann über den Block `TSplineSurface.ExportToTSS` in eine T-Spline-Szenendatei exportiert.
