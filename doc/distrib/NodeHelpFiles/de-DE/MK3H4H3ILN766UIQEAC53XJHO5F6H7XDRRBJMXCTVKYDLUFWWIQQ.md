## Im Detail
Der `>`-Block ist der Größer-als-Operator. Wenn die Eingabe für `x` größer ist als die Eingabe für `y`, wird True zurückgegeben. Wenn `x` kleiner als `y` ist oder die beiden Werte gleich sind, wird False zurückgegeben. Im folgenden Beispiel wird mithilfe eines `>`-Blocks ermittelt, ob die Eingabe für `x` größer als die Eingabe `für y` ist. Die Eingaben für den `>`-Operator werden mit zwei Zahlen-Schiebereglern gesteuert.
