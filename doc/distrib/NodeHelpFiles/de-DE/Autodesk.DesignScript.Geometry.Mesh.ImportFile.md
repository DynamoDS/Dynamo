## Im Detail
`Mesh.ImportFile` verwendet den Namen einer Datei als Eingabe aus dem definierten Dateipfad. Es kann sich um folgende Dateitypen handeln:
- .obj - Wavefront OBJ-Datei
- .mix - Meshmixer-Datei
- .stl - STL-Datei
- .ply - Polygon File Format
