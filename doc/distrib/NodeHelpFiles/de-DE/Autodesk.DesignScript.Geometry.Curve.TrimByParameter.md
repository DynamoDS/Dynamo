## Im Detail
Mit Trim By Parameter werden Start und Ende einer Eingabekurve durch Stutzen der Kurve an den angegebenen Parametern entfernt, und der resultierende mittlere Abschnitt der Kurve wird zurückgegeben. Im folgenden Beispiel erstellen Sie zunächst eine NURBS-Kurve mit einem ByControlPoints-Block, wobei eine Reihe zufällig generierter Punkte als Eingabe verwendet wird. Die Eingaben für den Start- und Endparameter für einen TrimByParameter-Block werden mit zwei auf den Bereich 0 bis 1 eingestellten Zahlen-Schiebereglern gesteuert.
