## En detalle:
`Point.Z` devuelve el valor Z del espacio universal de un punto de entrada.

En el ejemplo siguiente, utilizamos `Point.BySphericalCoordinates` para crear un punto y, a continuaci√≥n, se utiliza `Point.Z` para devolver su coordenada Z.
