## En detalle:
ByCoordinates (x, y, z, normalized) devolverá un vector a partir de los valores de coordenadas y un conmutador booleano normalizado. En el siguiente ejemplo, se devuelve un vector con una longitud de 2.83 que se puede normalizar a 1. El vector se representa como una línea para mostrar el cambio de longitud al activar o desactivar la normalización.
