## En detalle:
En el ejemplo siguiente, se utiliza `TSplineTopology.VertexByIndex` para mostrar una lista de vértices en una superficie de T-Spline plana.
La superficie presenta vértices extruidos; algunos de ellos se aplanan mediante el nodo `TSplineSurface.FlattenVertices(vertices)`.
Los nodos `TSplineVertex.UVNFrame` y `TSplineUVNFrame.Position` se utilizan para resaltar los vértices que se van a aplanar.
