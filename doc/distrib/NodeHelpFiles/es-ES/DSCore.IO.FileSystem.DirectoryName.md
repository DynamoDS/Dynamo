## En detalle:
DirectoryName devolverá el directorio de una ruta de acceso de directorio o archivo. En el siguiente ejemplo, se usa un nodo Directory.Path para extraer el directorio de una ruta de acceso de archivo. Además de FileName y Extension, esto permite separar una ruta de archivo en componentes independientes.
