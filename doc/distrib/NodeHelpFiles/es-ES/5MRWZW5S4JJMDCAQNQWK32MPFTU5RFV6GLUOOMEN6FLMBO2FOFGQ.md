## En detalle:
En el ejemplo siguiente, se selecciona un conjunto de aristas de borde de una superficie de T-Spline y se utiliza como entrada para el nodo `TSplineSurface.ExtrudeEdges`. El resultado se traslada a un lado para obtener una mejor vista preliminar.
