## Description approfondie
Watch affiche la sortie d'un nœud sous forme de liste. Dans l'exemple ci-dessous, le contenu d'un fichier Excel est renvoyé sous forme de liste et affiché dans un nœud Watch. Watch est utile pour comprendre les structures de liste, les types de géométrie, etc. renvoyés par différents nœuds.
