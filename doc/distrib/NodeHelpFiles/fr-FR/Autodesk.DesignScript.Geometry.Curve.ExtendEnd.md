## Description approfondie
ExtendEnd prolonge une courbe d'entrée selon une distance donnée, en partant du point d'arrivée de la courbe. Dans l'exemple ci-dessous, nous créons d'abord une courbe Nurbs à l'aide d'un nœud ByControlPoints, avec un ensemble de points générés de façon aléatoire comme entrée. Un curseur numérique permet de contrôler la distance de l'extension.
