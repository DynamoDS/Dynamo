## Description approfondie
PlaneAtParameter renvoie un plan aligné sur une courbe selon un paramètre spécifié. Le vecteur normal du plan obtenu correspond à la tangente de la courbe. Le paramétrage d'une courbe est mesuré dans l'intervalle de 0 à 1, 0 représentant le début de la courbe et 1 la fin de la courbe. Dans l'exemple ci-dessous, nous créons d'abord une courbe Nurbs à l'aide d'un nœud ByControlPoints, avec un ensemble de points générés de façon aléatoire comme entrée. Un curseur numérique défini sur un intervalle de 0 à 1 est utilisé pour contrôler l'entrée de paramètre pour un nœud PlaneAtParameter.
