## Description approfondie
EndPoint renvoie le point d'arrivée d'une courbe d'entrée. Dans l'exemple ci-dessous, nous créons d'abord une courbe Nurbs à l'aide d'un nœud ByControlPoints, avec un ensemble de points générés de façon aléatoire comme entrée. Nous pouvons trouver le point d'arrivée de cette courbe à l'aide d'un nœud EndPoint.
