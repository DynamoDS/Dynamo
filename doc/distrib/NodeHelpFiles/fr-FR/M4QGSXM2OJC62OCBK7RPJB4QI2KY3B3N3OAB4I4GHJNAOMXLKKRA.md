## Description approfondie
`Geometry.Translate (direction, distance)` déplace la géométrie d'entrée dans une direction de vecteur donnée. L'entrée `distance` contrôle la distance de déplacement.

Dans l'exemple ci-dessous, un cône est translaté le long de l'axe X. Sa distance par rapport au cône d'origine est contrôlée par un curseur numérique.

Dans l'exemple ci-dessous, un cuboïde est converti en un fichier Solid Def. Copiez le contenu du noeud Watch pour utiliser la représentation de chaîne Solid Def dans un autre graphique.
