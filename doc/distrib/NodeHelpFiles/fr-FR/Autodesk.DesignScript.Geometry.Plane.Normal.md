## Description approfondie
Plane Normal recherche le vecteur normal d'un plan d'entrée. Cela équivaut à prendre le produit cartésien du plan Axe X et Axe Y. Dans l'exemple ci-dessous, nous utilisons d'abord un ensemble de points aléatoires pour créer un plan ByBestFitThroughPoints. Nous pouvons ensuite trouver le vecteur normal de ce plan.
