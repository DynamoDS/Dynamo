## Description approfondie
Le noeud "<" est l'opérateur Inférieur à. Si l'entrée "x" est inférieure à l'entrée "y", elle renvoie la valeur True. Si "x" est supérieur à "y" ou si les deux valeurs sont égales, cet opérateur renvoie la valeur False. Existe dans le dictionnaire 1.x; nom de graphique modifié: LessThan.dyn Y

Dans l'exemple ci-dessous, un noeud "<" est utilisé pour déterminer si l'entrée "x" est inférieure à l'entrée "y". Nous utilisons deux curseurs numériques pour contrôler les entrées de l'opérateur "<".
