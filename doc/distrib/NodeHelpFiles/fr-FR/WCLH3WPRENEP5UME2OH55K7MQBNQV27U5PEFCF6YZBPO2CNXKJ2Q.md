## Description approfondie
ApproximateWithArcAndLineSegments prend une courbe en entrée et renvoie une liste de courbes de ligne droite et d'arc qui se rapprochent de la courbe d'origine. Dans l'exemple ci-dessous, nous créons d'abord une courbe Nurbs à l'aide d'un nœud ByControlPoints, avec un ensemble de points générés de façon aléatoire comme entrée. Cette courbe est ensuite utilisée comme entrée pour un nœud ApproximateWithArcAndLineSegments.
