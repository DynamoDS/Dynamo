## Description approfondie
`CoordinateSystem.Translate (coordinateSystem, direction)` renvoie un nouveau CoordinateSystem converti par un vecteur d'entrée (direction).

Dans l'exemple ci-dessous, un CoordinateSystem à l'origine est converti à une certaine distance et direction, comme défini par le vecteur d'entrée de (3.8, 2.7, 0). Le nouveau CoordinateSystem est identifié par un cylindre créé sur celui-ci.
