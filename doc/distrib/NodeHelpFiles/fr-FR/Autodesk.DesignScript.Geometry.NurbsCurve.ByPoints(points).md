## Description approfondie
NurbsCurve ByPoints permet d'entrer une liste de points par lesquels tracer une courbe Nurbs. Cet exemple montre comment générer six points aléatoires sur le plan XY du SCG puis les connecter dans l'ordre selon les directions X et Y.
