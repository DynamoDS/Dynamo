## Description approfondie
ParameterAtSegmentLength renvoie le paramètre d'un point qui est une longueur donnée le long de la courbe à partir du point de départ. Dans l'exemple ci-dessous, nous créons d'abord une courbe Nurbs à l'aide d'un nœud ByControlPoints, avec un ensemble de points générés de façon aléatoire comme entrée. Un curseur numérique est utilisé pour contrôler la longueur du segment pour lequel trouver un paramètre. Si la longueur du segment d'entrée est plus longue que la courbe, ce nœud renvoie la valeur de point d'arrivée de la courbe.
