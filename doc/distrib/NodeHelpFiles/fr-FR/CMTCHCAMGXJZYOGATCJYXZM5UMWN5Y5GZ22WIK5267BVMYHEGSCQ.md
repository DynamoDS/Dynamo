## Description approfondie
Le nœud Rectangle ByWidthLength crée un rectangle sur le plan de base Z avec une longueur et une largeur spécifiées. Le centre du rectangle est situé au niveau de l'origine de la grille. Dans cet exemple, nous utilisons deux curseurs numériques pour définir la forme du rectangle.
