## Description approfondie
`PolyCurve.ByJoinedCurves` crée des courbes jointes à partir d'un jeu de courbes avec des sommets correspondants. Vous pouvez définir la taille autorisée de l'écart entre les courbes à relier à l'aide de l'entrée `joinTolerance`.

Dans l'exemple ci-dessous, un `Polygon.RegularPolygon` est décomposé en une liste de courbes qui sont ensuite reliées en une PolyCurve.
