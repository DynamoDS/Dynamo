## Description approfondie
IsPlanar renvoie une valeur booléenne qui indique si la courbe d'entrée est plane ou non. Dans l'exemple ci-dessous, nous créons d'abord une courbe Nurbs à l'aide d'un nœud ByControlPoints, avec un ensemble de points générés de façon aléatoire comme entrée. Nous pouvons ensuite tester si cette courbe est plane à l'aide d'un nœud IsPlanar.
