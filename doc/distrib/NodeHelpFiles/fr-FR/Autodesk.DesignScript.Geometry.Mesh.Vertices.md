## Détails
'Mesh.Vertices' renvoie une liste de sommets uniques d'un maillage sous forme de liste de points. La structure de la liste de sortie est compatible avec les index de sortie du noeud 'Mesh.VertexIndicesByTri' et les deux peuvent être utilisés ensemble pour reconstruire ou modifier le maillage.
