## Description approfondie
StartPoint renvoie le point de départ d'une courbe d'entrée. Dans l'exemple ci-dessous, nous créons d'abord une courbe Nurbs à l'aide d'un nœud ByControlPoints, avec un ensemble de points générés de façon aléatoire comme entrée. Nous pouvons trouver le point de départ de cette courbe à l'aide d'un nœud StartPoint.
