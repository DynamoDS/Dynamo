## Description approfondie
PullOntoPlane permet de créer une nouvelle courbe en projetant une courbe d'entrée sur un plan d'entrée, en utilisant la normale du plan comme direction de projection. Dans l'exemple ci-dessous, nous créons d'abord une courbe Nurbs à l'aide d'un nœud ByControlPoints, avec un ensemble de points générés de façon aléatoire comme entrée. Nous utilisons le plan XY du SCG comme plan de projection dans un nœud PullOntoPlan. Le résultat obtenu est une courbe plane sur le plan XY.
