## Description approfondie
Dans l'exemple ci-dessous, une face simple d'une surface de T-Spline de boîte est sélectionnée à l'aide du noeud `TSplineTopology.FaceByIndex`. La face est ensuite extrudée en suivant une `courbe` fournie avec le noeud `TSplineSurface.ExtrudeFacesAlongCurve`. Le résultat est une forme avec quatre faces extrudées, ce qui s'explique par la symétrie initiale appliquée à la forme.
