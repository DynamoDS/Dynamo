## Description approfondie
Dans l'exemple ci-dessous, `TSplineTopology.VertexByIndex` est utilisé pour afficher une liste de sommets sur une surface de T-Spline plane.
La surface présente des sommets extrudés, dont certains sont ensuite aplatis à l'aide du noeud `TSplineSurface.FlattenVertices(vertices)`.
Les noeuds `TSplineVertex.UVNFrame` et `TSplineUVNFrame.Position` sont utilisés pour mettre en surbrillance les sommets aplatis.
