## Description approfondie
Le noeud ">" est l'opérateur Supérieur à. Si l'entrée "x" est supérieure à l'entrée "y", il renvoie la valeur True. Si "x" est inférieur à "y" ou si les deux valeurs sont égales, il renvoie la valeur False. Dans l'exemple ci-dessous, un noeud ">"  est utilisé pour déterminer si l'entrée "x" est supérieure à l'entrée "y". Deux curseurs numériques permettent de contrôler les entrées de l'opérateur ">".
