## Description approfondie
`PolySurface.LocateSurfacesByPoint` renvoie la première surface sécante dans la direction vers l'avant, comme définie par un vecteur.

Dans l'exemple ci-dessous, la surface renvoyée est coupée par un vecteur dont l'origine est le point. Le vecteur est représenté par une ligne.
