## Description approfondie
TrimByStartParameter supprime le début d'une courbe d'entrée en ajustant la courbe au paramètre spécifié. Dans l'exemple ci-dessous, nous créons d'abord une courbe Nurbs à l'aide d'un nœud ByControlPoints, avec un ensemble de points générés de façon aléatoire comme entrée. Un curseur numérique défini sur l'intervalle de 0 à 1 est utilisé pour contrôler l'entrée de paramètre pour un nœud TrimByStartParameter.
