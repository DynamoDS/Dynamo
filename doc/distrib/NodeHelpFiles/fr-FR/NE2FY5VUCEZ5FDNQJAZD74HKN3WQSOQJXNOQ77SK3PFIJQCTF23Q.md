## Description approfondie
`CoordinateSystem.Scale (coordinateSystem, plane, xamount, yamount, zamount)` met à l'échelle un système de coordonnées de façon non uniforme autour d'un plan donné. Le système de coordonnées est mis à l'échelle sur place en utilisant l'origine du système de coordonnées de base pour l'entrée de plan.
