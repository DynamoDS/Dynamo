## Description approfondie
`List.IsRectangular` renvoie une valeur booléenne indiquant si les lignes d'une liste ont le même nombre d'éléments ou non.

Dans l'exemple ci-dessous, une valeur False est renvoyée pour une grille de points avec un point supplémentaire ajouté à une ligne. Une NurbsSurface peut être créée à partir d'un ensemble de points irréguliers, mais elle peut ne pas se comporter comme prévu.
