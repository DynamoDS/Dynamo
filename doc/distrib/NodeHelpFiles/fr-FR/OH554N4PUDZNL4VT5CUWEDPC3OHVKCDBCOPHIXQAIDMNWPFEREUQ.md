## Description approfondie
ByOriginVectors (origin, xAxis, yAxis) renvoie un nouveau CoordinateSystem à un point d'origine d'entrée avec les axes X et Y d'entrée. Dans l'exemple ci-dessous, le CoordinateSystem est placé à une origine avec de nouveaux vecteurs pour les axes X et Y. Le résultat est un CoordinateSystem pivoté par rapport au CoordinateSystem d'origine.
