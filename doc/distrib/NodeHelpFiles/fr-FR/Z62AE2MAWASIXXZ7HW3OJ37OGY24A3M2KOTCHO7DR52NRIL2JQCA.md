## Description approfondie
`Geometry.Rotate (origin, axis, degrees)` fait pivoter une géométrie d'entrée autour d'un plan de base selon un degré défini.

Dans l'exemple ci-dessous, un cône pivote autour du plan XY avec un curseur numérique.
