## Description approfondie
Dans l'exemple ci-dessous, deux surfaces cylindriques de T-Spline sont placées l'une au-dessus de l'autre. Deux sommets sont sélectionnés sur chaque surface et utilisés comme entrée pour le noeud `TSplineSurface.WeldVertices`. Le groupe de sommets fourni comme entrée `firstGroup` reste en place tandis que le `second Groupe` de sommets est déplacé pour effectuer l'opération de soudure. Facultatif : `keepSubdCreases` détermine si l'arête le long de la soudure est pliée.
