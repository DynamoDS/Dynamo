## Description approfondie
IsClosed renvoie une valeur booléenne qui indique si la courbe d'entrée est fermée ou non. Dans l'exemple ci-dessous, nous créons d'abord une courbe Nurbs à l'aide d'un nœud ByControlPoints, avec un ensemble de points générés de façon aléatoire comme entrée. Nous pouvons ensuite tester si cette courbe est fermée à l'aide d'un nœud IsClosed.
