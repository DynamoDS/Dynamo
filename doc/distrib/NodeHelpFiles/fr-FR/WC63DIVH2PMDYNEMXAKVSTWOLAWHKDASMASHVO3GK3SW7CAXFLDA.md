## Description approfondie
`CoordinateSystem.Translate (coordinateSystem, xTranslation, yTranslation, zTranslation)` renvoie un nouveau CoordinateSystem. Son emplacement est basé sur les distances de conversion dans les directions X, Y et Z.

Dans l'exemple ci-dessous, un nouveau CoordinateSystem est renvoyé à partir de distances de conversion de 3.8 sur l'axe X, de 2.7 sur l'axe Y et de 0.8 sur l'axe Z. Le nouveau CoordinateSystem est identifié par un cylindre créé sur celui-ci.
