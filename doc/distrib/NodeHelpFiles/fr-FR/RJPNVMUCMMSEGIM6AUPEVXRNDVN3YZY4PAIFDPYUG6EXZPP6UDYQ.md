## Description approfondie
Le noeud "/" est l'opérateur de division. Il renvoie le résultat de l'entrée "x" divisée par l'entrée "y". Dans l'exemple ci-dessous, un noeud "/" est utilisé pour diviser l'entrée "x" par l'entrée "y". Nous utilisons deux curseurs numériques pour contrôler les entrées de l'opérateur "/".
