## Description approfondie
Surface ByLoft avec crossSections prend une liste de courbes comme entrée. Le nœud produit une surface en lissant l'espace entre la liste de courbes dans l'ordre. Dans l'exemple ci-dessous, nous créons deux courbes, une ligne et une courbe sinusoïdale. Nous utilisons List.Create pour combiner ces deux courbes dans une liste, que nous utilisons ensuite comme entrée pour Surface ByLoft. Le résultat obtenu est une surface lissée entre une courbe sinusoïdale sur un côté, et une ligne sur l'autre côté.
