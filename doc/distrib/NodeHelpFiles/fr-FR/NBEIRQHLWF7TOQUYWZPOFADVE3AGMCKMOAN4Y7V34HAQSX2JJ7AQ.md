## Description approfondie
Le noeud "*" est l'opérateur de multiplication. Il renvoie le produit des deux nombres d'entrée. Dans l'exemple ci-dessous, un noeud "*" est utilisé pour multiplier x fois y. Nous utilisons deux curseurs numériques pour contrôler les entrées de l'opérateur "*".
