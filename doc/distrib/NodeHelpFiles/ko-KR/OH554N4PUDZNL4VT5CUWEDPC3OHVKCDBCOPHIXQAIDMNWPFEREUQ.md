## 상세
ByOriginVectors(origin, xAxis, yAxis)는 입력 X축과 Y축을 사용하여 입력 원점에서 새 CoordinateSystem을 반환합니다. 아래 예에서는 X축과 Y축에 대한 새 벡터를 사용하여 원점에 CoordinateSystem을 배치합니다. 그러면 CoordinateSystem이 원래 CoordinateSystem을 기준으로 회전합니다.
