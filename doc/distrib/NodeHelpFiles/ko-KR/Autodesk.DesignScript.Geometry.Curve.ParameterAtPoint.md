## 상세
Parameter at Point는 곡선을 따라 지정된 점의 매개변수 값을 반환합니다. 입력 점이 곡선에 없는 경우 Parameter At Point는 곡선에서 입력 점에 가까운 점의 매개변수를 반환합니다. 아래 예에서는 먼저 임의로 생성된 점 세트를 입력으로 사용하여 ByControlPoints 노드로 Nurbs 곡선을 작성합니다. x 및 y 좌표를 지정하기 위해 추가 단일 점이 코드 블록을 사용하여 작성됩니다. ParameterAtPoint 노드는 곡선을 따라 입력 점에 가장 가까운 점에서 매개변수를 반환합니다.
