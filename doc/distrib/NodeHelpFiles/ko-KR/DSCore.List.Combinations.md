## 상세
'List.Combinations'는 입력 리스트에 있는 항목의 가능한 모든 조합을 지정된 길이로 포함하는 내포된 리스트를 반환합니다. 조합의 경우 요소의 순서는 중요하지 않으므로 출력 리스트(0,1)은 (1,0)과 동일한 조합으로 간주됩니다. 'replace'를 True로 설정하면 항목이 원래 리스트로 대치되어 조합에서 반복적으로 사용될 수 있습니다.

아래 예에서는 코드 블록을 사용하여 1씩 증감하는 0에서 5까지의 숫자 범위를 생성합니다. 입력 길이가 3인 'List.Combinations'를 사용하여 범위에 있는 3개의 숫자를 결합하는 모든 방법을 생성합니다. 'replace' 부울은 True로 설정되어 있으므로 숫자가 반복적으로 사용됩니다.
