## 상세
Start Parameter는 입력 곡선을 평가할 수 있는 도메인의 시작을 나타내는 숫자를 반환합니다. 대부분의 경우 시작 매개변수는 0이 됩니다. 아래 예에서는 먼저 임의로 생성된 점 세트를 입력으로 사용하여 ByControlPoints 노드로 Nurbs 곡선을 작성합니다. 그런 다음 이 곡선을 StartParameter 노드에 대한 입력으로 사용합니다.
