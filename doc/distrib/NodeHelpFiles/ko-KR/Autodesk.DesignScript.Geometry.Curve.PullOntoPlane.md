## 상세
Pull Onto Plane은 평면의 법선을 투영 방향으로 사용해 입력 곡선을 입력 평면에 투영하여 새 곡선을 작성합니다. 아래 예에서는 먼저 임의로 생성된 점 세트를 입력으로 사용하여 ByControlPoints 노드로 Nurbs 곡선을 작성합니다. 표준 XY 평면을 평면으로 사용하여 PullOntoPlane 노드에 투영합니다. 그러면 XY 평면에 평면형 곡선이 생성됩니다.
