## 詳細
`||` ノードは、Or 演算子です。2 つのブール値を入力として取得し、一方または両方が True の場合は True を返します。次の例では、3 つの `||` ノードを使用して、少なくとも 1 つの入力ブール値が True かどうかを判断します。1 番目のノードには True と False の入力があるため True を返します。2 番目のノードには 2 つの True の入力があるため True を返します。3 番目のノードは 2 つの False の入力があるため False を返します。
