## 詳細
`Mesh.Translate (vector,distance)` ノードは、指定された `distance` と `direction` に応じてメッシュを移動させます。

次の例では、シンプルなメッシュがベクトルで定義された方向に移動しており、それが 2 つの点から構築された線分で表されています。`Line.ByStartPointEndPoint` ノードは、移動方向を視覚化するために使用されます。
