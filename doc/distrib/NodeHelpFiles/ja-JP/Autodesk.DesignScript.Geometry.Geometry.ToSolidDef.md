## 詳細
`Geometry.ToSolidDef` は、入力されたジオメトリを Solid Def JSON ファイルに変換します。このノードを使用すると、ファイルを送信する代わりに、ジオメトリの JSON 形式表現を共有できます。

次の例では、直方体を Solid Def に変換します。別のグラフで Solid Def 文字列表現を使用するには、Watch ノードの内容をコピーします。
