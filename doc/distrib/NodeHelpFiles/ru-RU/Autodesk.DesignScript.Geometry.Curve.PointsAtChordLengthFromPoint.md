## Подробности
PointsAtChordLengthFromPoint возвращают список точек на кривой, отмеряемых последовательно в соответствии с входной длиной хорды от заданной точки на кривой. В примере ниже сначала с помощью узла ByControlPoints создается NURBS-кривая, где в качестве входных элементов используется набор случайных точек. Узел PointAtParameter совместно с числовым регулятором, значение которого находится в диапазоне от 0 до 1, используется для определения исходной точки на кривой для узла PointsAtChordLengthFromPoint. Наконец, с помощью второго числового регулятора осуществляется корректировка длины хорды по прямой, которую требуется использовать.
