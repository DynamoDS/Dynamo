## Подробности
GetIsoline возвращает изолинию поверхности для входного параметра U или V. Входной элемент isoDirection используется для задания используемого направления — U или V. Значение isoDirection, равное 0, соответствует направлению U, а значение isoDirection, равное 1, — направлению V. В примере ниже сначала создается поверхность с помощью узла BySweep2Rails. Затем используются два числовых регулятора. Первый регулятор используется для задания направления изолиний (U или V), а второй — для задания параметра, для которого требуется найти изолинию.
