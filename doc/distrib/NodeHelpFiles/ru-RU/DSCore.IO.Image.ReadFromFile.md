## Подробности
ReadFromFile принимает в качестве входного элемента файл. Этот файл должен представлять собой текстовый файл, данные в котором разделяются запятой. Различные строки в файле соответствуют внешнему списку, в то время как отдельные значения в каждой строке соответствуют внутренним спискам. В примере ниже сначала с помощью узлов FilePath и File.FromPath создается объект file, указывающий на текстовый файл. Затем с помощью узла ReadFromFile создается список на основе файла CSV.
