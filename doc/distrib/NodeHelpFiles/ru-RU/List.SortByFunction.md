## Подробности
`List.SortByFunction` возвращает список, отсортированный по функции.

В примере ниже строится линия через список случайным образом расположенных точек, которые были отсортированы с помощью разных функций. Первая кривая в несортированном списке является ненаправленной. Сортировка списка по координатам X или Y каждой точки приводит к тому, что направленные линии проходят через Х и Y, соответственно.
