## Подробности
`List.DiagonalLeft` возвращает новый список, который содержит списки элементов из каждой диагонали в матрице списков от левого нижнего угла к правому верхнему.

В примере ниже создается сетка точек, через которые по диагонали проходят линии. `List.DiagonalLeft` возвращает список точек по диагоналям от левого нижнего угла сетки к правому верхнему.
