## Подробности
`Solid.ByRevolve` создает поверхность путем поворота заданной кривой профиля вокруг оси. Ось определяется точкой `axisOrigin` и вектором `axisDirection`. Начальный угол в градусах определяет, где начинается поверхность, а `sweepAngle` определяет расстояние, на которое следует продолжать поверхность вокруг оси.

В примере ниже кривая, созданная с помощью косинусной функции, используется в качестве кривой профиля, а два регулятора чисел — для управления `startAngle` и `sweepAngle`. `axisOrigin` и `axisDirection` сохраняют значения по умолчанию начала глобальной системы координат и глобальной координаты z для этого примера.
