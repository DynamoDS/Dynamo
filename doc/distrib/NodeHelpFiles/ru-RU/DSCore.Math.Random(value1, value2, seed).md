## Подробности
Math.Random(value1,value2,seed) создает случайное число между двумя значениями. Этот узел включает входное значение seed. Seed — это значение, которое инициирует рандомизированную функцию (позволяя использовать дополнительные параметры для каждого входного значения seed) и позволяет «вызывать» случайные значения с использованием того же значения для seed.
