## Подробности
ByLoftGuides возвращает объект PolySurface, полученный путем путем лофтинга по списку объектов PolyCurve(crossSections). В примере ниже объект PolySurface создается путем лофтинга по трем дугам и объекту guideCurve дуги. Дуги предоставляют лофту поперечное сечение, а guideCurve — направляющую.
