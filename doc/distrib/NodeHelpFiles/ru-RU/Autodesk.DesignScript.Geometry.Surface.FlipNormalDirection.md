## Подробности
FlipNormalDirection позволяет обратить направление векторов нормали поверхности. В примере ниже сначала создается поверхность с помощью узла BySweep2Rails. Направление нормали этой поверхности совпадает с положительным направлением оси Z. С помощью узла FlipNormalDirection создается новая поверхность с той же геометрией, однако направление нормалей при этом совпадает с отрицательным направлением оси Z.
