## Подробности
`List.GroupByKey` группирует элементы из входного списка на основе списка ключей. Список ключей должен иметь такое же количество элементов, что и список входных данных. Элементы исходного списка, которые соответствуют совпадающим ключам, группируются вместе в выходном списке.

В примере ниже диапазон чисел от 0 до 9 используется в качестве входного списка. Кроме того, создается список из 10 случайных чисел от 0 до 4 для использования в качестве ключей. `List.GroupByKey` выводит вложенный список элементов, сгруппированных по соответствующему ключу. Список также выводит список уникальных ключей.
