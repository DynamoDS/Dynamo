## In-Depth
Узел `TSplineEdge.IsBorder` возвращает значение `True`, если входное ребро Т-сплайна является границей.

В представленном ниже примере исследуются ребра двух Т-сплайновых поверхностей. Поверхности представляют собой цилиндр и его утолщенную версию. Для выбора всех ребер в обоих случаях используются узлы `TSplineTopology.EdgeByIndex`. Индексы задаются диапазоном целых чисел от 0 до n, где n — количество ребер, предоставленных узлом `TSplineTopology.EdgesCount`. Это альтернатива выбору ребер напрямую с помощью узла `TSplineTopology.DecomposedEdges`. Кроме того, в случае с утолщенным цилиндром используется узел `TSplineSurface.CompressIndices` для изменения порядка индексов ребер.
Узел `TSplineEdge.IsBorder` проверяет, какие из ребер являются ребрами границ. Положение ребер границ плоского цилиндра выделяется с помощью узлов `TSplineEdge.UVNFrame` и `TSplineUVNFrame.Position`. Утолщенный цилиндр не имеет ребер границ.
