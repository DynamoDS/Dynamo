## In-Depth
Узел `TSplineFace.Index` возвращает индекс грани на Т-сплайновой поверхности. Обратите внимание, что в топологии Т-сплайновой поверхности индексы Face, Edge и Vertex не всегда совпадают с порядковым номером элемента в списке. Для решения этой проблемы используйте узел `TSplineSurface.CompressIndices`.

В приведенном ниже примере узел `TSplineFace.Index` используется для отображения индексов всех обычных граней Т-сплайновой поверхности.
