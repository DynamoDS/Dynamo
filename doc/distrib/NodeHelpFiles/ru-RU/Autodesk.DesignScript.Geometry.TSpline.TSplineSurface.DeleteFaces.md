## Подробности

В приведенном ниже примере Т-сплайновая поверхность-примитив тетрагональной сферы создается с помощью узла `TSplineSurface.ByQuadballCenterRadius`. Набор граней выбирается с помощью узла `TSplineTopology.FaceByIndex` и служит в качестве входных данных для узла `TSplineSurface.DeleteFaces`.
