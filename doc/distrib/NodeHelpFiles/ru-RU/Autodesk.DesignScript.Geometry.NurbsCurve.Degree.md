## Подробности
Degree возвращает степень NURBS-кривой. В данном примере сначала создается несколько случайных точек, а затем с помощью узла NurbsCurve.ByPoints создается кривая, интерполированная по точкам. После этого можно использовать узел Degree для получения степени кривой. Так как при создании кривой мы не указывали степень, по умолчанию была использована степень, равная трем (ломаная из прямых имеет степень, равную единице, а кривые, состоящие не из прямых отрезков, — степень, равную трем).
