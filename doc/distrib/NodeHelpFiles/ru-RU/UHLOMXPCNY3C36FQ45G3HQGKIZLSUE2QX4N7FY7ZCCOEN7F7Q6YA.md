## Подробности
Узел `TSplineSurface.Thicken(distance, softEdges)` утолщает Т-сплайновую поверхность наружу (или внутрь, если для параметра `distance` задано отрицательное значение) на заданное расстояние (`distance`) вдоль нормалей к ее граням. Логический входной параметр `softEdges` определяет, сглаживаются ли (True) или сгибаются (False) итоговые ребра.

В приведенном ниже примере Т-сплайновая поверхность цилиндра утолщается с помощью узла `TSplineSurface.Thicken(distance, softEdges)`. Итоговая поверхность смещается в сторону для удобства визуализации.
