## Подробности
В приведенном ниже примере две Т-сплайновые поверхности цилиндров расположены друг над другом. На каждой поверхности выбраны две вершины для использования в качестве входных параметров узла `TSplineSurface.WeldVertices`. Группа вершин, указанная в качестве входного параметра `firstGroup`, остается на месте, а группа вершин `secondGroup` смещается для выполнения операции соединения. При необходимости можно настроить входной параметр `keepSubdCreases`, который определяет, сгибается ли ребро вдоль соединения.
