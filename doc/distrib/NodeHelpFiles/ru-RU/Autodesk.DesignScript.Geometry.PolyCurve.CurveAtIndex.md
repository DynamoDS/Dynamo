## Подробности
CurveAtIndex возвращает сегмент кривой с входным индексом заданной сложной кривой. Если число кривых в сложной кривой меньше заданного индекса, узел CurveAtIndex возвращает значение 0. В качестве входного значения endOrStart используется логическое значение True или False. Если задано значение True, CurveAtIndex начинает подсчет с первого сегмента сложной кривой. В противном случае подсчет выполняется в обратном порядке, начиная с последнего сегмента. В примере ниже создается набор случайных точек, а затем с помощью PolyCurve.ByPoints создается незамкнутый объект PolyCurve. После этого можно использовать узел CurveAtIndex для извлечения конкретных сегментов из объекта PolyCurve.
