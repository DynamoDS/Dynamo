## Подробности
ColorRange позволяет создать градиент по набору входных цветов. Этот узел позволяет выбирать цвета из этого градиента с использованием списка входных значений. Первое входное значение colors представляет собой список цветов, используемых в градиенте. Второе входное значение indices определяет относительное расположение входных цветов в градиенте. Этот список должен соответствовать списку цветов, а каждое значение списка должно находиться в диапазоне от 0 до 1. Точное значение неважно — важно только относительное расположение значений. Цвет, соответствующий наименьшему значению, находится в левой части градиента, а цвет, соответствующий наибольшему значению, — в правой. Ввод итоговых значений позволяет выбрать в градиенте точки в диапазоне от 0 до 1, которые будут получены на выходе. В приведенном ниже примере сначала создается два цвета: красный и синий. Порядок этих цветов в градиенте определяется списком, который создается в блоке кода. Третий блок кода используется для создания диапазона чисел от 0 до 1, который будет определять выходные цвета из градиента. Вдоль оси X создается ряд кубов, раскрашенных в соответствии с градиентом с помощью узла Display.ByGeometryColor.
