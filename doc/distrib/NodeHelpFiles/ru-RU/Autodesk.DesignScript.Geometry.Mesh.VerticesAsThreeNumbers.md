## Подробности
`Mesh.VerticesAsThreeNumbers` определяет координаты X, Y и Z вершин в предоставленной сети, где на каждую вершину приходится три числа. В приведенном ниже примере узлы `Mesh.Cuboid` и `Number.Slider` используются для создания сети-кубоида, которая затем используется в качестве входных данных для определения координат каждой вершины. В дополнение к этому узел `Mesh.Vertices` используется для вывода списка координат по каждой вершине и отображения вершин в окне предварительного просмотра. Как показано в примере, значение, сообщаемое узлом `Mesh.VerticesAsThreeNumbers`, будет в три раза больше значения из узла `Mesh.VertexCount`.
