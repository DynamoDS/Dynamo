## In-Depth
В приведенном ниже примере рассматривается случай с неоднородным ребром. Поверхность создается путем объединения двух поверхностей с общим внутренним ребром. Итоговая поверхность не имеет ни четкого переднего, ни четкого заднего края. Неоднородная поверхность отображается только в режиме рамки до тех пор, пока не будет восстановлена. В данном случае узел `TSplineEdge.IsManifold` используется для того, чтобы выделить внутренние ребра и ребра границ, являющиеся однородными.
