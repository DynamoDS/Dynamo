## In-Depth
Узел `TSplineInitialReflection.Plane` возвращает плоскость, относительно которой изначально выполняется отражение.

В приведенном ниже примере узел `TSplineSurface.Reflections` используется для запроса отражений, примененных к входной Т-сплайновой поверхности. Затем узел `TSplineReflection.Plane` используется для возврата плоскости, использованной для отражения.
