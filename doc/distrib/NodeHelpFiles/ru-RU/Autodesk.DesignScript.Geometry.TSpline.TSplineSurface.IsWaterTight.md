## Подробности
Непроницаемые Т-сплайновые поверхности — это геометрические объекты, которые при погружении не пропускают жидкость.


В приведенном ниже примере узел `TSplineSurface.IsWaterTight` используется для Т-сплайновой сферы-примитива и возвращает значение `true`. Это иллюстрирует один из случаев, когда Т-сплайновая поверхность является непроницаемой, несмотря на свою незамкнутость (вершины на полюсах сферы не объединены).
