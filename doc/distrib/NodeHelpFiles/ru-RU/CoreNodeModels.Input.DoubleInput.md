## Подробности
Узел Number позволяет ввести статическое число. Этот узел используется, например, для ввода постоянных, значение которых не меняется. Числа также можно создать с помощью узла Code Block. В примере ниже число связывается с блоком кода для создания последовательности списков с переменным шагом.
