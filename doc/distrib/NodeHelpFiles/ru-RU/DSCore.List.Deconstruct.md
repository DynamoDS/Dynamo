## Подробности
`List.Deconstruct` отделяет первый элемент списка от остальных. Этот узел имеет два выхода. В первом содержится первый элемент исходного списка. Во втором — список с оставшимися элементами из исходного списка.

В примере ниже с помощью блока кода создается диапазон чисел от 0 до 5 с шагом 1. Затем с помощью узла `List.Deconstruct` первый элемент отделяется от остальных. В результате получается элемент 0 в первом выходе и список элементов исходного списка от 1 до 5 во втором выходе.
