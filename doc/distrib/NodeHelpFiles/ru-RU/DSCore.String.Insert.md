## Подробности
Insert позволяет вставить заданную строку в базовую строку по указанному индексу. В примере ниже в качестве базовой строки используется строка Hello World. В качестве вставляемой строки используется вторая строка «Hi!». Индекс, по которому вторая строка вставляется в базовую строку, задается с помощью целочисленного регулятора.
