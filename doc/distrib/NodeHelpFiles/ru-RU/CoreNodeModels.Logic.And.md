## Подробности
And принимает переменное количество входных логических значений. Чтобы увеличить или уменьшить количество входных значений, нажимайте кнопки + и - в узле And. Каждый входной элемент должен представлять собой отдельное логическое значение. And возвращает значение True, если все входные элементы имеют значение True. В противном случае возвращается значение False. В примере ниже в качестве входных элементов для узла And используются три переключателя логических значений. Если все три переключателя установлены в положение True, возвращается значение True.
