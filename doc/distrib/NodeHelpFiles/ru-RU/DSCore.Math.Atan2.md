## Подробности
Atan2 возвращает обратный синус, также известный как арктангенс, отношения входных числителя и знаменателя. Этот выходной угол возвращается в градусах. В примере ниже входные значения для узла Atan2 задаются с помощью двух числовых регуляторов.
