## Подробности
`LoopWhile` возвращает результат циклической функции, пока не будет достигнуты ограничения.

В приведенном ниже примере создается простой цикл, в котором добавляется значение 10, начиная с 1, пока результат не превысит 100. Узел `LoopWhile` принимает переменные для входного порта `initialValue` и функции для входных портов `continueWhile` и `loopBody`. Порт `continueWhile` определяет ограничения для цикла, в данном примере это функция Less Than с аргументом 100. Порт `loopBody` определяет действие цикла, в данном примере это функция Add с аргументом 10.
