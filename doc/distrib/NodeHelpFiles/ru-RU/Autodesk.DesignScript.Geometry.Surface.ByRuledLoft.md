## Подробности
Surface.ByRuledLoft принимает в качестве входных данных упорядоченный список кривых и выполняет лофтинг для поверхности соединения прямых линий между кривыми. По сравнению с узлом ByLoft узел ByRuledLoft работает чуть быстрее, однако результирующая поверхность является менее плавной. В примере ниже представлена линия вдоль оси X. Эта линия преобразуется в последовательность линий, следующих синусоиде в направлении Y. Использование этого результирующего списка линий в качестве входного элемента для узла Surface.ByRuledLoft позволяет получить поверхность с прямыми сегментами между входными кривыми.
