## Подробности
`Mesh.BooleanUnion` возвращает результат объединения двух входных сетей в одну и удаления накладывающихся частей сетей.
В приведенном ниже примере `Mesh.BooleanUnion` используется для объединения двух сетей: `Mesh.Sphere` и `Mesh.Cuboid`.

(Для визуализации структуры сети используется узел `Mesh.Edges`.)
