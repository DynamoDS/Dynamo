## Подробности
PrincipalDirectionsAtParameter использует входные параметры U и V и возвращает значения кривизны в направлениях U и V. Этот узел возвращает список из двух векторов, первый из которых представляет собой направляющий вектор в направлении U, а второй — направляющий вектор в направлении V. В примере ниже сначала создается поверхность с помощью узла BySweep2Rails. Затем с помощью двух числовых регуляторов задаются параметры U и V для определения направлений с помощью узла PrincipalDirectionsAtParameter.
