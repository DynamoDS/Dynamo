## Подробности
NumberOfCurves возвращает количество сегментов кривой, составляющих заданный объект PolyCurve. В примере ниже создается набор случайных точек, а затем с использованием узла PolyCurve.ByPoints создается объект PolyCurve. Если параметр connectLastToFirst имеет значение False, количество кривых будет на единицу меньше количества точек. В противном случае, если создать замкнутый объект PolyCurve, количество кривых будет равно количеству исходных точек.
