## Подробности
В приведенном ниже примере узел `TSplineTopology.FaceByIndex` извлекает грань Т-сплайновой поверхности. Затем эта грань используется в качестве входного параметра для выполнения операции выдавливания с помощью узла `TSplineSurface.ExtrudeFaces`.
