## Подробности
AngleWithVector возвращает значение угла между двумя векторами в диапазоне от 0 до 180 градусов. В примере ниже значение угла возвращается для двух векторов. Перемещайте регуляторы отметки в направлении Z и угла поворота, чтобы увидеть, как они влияют на результат измерения угла. Векторы представлены в виде линий.
