## Подробности
Data.ExportToExcel позволяет экспортировать входные данные в файл формата XLSX. Этот узел принимает входные значения startRow и startColumn в формате целых чисел и выводит данные для использования на последующих этапах. Входные параметры overWrite и writeAsString являются необязательными и имеют значения по умолчанию. Параметр overWrite полностью заменяет содержимое файла Excel, а параметр writeAsString представляет все данные в виде строк.

Примечание. Этот узел не открывает Excel во время выполнения.
