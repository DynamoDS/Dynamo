## Подробности
NurbsCurve.ByPoints принимает список точек для построения NURBS-кривой, а также позволяет изменять степень кривой. В данном примере на плоскости XY МСК генерируется шесть случайных точек, которые соединяются по порядку в направлениях X и Y.
