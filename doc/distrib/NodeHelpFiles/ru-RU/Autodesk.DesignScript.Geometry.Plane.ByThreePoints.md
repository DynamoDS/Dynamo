## Подробности
Plane.ByThreePoints позволяет создать плоскость, проходящую через три входные точки. Точки не должны быть коллинеарными. В примере ниже три блока кода используются для задания трех исходных точек, после чего с помощью функции ByThreePoints создается плоскость.
