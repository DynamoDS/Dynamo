## Подробности
`Mesh.ExportMeshes` экспортирует сеть в файл. Тип сети для экспорта определяется расширением файла. Например, если имя файла оканчивается на «.stl», сеть будет экспортирована в файл формата STL, как в примере ниже.
Поддерживаются следующие форматы:
- .mix — Meshmixer;
- .obj — Wavefront OBJ;
- .stl — формат STL;
- .dae — COLLADA;
- .ply — файл формата Polygon.
