## Подробности
В приведенном ниже примере Т-сплайновая поверхность создается путем вращения кривой профиля, которая задается параметром `profile` вокруг заданного значения `axisOrigin` в направлении `axisDirection`. Для управления степенью вращения используются параметры `startAngle` и `sweepAngle`. Определение задается с помощью параметров `radialSpans` и `axialSpans`. Входной параметр `uniform` управляет распределением пролетов вдоль профиля: оно может быть равномерным или учитывать кривизну. Исходная симметрия формы задается с помощью входного параметра `symmetry`. Входной параметр `inSmoothMode` используется для переключения между режимами сглаживания и рамки для предварительного просмотра T-сплайновой поверхности.
