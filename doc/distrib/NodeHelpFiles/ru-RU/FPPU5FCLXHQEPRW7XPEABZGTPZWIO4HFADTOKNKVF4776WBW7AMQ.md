## Подробности
`CoordinateSystem.Scale (coordinateSystem, xamount, yamount, zamount)` масштабирует систему координат неравномерно вокруг начала координат.

В приведенном ниже примере система координат создается на основе точки по координатам, а затем масштабируется с помощью узла `CoordinateSystem.Scale`.
