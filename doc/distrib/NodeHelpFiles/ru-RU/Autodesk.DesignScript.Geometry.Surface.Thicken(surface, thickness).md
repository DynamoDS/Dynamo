## Подробности
`Surface.Thicken` создает тело путем смещения поверхности в соответствии с введенной толщиной, закрывая концы для замыкания тела.

В примере ниже сначала создается поверхность с помощью узла `Surface.BySweep2Rails`. Затем создается тело с помощью регулятора чисел, чтобы определить входное значение толщины узла `Surface.Thicken`.
