## Подробности
PointAtChordLength возвращает точку, находящуюся на расстоянии, равном заданной длине хорды, от исходного заданного параметра. В примере ниже сначала с помощью узла ByControlPoints создается NURBS-кривая, где в качестве входных элементов используется набор случайных точек. Числовой регулятор используется для управления расстоянием хорды по прямой, на котором требуется найти точку. Второй регулятор, значение которого находится в диапазоне от 0 до 1, используется для управления начальной точкой вдоль кривой, от которой будет измеряться расстояние хорды. Наконец, с помощью логического переключателя определяется направление, в котором измеряется расстояние хорды.
