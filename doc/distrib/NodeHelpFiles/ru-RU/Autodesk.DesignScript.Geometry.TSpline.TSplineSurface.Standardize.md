## Подробности
Узел `TSplineSurface.Standardize` используется для стандартизации Т-сплайновой поверхности.
Стандартизация означает подготовку Т-сплайновой поверхности к NURBS-преобразованию и подразумевает удлинение всех Т-точек до тех пор, пока они не будут отделены от нулевых точек по крайней мере двумя изолиниями. Стандартизация не меняет форму поверхности, но может добавлять управляющие точки для соответствия требованиям геометрии и обеспечения совместимости поверхности с NURBS.

В приведенном ниже примере одна из граней Т-сплайновой поверхности, созданной с помощью узла `TSplineSurface.ByBoxLengths`, разделена.
Узел `TSplineSurface.IsStandard` проверяет, является ли поверхность стандартной, но возвращает отрицательный результат.
Затем для стандартизации поверхности используется узел `TSplineSurface.Standardize`. Итоговая поверхность проверяется с помощью узла `TSplineSurface.IsStandard`, который подтверждает, что она является стандартной.
The nodes `TSplineFace.UVNFrame` and `TSplineUVNFrame.Position` are used to highlight the subdivided face in the surface.
