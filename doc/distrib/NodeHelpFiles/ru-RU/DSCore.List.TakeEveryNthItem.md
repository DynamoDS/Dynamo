## Подробности
`List.TakeEveryNthItem` создает новый список, содержащий только элементы из входного списка, которые находятся в интервале, заданном входным значением n. Начальную точку интервала можно изменить с помощью входного параметра `offset`. Например, если для n указать 3 и оставить смещение 0 по умолчанию, сохранятся элементы с индексами 2, 5, 8 и т. д., а при смещении 1 — элементы с индексами 0, 3, 6 и т. д. Обратите внимание, что смещение переносится на весь список. Если выбранные элементы необходимо удалить, а не сохранить, см. `List.DropEveryNthItem`.

В приведенном ниже примере сначала создается список чисел с использованием `Range`, а затем каждое второе число сохраняется, поскольку для n указано значение 2.
