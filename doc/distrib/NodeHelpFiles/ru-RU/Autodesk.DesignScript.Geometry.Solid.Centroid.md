## Подробности
Centroid возвращает точку, находящуюся в центроиде тела. Центроид можно рассматривать как центр масс тела. Обратите внимание, что центр масс может существовать вне фактического тела. В примере ниже для создания объекта Solid.ByLoft используется набор окружностей, а полученное тело используется в качестве входного элемента для узла Centroid.
