## Подробности
`Mesh.PlaneCut` возвращает сеть, разрезанную с помощью заданной плоскости. В результате остается часть сети, расположенная на стороне плоскости в направлении нормали входных данных `plane`. Параметр `makeSolid` определяет, обрабатывается ли сетка как объект `Solid`. В этом случае срез заполняется минимальным количеством треугольников для покрытия каждого отверстия.

В приведенном ниже примере полая сеть, полученная в результате операции `Mesh.BooleanDifference`, разрезается плоскостью под углом.
