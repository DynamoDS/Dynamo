## Подробности
NurbsCurve.ByControlPoints принимает список точек, которые используются в качестве контрольных, и выводит NURBS-кривую. В данном примере с помощью Code Block создаются два списка, соответствующих местоположениям X и Y последовательности точек. Эти точки используются для построения NURBS-кривой.
