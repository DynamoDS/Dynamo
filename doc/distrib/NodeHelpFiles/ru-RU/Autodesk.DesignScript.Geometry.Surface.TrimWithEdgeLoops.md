## Подробности
TrimWithEdgeLoops возвращает новую поверхность, которая представляет собой обрезанную входную поверхность. В примере ниже из поверхности вырезаются два контура, в результате чего возвращаются две новые поверхности, выделенные синим цветом. С помощью числового регулятора можно скорректировать форму новых поверхностей. Входной элемент loops принимает сложную кривую или список сложных кривых.
