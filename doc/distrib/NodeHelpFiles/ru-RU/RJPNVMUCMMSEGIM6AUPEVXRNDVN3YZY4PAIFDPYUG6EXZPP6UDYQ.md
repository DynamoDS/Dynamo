## Подробности
Узел «/» является оператором деления. Он возвращает результат деления входного значения «x» на входное значение «y». В примере ниже узел «/» используется для деления значения «x» на значение «y». Для управления входными данными для оператора «/» используются два регулятора числовых значений.
