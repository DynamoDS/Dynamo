## Подробности
ByOrigin(x, y, z) возвращает новую систему координат, где в качестве значений X, Y и Z начала координат используются входные данные узла. В примере ниже создается новая система координат с началом в точке (4,2,1) на основе входных значений 4, 2 и 1 для X, Y, и Z соответственно.
