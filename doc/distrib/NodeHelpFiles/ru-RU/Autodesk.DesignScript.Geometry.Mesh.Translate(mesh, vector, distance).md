## Подробности
Узел `Mesh.Translate (vector,distance)` смещает сеть на заданное расстояние (`distance`) в заданном направлении (`direction`).

В приведенном ниже примере простая сеть смещается в направлении вектора и представляется в виде линии, построенной через две точки. Для визуализации направления смещения используется узел `Line.ByStartPointEndPoint`.
