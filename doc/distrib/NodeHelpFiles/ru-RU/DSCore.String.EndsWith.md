## Подробности
EndsWith возвращает логическое значение, показывающее, заканчивается ли заданная строка второй входной строкой. По умолчанию этот узел чувствителен к регистру. Для входного параметра ignoreCase можно использовать логическое значение, чтобы узел игнорировал регистр строк. В примере ниже в качестве исходной строки используется строка Hello World, а качестве искомой строки — orld. Так как строка Hello World заканчивается строкой orld, узел EndsWith возвращает значение True.
