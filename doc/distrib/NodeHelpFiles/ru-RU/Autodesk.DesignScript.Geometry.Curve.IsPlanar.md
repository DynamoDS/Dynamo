## Подробности
IsPlanar возвращает логическое значение, показывающее, является ли входная кривая плоской. В примере ниже сначала с помощью узла ByControlPoints создается NURBS-кривая, где в качестве входных элементов используется набор случайных точек. После этого с помощью узла IsPlanar можно проверить, является ли кривая плоской.
