## Подробности
Узел `TSplineSurface.BuildFromLines` позволяет создать более сложную Т-сплайновую поверхность для использования в качестве конечной геометрии или пользовательского примитива, форма которого ближе к требуемой, чем у стандартных примитивов. Результат представляет собой замкнутую или незамкнутую поверхность, которая может содержать отверстия и/или согнутые ребра.

Для входного параметра узла задается список кривых, представляющих собой «управляющую клетку» для Т-сплайновой поверхности. Настройка списка отрезков требует некоторой подготовки, и выполнять ее следует в соответствии с определенными рекомендациями.
— Линии не должны перекрываться.
— Граница многоугольника должна быть замкнутой, а конечная точка каждой линии должна совпадать по крайней мере с одной другой конечной точкой. Линии должны пересекаться в точках.
— В областях с большей детализацией требуется большая плотность многоугольников.
— Квадраты предпочтительнее треугольников и nGon, так как ими легче управлять.

В приведенном ниже примере использование этого узла демонстрируется путем создания двух Т-сплайновых поверхностей. Для параметра `maxFaceValence` оставлено значение по умолчанию в обоих случаях, а значение `snappingTolerance` корректируется таким образом, чтобы отрезки в пределах значения допуска можно было считать соединенными. Для формы слева параметру `creaseOuterVertices` назначается значения False, чтобы две вершины углов остались острыми и не скругленными. Форма слева не содержит внешних вершин, и для этого входного параметра оставлено значение по умолчанию. Параметр `inSmoothMode` активирован для обеих форм для удобства предварительного просмотра.
