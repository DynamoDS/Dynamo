## Подробности
Узел «<» является оператором «Меньше». Если входное значение «x» меньше входного значения «y», возвращается значение «Истина». Если «x» больше значения «y» или если эти значения равны, этот оператор возвращает значение «Ложь». Существует в словаре 1.x; измененное имя графа: LessThan.dyn Y

В примере ниже узел «<» используется для определения того, меньше ли входное значение «x» чем входное значение «y». Для управления входными данными для оператора «<» используются два регулятора числовых значений.
