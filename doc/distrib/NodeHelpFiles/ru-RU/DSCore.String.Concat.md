## Подробности
Concat позволять сжать несколько строк в одну. Изменить количество входных строк можно с помощью кнопок + и - в узле Concat. В примере ниже используются две отдельные строки Hello и World. Они объединяются в строку HelloWorld с помощью узла Concat.
