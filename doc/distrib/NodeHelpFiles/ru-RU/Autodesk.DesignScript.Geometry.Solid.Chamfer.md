## Подробности
Chamfer возвращает новое тело со скошенными ребрами. Входные значения ребер указывают, какие ребра скашивать, а входное значение смещения определяет границы фаски. В примере ниже представлен куб, использующий входные значения по умолчанию. Чтобы получить соответствующие ребра куба, сначала необходимо расчленить куб и представить его грани в виде списка поверхностей. Затем ребра куба извлекаются с помощью узла Face.Edges. Первое ребро каждой грани извлекается с помощью GetItemAtIndex. Для управления длиной смещения фаски используется числовой регулятор.
