## Подробности
Узел `Mesh.Explode` выбирает одну сеть и возвращает список граней сети в виде независимых сетей.

В приведенном ниже примере показана сеть-купол, которая разделена с помощью `Mesh.Explode` и каждая грань которой смещена в направлении своей нормали с помощью узлов `Mesh.TriangleNormals` и `Mesh.Translate`. Несмотря на то, что в этом примере грани сети выглядят как квадраты, на самом деле они являются треугольниками с одинаковыми нормалями.
