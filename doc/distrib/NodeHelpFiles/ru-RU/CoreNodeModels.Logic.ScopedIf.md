## Подробности
ScopeIf возвращает входное значение входного элемента True или False в зависимости от того, какое логическое значение выбрано для входного параметра test. В приведенном ниже файле примера большой и маленький списки чисел подключены к входным значениям True и False. Отправляя результат работы узла ScopeIfThrough в узел создания точки, можно переключаться между сетками точек двух разных размеров.
