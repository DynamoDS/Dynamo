## Подробности
WriteToFile возвращает список списков, записанный в файл Excel. Имя листа, начальную строку и начальный столбец для записи данных можно задать в качестве входных значений. Кроме того, можно указать, следует ли перезаписывать существующие данные. В примере ниже список списков, содержащий последовательность чисел, записывается в файл sampleexcel.xlsx со строки 0 (или 1 в Excel ) и столбца 1 (или B в Excel). Теперь файл Excel содержит девять новых столбцов с данными в двух существующих строках. Для отображения данных в формате Excel накладываются снимки экрана с файлом sampleexcel.xlsx до и после записи.
