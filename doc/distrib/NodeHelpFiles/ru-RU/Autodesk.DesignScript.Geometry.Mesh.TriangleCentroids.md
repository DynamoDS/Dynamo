## Подробности
`Mesh.TriangleCentroids` возвращает центральную точку каждой треугольной грани сети.

В приведенном ниже примере используется `Mesh.TriangleCentroids` в сочетании с `Mesh.TriangleNormals` для построения нормалей каждой грани сети.
