## Подробности
ApproximateWithArcAndLineSegments принимает в качестве входного элемента кривую и возвращает список прямых линий и дуг-кривых, аппроксимирующих исходную кривую. В примере ниже сначала с помощью узла ByControlPoints создается NURBS-кривая, в качестве входных данных которой используется набор случайных точек. Затем эта кривая используется в качестве входного элемента для узла ApproximateWithArcAndLineSegments.
