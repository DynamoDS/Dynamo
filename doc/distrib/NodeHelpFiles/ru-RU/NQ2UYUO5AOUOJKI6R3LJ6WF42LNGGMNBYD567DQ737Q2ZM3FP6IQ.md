## Подробности
PrincipalCurvaturesAtParameter использует входные параметры U и V и возвращает значения кривизны в направлениях U и V. Этот узел возвращает список из двух чисел, первое из которых представляет собой кривизну в направлении U, а второе — кривизну в направлении V. В примере ниже сначала создается поверхность с помощью узла BySweep2Rails. Затем с помощью двух числовых регуляторов задаются параметры U и V для определения значений кривизны с помощью узла PrincipalCurvaturesAtParameter.
