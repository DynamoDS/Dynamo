## Подробности
`List.IndexOf` возвращает индекс заданного элемента в списке. Если заданный элемент не найден в списке или тип его данных не соответствует элементу списка (например, входной элемент является числом с двойной точностью, а элемент списка — строкой), то `List.IndexOf` возвращает нулевое значение.

Обратите внимание, что этот узел возвращает только первый индекс, даже если элемент присутствует в списке несколько раз. Для возврата нескольких индексов используйте `List.AllIndicesOf`.
