## Подробности
Операцию `Mesh.MakeHollow` можно использовать для создания полости в объекте сети при подготовке к 3D-печати. Создание полости в сети значительно сокращает количество материала, время и стоимость печати. Входной параметр `wallThickness` определяет толщину стенок объекта сети. При необходимости с помощью `Mesh.MakeHollow` может создавать технические отверстия для удаления лишнего материала в процессе печати. Размер отверстий и их количество определяются входными параметрами `holeCount` и `holeRadius`. Входные параметры `meshResolution` и `solidResolution` определяют разрешение итоговой сети. Чем выше значение параметра `meshResolution`, тем точнее внутренняя часть смещает исходную сеть и тем больше треугольников. Чем выше значение `solidResolution`, тем больше мелких деталей исходной сети сохраняется на внутренней части полой сети.
В приведенном ниже примере узел `Mesh.MakeHollow` используется для сети в форме конуса. В его основание добавлены пять технических отверстий.
