## Подробности
BySurfaceColors позволяет отобразить входной объект геометрии с градиентом цветов. В примере ниже отображается окружность с градиентом двух цветов RGB. Если соответствующая функция не отключена, новый цветной объект будет перекрыт предварительным просмотром исходного объекта геометрии.
