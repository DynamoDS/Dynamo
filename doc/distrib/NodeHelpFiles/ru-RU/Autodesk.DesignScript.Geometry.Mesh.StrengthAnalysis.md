## Подробности
 Узел `Mesh.StrengthAnalysis` возвращает список цветовых обозначений для каждой вершины. Полученный результат можно использовать вместе с узлом `Mesh.ByMeshColor`. Более прочные участки сети окрашены в зеленый цвет, а более слабые обозначаются тепловой картой от желтого до красного. Анализ может привести к ложным срабатываниям, если сеть слишком грубая или нерегулярная (т. е. включает много длинных, тонких треугольников). Можно использовать `Mesh.Remesh` для создания регулярной сети, а затем `Mesh.StrengthAnalysis` — для получения лучших результатов.

В приведенном ниже примере `Mesh.StrengthAnalysis` используется для цветовой маркировки структурной прочности сети в форме сетки. В результате будет получен список цветов, соответствующих длине вершин сети. Этот список можно использовать вместе с узлом `Mesh.ByMeshColor` для окрашивания сети.
