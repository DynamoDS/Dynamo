## Подробности
`Thread.Pause` возвращает результат работы узла после приостановки выполнения графика (потока) на заданное время.

В приведенном ниже примере окружность выдавливается в поверхность через 5 секунд после запуска графика. Чтобы использовать задержку, необходимо задать режим запуска «Вручную».
