## In-Depth
Объект UVNFrame грани предоставляет полезную информацию о положении и ориентации грани, возвращая вектор нормали и направления UV.
В приведенном ниже примере узел `TSplineFace.UVNFrame` используется для визуализации распределения граней на примитиве тетрагональной сферы. Узел `TSplineTopology.DecomposedFaces` используется для запроса всех граней, после чего используется узел `TSplineFace.UVNFrame` для извлечения положений центроидов граней в виде точек. Для визуализации точек используется узел `TSplineUVNFrame.Position`. Чтобы просмотреть метки в фоновом режиме, включите в контекстном меню узла параметр «Показать метки».
