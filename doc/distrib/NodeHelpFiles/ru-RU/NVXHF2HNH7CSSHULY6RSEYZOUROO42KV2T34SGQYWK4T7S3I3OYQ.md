## In-Depth
В приведенном ниже примере создается Т-сплайновая тетрагональная сфера-примитив, расположенная в локальной системе координат `cs`, имеющая заданный радиус (`radius`) и разделенная на заданное количество пролетов (`spans`). Исходная симметрия формы определяется входным параметром `symmetry`. Входной параметр `inSmoothMode` используется для переключения между режимами сглаживания и рамки при предварительном просмотре Т-сплайновой поверхности.
