## Подробности
`Geometry.Translate (direction, distance)` перемещает входную геометрию в заданное векторное направление. Входное значение `distance` определяет расстояние перемещения.

В примере ниже конус переносится вдоль оси Х. Расстояние от исходного конуса определяется с помощью регулятора чисел.

В примере ниже кубоид преобразуется в Solid Def. Скопируйте содержимое узла Watch, чтобы использовать строковое представление Solid Def на другом графике.
