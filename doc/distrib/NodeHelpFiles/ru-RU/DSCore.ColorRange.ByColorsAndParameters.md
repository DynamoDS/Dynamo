## Подробности
ByColorsAndParameters позволяет создать цветовой 2D-диапазон на основе списка входных цветов и соответствующего списка заданных параметров UV в диапазоне от 0 до 1. В примере ниже в блоке кода создаются три разных цвета (в данном случае зеленого, красного и синего), которые затем объединяются в список. В отдельном блоке кода создаются три параметра UV — по одному для каждого цвета. Эти два списка используются в качестве входных элементов для узла ByColorsAndParameters. Последующий узел GetColorAtParameter и узел Display.ByGeometryColor используются для визуализации цветового 2D-диапазона на наборе кубов.
