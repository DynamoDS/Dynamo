## Подробности
`List.Combinations` возвращает вложенный список, который включает все возможные сочетания заданной длины из элементов входного списка. Для сочетаний порядок элементов не имеет значения, поэтому выходной список (0,1) считается таким же сочетанием, что и (1,0). Если `replace` имеет значение True, элементы исходного списка заменяются, что позволяет использовать их в сочетании повторно.

В примере ниже с помощью блока кода создается диапазон чисел от 0 до 5 с шагом 1. В качестве аргумента для `List.Combinations` передается число 3, что позволяет сгенерировать все различные сочетания из 3 чисел в диапазоне. Логическое значение `replace` равно True, поэтому числа будут использоваться повторно.
