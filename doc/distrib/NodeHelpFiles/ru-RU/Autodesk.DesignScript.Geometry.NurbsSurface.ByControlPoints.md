## Подробности
NurbsSurface.ByControlPoints принимает двумерный массив контрольных вершин и выводит NURBS-поверхность. Этот узел также позволяет калибровать степень в направлениях U и V. В данном примере с помощью диапазона в Code Block создается сетка точек 10 x 10. Затем сетка случайным образом масштабируется в направлении оси Z, при этом изменяя выходную поверхность из узла ByControlPoints.
