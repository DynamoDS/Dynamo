## Подробности
IsPeriodic возвращает логическое значение True, если кривая является периодической, в противном случае возвращается значение False. В данном примере сначала генерируется несколько случайных точек, после чего с помощью узла NurbsCurve.ByPoints создается кривая, интерполированная по точкам. С помощью узла IsPeriodic можно определить, является ли кривая периодической.
