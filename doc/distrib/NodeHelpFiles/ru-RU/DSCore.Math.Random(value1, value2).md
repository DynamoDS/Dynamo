## Подробности
Узел Random с двумя входными значениями позволяет управлять диапазоном сгенерированного случайного числа. Этот узел создает случайные числа между двумя входными значениями. В примере ниже два входных значения для узла Random задаются с помощью двух числовых регуляторов.
