## Подробности
`List.SetDifference` возвращает новый список, в который входят только те объекты из списка list1, которых нет в списке list2.

В примере ниже список чисел от 0 до 9 с шагом 1 сравнивается со списком чисел от 0 до 18 с шагом 2. `List.SetDifference` возвращает нечетные числа меньше 10, поскольку они содержатся только в списке List1 и отсутствуют в списке List2.
