## Подробности
SelfIntersections возвращает список всех точек, в которых многоугольник пересекает сам себя. В примере ниже сначала создается список случайных несортированных углов и радиусов, которые используются в узле Points.ByCylindricalCoordinates. Поскольку отметка оставалась постоянной и углы этих точек не сортировались, многоугольник, созданный с помощью узла Polygon.ByPoints, будет плоским и, вероятно, самопересекающимся. Затем с помощью узла SelfIntersections можно найти точки пересечения.
