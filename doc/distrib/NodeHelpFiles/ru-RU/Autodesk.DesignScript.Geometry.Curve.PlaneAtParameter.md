## Подробности
PlaneAtParameter возвращает плоскость, совмещенную с кривой по заданному параметру. Вектор нормали результирующей плоскости будет соответствовать касательной кривой. Параметризация кривой измеряется в диапазоне от 0 до 1, где 0 обозначает начало кривой, а 1 — ее конец. В примере ниже сначала с помощью узла ByControlPoints создается NURBS-кривая, где в качестве входных элементов используется набор случайных точек. Для управления входным элементом parameter узла PlaneAtParameter используется числовой регулятор, значение которого находится в диапазоне от 0 до 1.
