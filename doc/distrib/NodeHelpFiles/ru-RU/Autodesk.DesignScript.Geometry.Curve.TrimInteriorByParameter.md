## Подробности
TrimInteriorByParameter позволяет удалить внутреннюю часть заданной кривой на основе двух входных параметров. Этот узел выводит две оставшиеся наружные части кривой в виде списка из двух кривых. В примере ниже для создания списка точек используются два набора случайных чисел. Эти точки используются для создания NURBS-кривой по контрольным точкам. Для управления входными элементами parameter в узле TrimInteriorByParameter используются два числовых регулятора с диапазоном значений от 0 до 1.
