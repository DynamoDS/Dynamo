## Подробности
Узел «<=» является оператором «Меньше или равно».Если входное значение «x» меньше входного значения «y» или если эти значения равны, этот оператор возвращает значение «Истина».  Если значение «x» больше значения «y», возвращается значение «Ложь». В примере ниже оператор «<=» используется для определения того, меньше ли значение «x» чем значение «y» или равно ему. Для управления входными данными для оператора «<=» используются два регулятора числовых значений.
