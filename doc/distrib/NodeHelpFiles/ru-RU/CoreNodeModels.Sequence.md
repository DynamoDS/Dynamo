## Подробности
Sequence позволяет создать список чисел. Список начинается с входного значения Start. К каждому последующему значению в списке прибавляется входное значение Step. Входное значение Amount определяет количество элементов в списке. В примере создается список из 25 чисел. Список начинается с 1, а шаг равен 2.
