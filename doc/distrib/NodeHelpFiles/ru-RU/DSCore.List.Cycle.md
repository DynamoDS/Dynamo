## Подробности
`List.Cycle` получает на входе список и возвращает новый список, который состоит из входного списка, повторяющегося несколько раз. Количество повторов указывается во входном параметре `amount`.

В примере ниже с помощью блока кода создается диапазон чисел от 0 до 5 с шагом 1. Затем используется узел `List.Cycle` с входным параметром amount равным 3. В результате получается список с 18 элементами, состоящий из 3 повторов исходного списка.
