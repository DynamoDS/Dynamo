## Подробности
`List.Sort` организует список элементов в соответствии с порядком по умолчанию для определенного типа данных. Например, список чисел сортируется от меньшего к большему, а список строк — в алфавитном порядке.

В приведенном ниже примере узел `Math.RandomList` используется для формирования списка случайных чисел от 0 до 1. Затем значения умножаются на 10 и используется узел `Math.Floor` для получения списка случайных чисел от 0 до 9. Затем `List.Sort` создает список, сортированный от меньшего числа к большему.
