## Подробности
`Surface.TrimWithEdgeLoops` обрезает поверхность с набором из одной или нескольких замкнутых объектов PolyCurve, которые должны лежать на поверхности в пределах заданного допуска. Если одно или несколько отверстий необходимо вырезать во входной поверхности, то для каждого отверстия необходимо задать один наружный контур границы поверхности и один внутренний контур. Если необходимо обрезать область между границей поверхности и отверстиями, необходимо указать только контур для каждого отверстия. Для периодической поверхности без наружного контура, такой как сферическая поверхность, обрезанная область может определяться путем изменения направления кривой контура.

Допуск используется при определении того, совпадают ли концы кривой и совпадает ли кривая и поверхность. Предоставленный допуск не может быть меньше любого из допусков, используемых при создании входных объектов PolyCurve. Значение по умолчанию 0.0 означает, что будет использоваться наибольший допуск, используемый при создании входных объектов PolyCurve.

В примере ниже два контура вырезаются в поверхности, возвращая две новые поверхности, выделенные синим цветом. С помощью регулятора чисел корректируется форма новых поверхностей.
