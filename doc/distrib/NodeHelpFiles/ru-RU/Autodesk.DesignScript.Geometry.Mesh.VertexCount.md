## Подробности
Этот узел подсчитывает количество вершин в предоставленной сети. В приведенном ниже примере `Mesh.Cuboid` используется для создания сети-кубоида, которая затем используется в качестве входных данных для подсчета вершин. В дополнение к этому узел `Mesh.Vertices` используется для выделения вершин куба для удобства предварительного просмотра и просмотра списка координат вершин.
