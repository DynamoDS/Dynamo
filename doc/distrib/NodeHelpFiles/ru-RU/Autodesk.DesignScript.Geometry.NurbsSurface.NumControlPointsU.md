## Подробности
NumControlPointsU позволяет определить количество контрольных точек в направлении U объекта NurbsSurface. Этот узел возвращает целое число. В примере ниже NurbsSurface возвращает для своих контрольных точек U целое число, равное 22. Контрольные точки возвращаются в виде объектов Point с помощью узла NurbsSurface.ControlPoints.
