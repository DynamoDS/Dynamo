## Подробности
`Mesh.Cuboid` создает сеть-кубоид по значениям `width`, `length` и  `height` с центром во входной точке и заданным количеством делений по осям X, Y, Z. Если количество делений не задано или любой из входных параметров `xDivisions`, `yDivisions` или `zDivisions` равен нулю, по всем направлениям используется значение по умолчанию — 5 делений.
В приведенном ниже примере узел `Mesh.Cuboid` используется для создания сети-кубоида, а узел `Mesh.Triangles` — для визуализации распределения треугольников сети.
