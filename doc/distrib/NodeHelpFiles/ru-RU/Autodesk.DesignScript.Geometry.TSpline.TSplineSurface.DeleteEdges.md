## Подробности

В приведенном ниже примере Т-сплайновая поверхность-примитив тетрагональной сферы создается с помощью узла `TSplineSurface.ByQuadballCenterRadius`. Набор ребер выбирается с помощью узла `TSplineTopology.EdgeByIndex` и служит в качестве входных данных для узла `TSplineSurface.DeleteEdges`.
