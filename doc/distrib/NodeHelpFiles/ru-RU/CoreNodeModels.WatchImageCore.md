## Подробности
WatchImage позволяет предварительного просматривать растровые изображения. В примере ниже в узле WatchImage выполняется предварительный просмотр растрового изображения, используемого в качестве поля высот. Это позволяет сравнить полученный объект геометрии с исходным изображением.
