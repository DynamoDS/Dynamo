## Подробности
PlaneByBestFitThroughPoints позволяет найти плоскость, максимально приближенную к заданному набору точек. В примере последовательность компонентов RandomList используется для создания случайных координат x, y и z. Точки создаются по координатам, а затем список точек используется в качестве входных данных для ByBestFitThroughPoints.
