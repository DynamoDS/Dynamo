## Подробности
В приведенном ниже примере некоторые грани Т-сплайновой поверхности удаляются, в результате чего поверхность становится недопустимой. Режим сглаживания нельзя активировать для недопустимых поверхностей, и результат узла `TSplineSurface.IsExtractable` показывает значение `false`.
