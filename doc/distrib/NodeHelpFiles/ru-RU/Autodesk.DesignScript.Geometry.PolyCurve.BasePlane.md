## Подробности
BasePlane возвращает плоскость плоской сложной кривой. Если кривая не является плоской, возникает ошибка. В примере ниже создается набор случайных точек, а затем узел PolyCurve.ByPoints используется для создания объекта PolyCurve. Так как использовались только координаты X и Y, созданная сложная кривая является плоской на плоскости XY МСК.
