## Подробности
ExtendEnd позволяет удлинить входную кривую на заданное входное расстояние, начиная с конечной точки кривой. В примере ниже сначала с помощью узла ByControlPoints создается NURBS-кривая, где в качестве входных элементов используется набор случайных точек. Числовой регулятор позволяет управлять расстоянием удлинения.
