## Подробности
FromPixels позволяет создать объект изображения на основе входного двумерного массива цветов. В примере ниже сначала в блоке кода генерируется диапазон чисел от 0 до 255. Для создания цветов на основе этого диапазона используется узел Color.ByARGB. Чтобы создать двумерный массив, для параметра переплетения этого узла задается векторное произведение. Далее используется узел Image.FromPixels для создания изображения. Выполнить предварительный просмотр созданного изображения можно с помощью узла WatchImage.
