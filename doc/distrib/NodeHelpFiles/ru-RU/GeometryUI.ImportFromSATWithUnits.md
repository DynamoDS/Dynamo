## Подробности
`Geometry.ImportFromSATWithUnits` импортирует геометрию в Dynamo из файла SAT и `DynamoUnit.Unit`, конвертируемого из миллиметров. Этот узел принимает объект файла или путь к файлу в качестве первого входного параметра и `dynamoUnit` — в качестве второго. Если `dynamoUnit` имеет значение NULL, геометрия SAT импортируется без единиц измерения, при этом геометрические данные импортируются в файл без преобразования единиц измерения. Если передается единица измерения, внутренние единицы файла SAT преобразуются в указанные единицы.

В Dynamo не используются единицы измерения, однако числовые значения на графике Dynamo, скорее всего, по-прежнему имеют неявные единицы измерения. Входное значение `dynamoUnit` можно использовать для масштабирования внутренней геометрии файла SAT до этой системы единиц измерения.

В примере ниже геометрия импортируется из файла SAT с футами в качестве единицы измерения. Чтобы этот файл примера можно было использовать для работы на компьютере, скачайте этот файл примера SAT и укажите в узле `File Path` недопустимый файл SAT.
