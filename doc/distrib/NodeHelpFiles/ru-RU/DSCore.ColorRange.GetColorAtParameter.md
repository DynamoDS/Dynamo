## Подробности
GetColorAtParameter принимает входной цветовой 2D-диапазон и возвращает список цветов с заданными параметрами UV в диапазоне от 0 до 1. В примере ниже сначала создается цветовой 2D-диапазон с помощью узла ByColorsAndParameters, в который передается список цветов и список параметров для задания диапазона. В блоке кода генерируется диапазон чисел от 0 до 1, который используется в качестве входных значений u и v для узла UV.ByCoordinates. Для параметра переплетения этого узла задается векторное произведение. Аналогичным образом создается набор кубов, который используется узлом Point.ByCoordinates с переплетением типа «Векторное произведение» для создания массива кубов. Затем используется узел Display.ByGeometryColor с массивом кубов и списком цветов, полученным от узла GetColorAtParameter.
