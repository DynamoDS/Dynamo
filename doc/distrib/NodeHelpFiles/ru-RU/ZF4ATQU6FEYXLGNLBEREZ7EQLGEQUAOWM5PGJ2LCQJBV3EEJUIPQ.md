## Подробности
`CoordinateSystem.Rotate (coordinateSystem, plane, degrees)` возвращает новый объект CoordinateSystem, повернутый вокруг плоскости на заданное число градусов.

В примере ниже система координат повернута вокруг базовой плоскости YZ, заданной в точке вставки 0, 0, 0 на 135 градусов.
