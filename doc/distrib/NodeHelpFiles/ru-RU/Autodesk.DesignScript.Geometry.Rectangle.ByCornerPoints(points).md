## Подробности
Узел Rectangle.ByCornerPoints принимает список из четырех входных точек и создает прямоугольник на их основе. Входные точки должны отражать форму прямоугольника. В данном примере для создания точек по координатам в вершинах прямоугольника используются два числовых регулятора. Похожий по функциям узел Rectangle.ByCornerPoints принимает в качестве входных элементов четыре отдельные точки, а не один список. В данном примере демонстрируются оба узла.
