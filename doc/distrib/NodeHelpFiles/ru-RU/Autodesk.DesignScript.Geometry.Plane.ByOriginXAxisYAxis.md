## Подробности
Plane.ByOriginXAxisYAxis использует точку начала координат и два вектора для определения плоскости. Вектор нормали плоскости вычисляется как векторное произведение векторов осей X и Y. В примере ниже YAxis МСК используется в качестве xAxis, а точка начала координат и входной вектор xAxis задаются в двух блоках кода.
