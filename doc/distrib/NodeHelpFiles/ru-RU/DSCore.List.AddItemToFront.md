## Подробности
`List.AddItemToFront` вставляет заданный элемент в начало заданного списка. Индекс нового элемента равен 0, а индексы всех исходных элементов сдвигаются на 1. Обратите внимание, что если добавляемый элемент является списком, он добавляется как единый объект, образуя вложенный список. Сведения об объединении двух списков в один неструктурированный список см. в разделе `List.Join`.

В примере ниже с помощью блока кода создается диапазон чисел от 0 до 5 с шагом 1. Затем в начало списка с помощью `List.AddItemToFront` добавляется новый элемент, число 20.
