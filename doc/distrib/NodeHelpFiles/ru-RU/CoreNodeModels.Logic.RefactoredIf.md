## Подробности
Узел If играет роль узла управления на основе условий. Входной элемент test принимает логическое значение, а входные элементы true и false могут принимать данные любого типа. Если значение test равно True, узел возвращает элемент из входных данных true. В противном случае узел возвращает элемент из входных данных false. В примере ниже сначала создается список случайных чисел от нуля до 99. Количество элементов в списке задается с помощью целочисленного регулятора. Блок кода с формулой x%a==0 используется для проверки делимости на второе число, которое задается с помощью второго численного регулятора. В результате генерируется список логических значений, соответствующих делимости элементов случайного списка на число, заданное с помощью второго целочисленного регулятора. Этот список логических значений используется в качестве входного элемента test для узла If. В качестве входного элемента true используется стандартная сфера, а в качестве входного элемента false — стандартный кубоид. Результат работы узла If представляет собой список сфер или кубоидов. Наконец, узел Translate используется для разделения списка объектов геометрии.

Узел IF реплицируется на всех узлах так, КАК ЕСЛИ БЫ БЫЛО ЗАДАНО ЗНАЧЕНИЕ SHORTEST. Причина этого очевидна из приложенных примеров, особенно если взглянуть на результаты, когда к узлу формулы применяется значение LONGEST и передается «короткая» ветвь условия. Кроме того, эти изменения были внесены для обеспечения прогнозируемого поведения при использовании отдельных входных логических значений или списка логических значений.
