## Подробности
`List.MaximumItem` выполняет поиск по списку значений и возвращает элемент с наибольшим значением.

В примере ниже создается список из 10 случайных чисел в диапазоне от 0 до 49. Затем с помощью узла `List.MaximumItem` находится самый большой элемент в списке.
