## In-Depth
Узел `TSplineUVNFrame.Position` возвращает положение кадра UVN как объект-точку. Этот узел можно использовать для визуализации положения вершины или грани поверхности при выборе этих элементов по индексу.
В приведенном ниже примере этот узел используется для визуализации направления каждой обычной грани на поверхности.
