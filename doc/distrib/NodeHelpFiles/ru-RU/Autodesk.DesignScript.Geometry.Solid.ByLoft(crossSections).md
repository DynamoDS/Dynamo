## Подробности
В качестве входных данных в узле `Solid.ByLoft (crossSections)` используется список замкнутых кривых. Тело формируется путем последовательного лофтинга между элементами в списке кривых.

В примере ниже сначала создается окружность. Затем создается серия преобразованных окружностей на основе синусоиды. Этот список кривых используется как входной параметр для узла `Solid.ByLoft`.
