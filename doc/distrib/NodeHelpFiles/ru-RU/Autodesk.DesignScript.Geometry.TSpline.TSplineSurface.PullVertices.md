## Подробности
В приведенном ниже примере узел `TSplineTopology.InnerVertices`извлекает все внутренние вершины плоской Т-сплайновой поверхности. Вершины и поверхность, к которой они относятся, используются в качестве входных данных для узла `TSplineSurface.PullVertices`. Входной параметр `geometry` представляет собой сферу, расположенную над плоской поверхностью. Для входного параметра `surfacePoints` задается значение False, а для выполнения операции вытягивания используются управляющие точки.
