## Подробности

Функция «Круговая диаграмма» создает диаграмму в виде круга, разделенного на сегменты. Для каждого сегмента можно назначить метку, цвет и значения.

Для добавления меток сегментов укажите список строковых значений в поле входных данных. По умолчанию метки имеют значения Item1, Item2 и Item3. Метки отображаются под круговой диаграммой.

Задайте значения для каждого сегмента, указав список двойных значений в поле входных данных. Обратите внимание на то, что количество значений должно быть равно количеству меток, указанных в поле ввода меток. Значения могут превышать 100.

Группам можно присвоить цвета, указав список цветов в соответствующем поле. Если пользователь задает свои цвета, количество цветов должно соответствовать количеству строковых значений в метках. Если не указать цвета, каждой группе будет назначен случайный цвет.
