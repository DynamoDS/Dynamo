## Подробности
NurbsCurve.ByPoints принимает список точек для построения NURBS-кривой, а также позволяет замкнуть кривую. В данном примере на плоскости XY МСК генерируется четыре случайные точки, которые соединяются по порядку в направлениях X и Y, замыкая кривую в первой точке из списка.
