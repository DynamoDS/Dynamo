## Подробности

Функция «Линейный график XY» создает диаграмму с одной или несколькими линиями, расположение которых определяется значениями по осям X и Y. Помечайте линии или меняйте количество линий, вставляя список строковых значений в поле ввода меток. Каждая метка создает соответствующую цветовую линию. Если ввести только одно строковое значение, будет создана только одна линия.

Чтобы определить расположение каждой точки вдоль каждой линии, используйте список списков, где содержатся двойные значения для точек по осям X и Y. Для осей X и Y должно быть указано одинаковое количество значений. Количество вложенных списков должно также совпадать с количеством строковых значений в поле ввода меток.
Например, если требуется создать три линии по пять точек в каждой, предоставьте список с тремя строковыми значениями в поле ввода меток, чтобы обозначить каждую линию, и укажите три вложенных списка с пятью двойными значениями в каждом для координат по осям X и Y.

Линиям можно присвоить цвета, указав список цветов в соответствующем поле. Если пользователь задает свои цвета, количество цветов должно соответствовать количеству строковых значений в метках. Если не указать цвета, каждой линии будет назначен случайный цвет.
