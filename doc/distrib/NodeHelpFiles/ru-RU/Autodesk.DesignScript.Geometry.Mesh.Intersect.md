## Подробности
Узел `Mesh.Intersect` пересекает сеть с входной плоскостью, возвращая одну или несколько сложных кривых.

В приведенном ниже примере `Mesh.Intersect` используется для сети в форме кролика. Сеть пересекается с серией входных плоскостей для создания срезов в виде контуров. Это отправная точка для подготовки модели к использованию со станком для гидроабразивной или лазерной резки или фрезерным станком с ЧПУ.
