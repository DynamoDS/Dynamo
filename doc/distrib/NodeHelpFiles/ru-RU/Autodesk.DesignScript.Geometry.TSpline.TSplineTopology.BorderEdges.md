## Подробности
Узел `TSplineTopology.BorderEdges` возвращает список ребер границ, содержащихся в Т-сплайновой поверхности.

В приведенном ниже примере создаются две Т-сплайновые поверхности с помощью узла `TSplineSurface.ByCylinderPointsRadius`. Одна из поверхностей является незамкнутой, а другая утолщается с помощью узла `TSplineSurface.Thicken`, который превращает ее в замкнутую. Обе поверхности исследуются с помощью узла `TSplineTopology.BorderEdges`, после чего для первой из них возвращается список ребер границ, а для второй — пустой список. Это связано с тем, что у второй поверхности нет ребер границ, так как она замкнута.
