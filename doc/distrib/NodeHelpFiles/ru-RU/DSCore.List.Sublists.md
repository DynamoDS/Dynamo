## Подробности
`List.Sublists` возвращает ряд вложенных списков из данного списка на основе диапазона входных значений и смещения. Диапазон определяет элементы входного списка, которые помещаются в первый вложенный список. К диапазону применяется смещение, и новый диапазон определяет второй вложенный список. Этот процесс повторяется, увеличивая начальный индекс диапазона на заданное смещение, пока вложенный список не останется пустым.

В примере ниже сначала создается диапазон чисел от 0 до 9. Диапазон от 0 до 5 используется как диапазон вложенного списка со смещением 2. В получившихся вложенных списках первый список содержит элементы с индексами в диапазоне 0..5, а второй список содержит элементы с индексами 2..7. При повторении следующий вложенный список становится короче, так как конец диапазона превышает длину исходного списка.
