## Подробности
NurbsCurve.ByControlPoints принимает список контрольных точек для построения NURBS-кривой, а также позволяет управлять степенью кривой. Кроме того, кривую можно замкнуть в первой точке из списка. В данном примере на плоскости XY МСК генерируются четыре случайные точки, которые соединяются по порядку в направлениях X и Y.
