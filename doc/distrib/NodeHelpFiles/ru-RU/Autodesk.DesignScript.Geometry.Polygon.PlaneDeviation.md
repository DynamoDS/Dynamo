## Подробности
PlaneDeviation сначала вычисляет оптимально вписанную плоскость по точкам заданного многоугольника. Затем вычисляется расстояние от каждой точки до этой плоскости для нахождения максимального отклонения точек от оптимально вписанной плоскости. В примере ниже создается список случайных углов, отметок и радиусов, а затем с помощью узла Points.ByCylindricalCoordinates создается набор неплоских точек, которые будут использоваться в узле Polygon.ByPoints. Использование этого многоугольника в качестве входного элемента PlaneDeviation позволяет найти среднее отклонение точек от плоскости наилучшего вписывания.
