## Подробности
TotalMilliseconds возвращает количество миллисекунд в значении TimeSpan в виде числа двойной точности. В примере ниже общее количество миллисекунд, возвращенное на основе значения TimeSpan, равного 2.03:00:00, составляет 185 400 000.
