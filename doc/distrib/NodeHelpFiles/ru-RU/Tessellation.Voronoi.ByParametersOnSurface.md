## Подробности
ByParametersOnSurface возвращает представление Вороного для UV поверхности в виде списка кривых. В примере ниже представление Вороного создается на поверхности с использованием системы UV из 50 случайных значений. Чтобы представление Вороного остановилось на кромках поверхности, полученные кривые должны пересекаться с поверхностью.
