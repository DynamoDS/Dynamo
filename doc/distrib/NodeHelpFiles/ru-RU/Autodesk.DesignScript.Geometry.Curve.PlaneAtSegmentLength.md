## Подробности
PlaneAtSegmentLength возвращает плоскость, совмещенную с кривой в точке, находящейся на заданном расстоянии от начальной точки кривой. Если входная длина больше общей длины кривой, этот узел будет использовать конечную точку кривой. Вектор нормали результирующей плоскости будет соответствовать касательной кривой. В примере ниже сначала с помощью узла ByControlPoints создается NURBS-кривая, где в качестве входных элементов используется набор случайных точек. Для управления входным элементом parameter узла PlaneAtSegmentLength используется числовой регулятор.
