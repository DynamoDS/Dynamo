## Подробности
Узел Rectangle.ByCornerPoints принимает четыре входные точки и создает прямоугольник на их основе. Входные точки должны отражать форму прямоугольника. В данном примере для создания точек по координатам в вершинах прямоугольника используются два числовых регулятора. Похожий по функциям узел Rectangle.ByCornerPoints принимает в качестве входного элемента список точек, а не входные точки по отдельности. В данном примере демонстрируются оба узла.
