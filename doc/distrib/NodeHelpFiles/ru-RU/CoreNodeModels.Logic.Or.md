## Подробности
Or принимает переменное количество входных логических значений. Чтобы увеличить или уменьшить количество входных значений, нажимайте кнопки + и - в узле Or. Каждый входной элемент должен представлять собой отдельное логическое значение. Or возвращает значение True, если хотя бы один входной элемент имеет значение True. В противном случае этот узел возвращает значение False. В примере ниже в качестве входных элементов для узла Or используются три переключателя логических значений. Если все три переключателя установлены в положение False, возвращается значение False.
