## Подробности
`List.AddItemtoEnd` вставляет заданный элемент в конец заданного списка. Обратите внимание, что если в качестве добавляемого элемента указан объект списка, он добавляется как единый объект, образуя вложенный список. Сведения об объединении двух списков в один неструктурированный список см. в разделе `List.Join`.

В примере ниже с помощью блока кода создается диапазон чисел от 0 до 5 с шагом 1. Затем в конец этого списка с помощью `List.AddItemToEnd` добавляется новый элемент, число 20.
