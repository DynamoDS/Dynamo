## Подробности
В приведенном ниже примере выбирается отдельная грань Т-сплайновой поверхности рамки с помощью узла `TSplineTopology.FaceByIndex`. Затем эта грань выдавливается по заданной кривой (`curve`) с помощью узла `TSplineSurface.ExtrudeFacesAlongCurve`. В результате получается форма с четырьмя выдавленными гранями, что объясняется исходной симметрией, примененной к форме.
