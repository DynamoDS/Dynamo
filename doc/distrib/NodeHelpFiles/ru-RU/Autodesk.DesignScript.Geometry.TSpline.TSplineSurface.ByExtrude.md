## In-Depth
В приведенном ниже примере Т-сплайновая поверхность создается как выдавливание заданной кривой (`curve`) профиля. Кривая может быть замкнутой или незамкнутой. Выдавливание выполняется в заданном направлении (`direction`), которое может быть любым из двух и управляется входными параметрами `frontDistance` и `backDistance`. Пролеты можно задавать отдельно для каждого из двух направлений выдавливания с помощью заданных значений `frontSpans` и `backSpans`. Чтобы определить поверхность вдоль кривой, используйте параметр `profileSpans` для управления количеством граней и параметр `uniform` для их равномерного распределения либо учета кривизны. Параметр `inSmoothMode` управляет отображением поверхности: в режиме сглаживания или рамки.
