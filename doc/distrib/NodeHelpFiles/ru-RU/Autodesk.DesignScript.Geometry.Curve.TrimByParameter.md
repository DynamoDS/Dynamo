## Подробности
TrimByParameter позволяет удалить начало и конец входной кривой путем ее обрезки с заданными параметрам. Этот узел возвращает результирующий средний участок кривой. В примере ниже сначала с помощью узла ByControlPoints создается NURBS-кривая, где в качестве входных элементов используется набор случайных точек. Для управления начальным и конечным входными элементами parameter узла TrimByParameter используются два числовых регулятора с диапазоном значений от 0 до 1.
