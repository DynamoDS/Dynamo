## Подробности
`List.IsRectangular` возвращает логическое значение, которое указывает, содержат ли строки списка одинаковое количество элементов.

В примере ниже для сетки точек с добавленной в строку дополнительной точкой возвращается значение False. Из коллекции точек неправильной формы можно создать объект NurbsSurface, но его поведение может отличаться от ожидаемого.
