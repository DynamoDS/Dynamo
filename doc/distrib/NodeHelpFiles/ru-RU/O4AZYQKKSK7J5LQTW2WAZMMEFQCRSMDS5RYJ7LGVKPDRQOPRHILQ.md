## In-Depth
В приведенном ниже примере создается конус с основанием в начале системы координат, которая определяется входным параметром `cs`. Размер конуса определяется с помощью входных параметров `height`, `startRadius` и `endRadius`. Вертикальные и радиальные пролеты задаются параметрами `radiusSpans` и `heightSpans`. Исходная симметрия формы определяется входным параметром `symmetry`. Если для симметричности по оси X или Y задано значение True, количество радиальных пролетов должно быть кратно 4. Входной параметр `inSmoothMode` используется для переключения между режимами сглаживания и рамки при предварительном просмотре T-сплайновой поверхности.
