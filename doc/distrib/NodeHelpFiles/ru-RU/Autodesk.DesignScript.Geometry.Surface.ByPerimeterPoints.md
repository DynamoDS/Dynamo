## Подробности
Surface.ByPerimeterPoints принимает список точек, определяющих периметр замкнутого многоугольника, и создает поверхность внутри этого многоугольника. В примере ниже создается последовательность точек по цилиндрическим координатам. Точки размещаются через равные промежутки вокруг цилиндра со случайными входными значениями радиуса и отметки. Результирующий список точек используется в качестве входного элемента для узла Surface.ByPerimeterPoints.
