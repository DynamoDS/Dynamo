## Подробности
`List.Transpose` меняет местами строки и столбцы в списке списков. Например, список, содержащий 5 вложенных списков по 10 элементов каждый, будет перенесен в 10 списков по 5 элементов каждый. Значения NULL вставляются по мере необходимости, чтобы обеспечить наличие одинакового количества элементов в каждом вложенном списке.

В данном примере создается список чисел от 0 до 5 и список букв от A до E. Затем для их объединения используется `List.Create`. `List.Transpose` создает 6 списков по 2 позиции, по одному числу и одной букве на список. Обратите внимание, что поскольку один из исходных списков длиннее, `List.Transpose` вставляет значение NULL для непарного элемента.
