## Подробности
`Solid.ByLoft (crossSections, guideCurves)` позволяет ввести список направляющих кривых для создания различных профилей по периметру лофт-тела.

В примере ниже в качестве объекта `crossSections` используются два прямоугольника со смещением. По периметру создаются две различные направляющие кривые: одна на основе косинусной функции, а другая — простая линия.
