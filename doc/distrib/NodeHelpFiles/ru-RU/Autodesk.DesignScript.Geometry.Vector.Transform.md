## Подробности
Transform возвращает новый вектор на основе входного объекта CoordinateSystem. В примере ниже вектор преобразуется из объекта WorldCoordinateSystem в новый объект CoordinateSystem, заданный сферическими координатами. Настраивая регуляторы, вектор можно перенести в новое местоположение. Оба вектора представлены в виде линий.
