## Подробности
Этот узел подсчитывает количество ребер в сети. Если сеть состоит из треугольников, как сети в `MeshToolkit`, узел `Mesh.EdgeCount` возвращает только уникальные ребра, поэтому количество ребер не будет в три раза превышать количество треугольников в сети. Это допущение можно использовать для проверки того, не содержит ли сеть разъединенные грани (такое может случиться в импортированных сетях).

В приведенном ниже примере с помощью узлов `Mesh.Cone` и `Number.Slider` конус создается и используется в качестве входных данных для подсчета ребер. Для предварительного просмотра структуры и сетки сети можно использовать как узел `Mesh.Edges`, так и `Mesh.Triangles`. При этом для сложных и тяжелых сетей лучше использовать узел `Mesh.Edges`, так как у него выше производительность.
