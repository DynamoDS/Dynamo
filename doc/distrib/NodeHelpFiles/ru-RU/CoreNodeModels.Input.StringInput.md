## Подробности
Узел String позволяет ввести строковое значение. Строку также можно создать в Code Block, заключив ее в кавычки. В примере ниже для создания строк Hello и World используются узлы String и Code Block. Для объединения этих строк используется узел String.Join. Во втором узле String вводится пробел, служащий разделителем. В результате выводится строка Hello World.
