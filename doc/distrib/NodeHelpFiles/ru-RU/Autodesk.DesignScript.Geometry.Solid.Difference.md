## Подробности
Difference позволяет создать новое тело путем вычитания одного тела из другого. Входной элемент solid обозначает тело, из которого выполняется вычитание, а входной элемент tool — вычитаемое тело. В примере ниже в качестве тела, из которого выполняется вычитание, используется стандартный куб. Числовые регуляторы позволяют управлять положением и радиусом сферы, используемой в качестве инструмента. Если сфера пересекает куб, то результат представляет собой куб, из которого вычтена часть, пересекающаяся со сферой.
