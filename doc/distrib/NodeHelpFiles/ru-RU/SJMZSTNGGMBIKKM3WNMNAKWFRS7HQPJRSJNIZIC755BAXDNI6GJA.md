## Подробности
`Geometry.Scale (xamount, yamount, zamount)` масштабирует входную геометрию из центроида объекта на входной коэффициент.

В примере ниже масштаб конуса можно настроить с помощью трех регуляторов чисел. Исходные кромки конуса отрисовываются для визуального ориентира.
