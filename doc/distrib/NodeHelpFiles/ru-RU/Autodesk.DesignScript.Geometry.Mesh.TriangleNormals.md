## Подробности
В приведенном ниже примере `Mesh.TriangleNormals` используется для импортированной сети, а узел `Line.ByStartPointDirectionLength` — для визуализации векторов нормали. Узел `Mesh.TriangleCentroids` облегчает размещение линий в центре каждого треугольника.
