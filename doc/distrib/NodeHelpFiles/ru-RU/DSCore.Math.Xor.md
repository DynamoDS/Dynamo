## Подробности
`Math.Xor` принимает два логических значения или два списка логических значений и определяет, являются ли значения одинаковыми. Узел `Math.Xor` сравнивает входные значения и возвращает True, если только одно из значений равно True. Если оба значения равны True или оба значения равны False, `Math.Xor` возвращает False.

В приведенном ниже примере сначала создаются два списка логических значений. Вместе эти списки представляют 4 возможные комбинации значений True и False. Обратите внимание, что в отличие от узлов `And` и `Or` узел `Math.Xor` допускает только два значения.
