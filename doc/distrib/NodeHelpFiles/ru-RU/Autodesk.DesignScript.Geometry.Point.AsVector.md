## Подробности
Point.AsVector принимает точку в качестве входного элемента, а возвращает вектор с компонентами x, y и z, равными декартовым координатам точки. Это также можно рассматривать как создание вектора по двум точкам, где первая точка — начало координат МСК, а вторая — входная точка. В примере ниже точка создается с помощью узла PointBySphericalCoordinates, после чего узел AsVector используется для получения вектора, которым можно управлять с помощью сферических координат.
