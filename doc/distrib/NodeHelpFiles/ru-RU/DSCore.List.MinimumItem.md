## Подробности
`List.MinimumItem` выполняет поиск по списку значений и возвращает элемент с наименьшим значением.

В примере ниже создается список из 10 случайных чисел в диапазоне от 0 до 49. Затем с помощью узла `MinimumItem` выполняется поиск наименьшего элемента в списке.
