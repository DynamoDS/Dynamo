## Подробности
Math.Random создает одно число от 0 до 1. Этот узел включает входное значение seed. Seed — это значение, которое инициирует рандомизированную функцию (позволяя использовать дополнительные параметры для каждого входного значения seed) и позволяет «вызывать» случайные значения с использованием того же значения для seed.
