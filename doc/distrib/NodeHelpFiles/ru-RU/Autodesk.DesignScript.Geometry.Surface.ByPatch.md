## Подробности
Surface.ByPatch позволяет создать поверхность путем заполнения пространства, ограниченного замкнутой кривой. В примере ниже сначала создается замкнутая NURBS-кривая с помощью последовательности точек, созданных по цилиндрическим координатам. Числовой регулятор определяет количество создаваемых точек, а логический переключатель — то, является ли NURBS-кривая замкнутой. NURBS-кривая используется в качестве входного элемента для узла SurfaceByPatch, в результате чего создается поверхность внутри замкнутой кривой.
