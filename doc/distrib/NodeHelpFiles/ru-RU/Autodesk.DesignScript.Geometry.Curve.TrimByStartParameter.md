## Подробности
TrimByStartParameter позволяет удалить начало входной кривой путем ее обрезки с заданным параметром. В примере ниже сначала с помощью узла ByControlPoints создается NURBS-кривая, где в качестве входных элементов используется набор случайных точек. Для управления входным элементом parameter узла TrimByStartParameter используется числовой регулятор с диапазоном значений от 0 до 1.
