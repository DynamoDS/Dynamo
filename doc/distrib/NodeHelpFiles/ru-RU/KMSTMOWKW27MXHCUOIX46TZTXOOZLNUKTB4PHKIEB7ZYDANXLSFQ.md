## Подробности
ByLoft(crossSections, guideCurve) возвращает новый объект PolySurface, полученный путем лофтинга между кривыми в списке и направляющей кривой. В примере ниже возвращается новый объект PolySurface, созданный путем лофтинга между двумя окружностями. Возвращенный объект PolySurface деформируется путем корректировки дуги входного элемента guideCurve.
