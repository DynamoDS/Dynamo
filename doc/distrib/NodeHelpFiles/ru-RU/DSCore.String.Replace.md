## Подробности
Replace позволяет найти в заданной строке вторую входную строку. При обнаружении эта строка заменяется на третью входную строку. Этот узел чувствителен к регистру. В примере ниже используется строка Hello World. В качестве искомой строки используется строка «o», а в качестве строки для замены — строка «Hi!». Так как буква «o» встречается в исходной строке дважды, оба вхождения заменяется строкой «Hi!».
