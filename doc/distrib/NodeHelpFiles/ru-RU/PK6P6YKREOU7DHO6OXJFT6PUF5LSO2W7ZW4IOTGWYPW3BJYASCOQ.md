## In-Depth
Узел `TSplineInitialSymmetry.ByRadial` определяет, имеет ли геометрия Т-сплайна радиальную симметрию. Ее можно задать только для Т-сплайновых примитивов, допускающих ее применение: конуса, сферы, вращения, тора. Радиальная симметрия, единожды определенная при создании геометрии T-сплайна, влияет на все последующие операции и изменения.

Для применения симметрии необходимо определить требуемое количество симметричных граней (`symmetricFaces`), где минимальное значение — 1. Независимо от начального количества радиальных и вертикальных пролетов, T-сплайновая поверхность дальше разделяется на выбранное количество симметричных граней (`symmetricFaces`).

В приведенном ниже примере создается объект `TSplineSurface.ByConePointsRadii`, к которому применяется радиальная симметрия с помощью узла `TSplineInitialSymmetry.ByRadial`. Затем узлы `TSplineTopology.RegularFaces` и `TSplineSurface.ExtrudeFaces` используются для выбора и выдавливания грани Т-сплайновой поверхности соответственно. Выдавливание применяется симметрично, а регулятор количества симметричных граней демонстрирует подразделение радиальных пролетов.
