## Подробности
`Mesh.TriangleCount` вычисляет количество треугольников в сети. В приведенном ниже примере `Mesh.Cone` и `Number.Slider` используются для создания конуса, который затем используется в качестве входных данных для подсчета треугольников. Помимо этого, с помощью `Mesh.Triangles` выделяются треугольники сети для удобства предварительного просмотра.
