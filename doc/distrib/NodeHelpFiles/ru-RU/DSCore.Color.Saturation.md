## Подробности
Saturation возвращает значение насыщенности для входного цвета в диапазоне от 0 до 1. В примере ниже с помощью узлов ByARGB создаются два цвета, которые затем объединяются. Значение насыщенности для результирующего цвета можно найти с помощью узла Saturation.
