## Подробности
`Solid.Repair` пытается восстановить тела с недопустимой геометрией, а также оптимизировать их. `Solid.Repair` возвращает новое тело.

Этот узел используется для устранения ошибок, возникших при выполнении операций с импортированными или преобразованными объектами геометрии.

В приведенном ниже примере узел `Solid.Repair` используется для восстановления геометрии из файла **SAT**. Не удается вывести логическое значение или выполнить обрезку геометрии в файле, и узел `Solid.Repair` удаляет все *недопустимые объекты геометрии*, являющиеся причиной сбоя.

Обычно не требуется использовать эту функцию для объектов геометрии, создаваемых в Dynamo; она нужна только для объектов геометрии из внешних источников. В противном случае сообщите об ошибке команде разработчиков Dynamo на Github
