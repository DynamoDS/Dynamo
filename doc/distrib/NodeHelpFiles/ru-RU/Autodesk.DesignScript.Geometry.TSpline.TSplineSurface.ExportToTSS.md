## Подробности
In the example below, a T-Spline surface is created using `TSplineSurface.ByRevolve`, with a NURBS curve as a profile.
При наличии пути к файлу поверхность экспортируется в файл T-Spline Scene с помощью узла `TSplineSurface.ExportToTSS`.
