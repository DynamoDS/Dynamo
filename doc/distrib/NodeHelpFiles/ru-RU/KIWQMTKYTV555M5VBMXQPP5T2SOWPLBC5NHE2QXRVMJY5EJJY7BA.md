## Подробности
`Vector.Rotate (vector, plane, degrees)` поворачивает вектор вокруг начала координат плоскости и нормали на заданное количество градусов.

В приведенном ниже примере вектор поворачивается вокруг плоскости XY на 90 градусов и визуализируется в виде линии.
