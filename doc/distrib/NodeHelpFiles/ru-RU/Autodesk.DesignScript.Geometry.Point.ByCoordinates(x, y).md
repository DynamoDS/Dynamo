## Подробности
Point.ByCoordinates(XY) принимает в качестве входных элементов два числа и создает точку на базовой плоскости Dynamo. В первом примере два регулятора используются для динамического управления положением X и Y точки на плоскости Z. Во втором — два числовых регулятора и диапазон в Code Block используются для задания максимальных границ сетки и интервала между точками. Путем изменения значения параметра переплетения в узле Points.ByCoordinates на перекрестную ссылку можно создать многомерный набор точек вдоль осей X и Y.
