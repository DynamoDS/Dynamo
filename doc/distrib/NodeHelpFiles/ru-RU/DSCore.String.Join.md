## Подробности
Join позволяет сжать несколько строк в одну, вставляя между ними заданную разделительную строку. В примере ниже в качестве разделителя используется символ подчеркивания. Создаются три строки, которые требуется объединить. Количество входных строк, которые принимает этот узел, можно изменять с помощью кнопок + и - узла.
