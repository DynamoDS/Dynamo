## Подробности
Plane.ByOriginNormal позволяет создать плоскость с началом координат во входной точке и перпендикулярно входному вектору нормали. В примере ниже в блоке кода задаются координаты x, y и z для узла Point.ByCoordinates. Затем эта точка используется в качестве начала координат, а ось Y МСК — в качестве вектора нормали для узла Plane.ByOriginNormal.
