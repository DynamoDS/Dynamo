## Подробности
`List.IsEmpty` возвращает логическое значение True, если входной список пуст, и значение False, если он не пуст.

В примере ниже сначала с помощью блока кода создается диапазон чисел от 0 до 5 с шагом 1. Этот список передается на вход узлу `List.IsEmpty`. Поскольку список не пуст, `List.IsEmpty` возвращает значение False.
