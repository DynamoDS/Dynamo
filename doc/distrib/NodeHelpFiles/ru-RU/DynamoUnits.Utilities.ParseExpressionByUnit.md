## Подробности
Utilities.ParseExpressionByUnit считывает заданную строку (с обозначениями единиц измерения) и возвращает число, представляющее входное значение с выбранными единицами измерения. Примечание. Можно использовать любые комбинации типов единиц измерения при условии их ввода с правильным обозначением.
