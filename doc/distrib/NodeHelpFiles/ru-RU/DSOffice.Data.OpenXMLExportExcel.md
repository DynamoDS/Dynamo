## Подробности
Data.OpenXMLExportExcel позволяет записать данные в файл Microsoft Excel с использованием стандарта OpenXML. Эта версия узлов Excel не требует наличия приложения Excel на компьютере. Входные параметры overWrite и writeAsString являются необязательными и имеют значения по умолчанию. Параметр overWrite полностью заменяет содержимое файла Excel, а параметр writeAsString представляет все данные в виде строк.
