## In-Depth
В приведенном ниже примере создается форма с разогнутыми ребрами с помощью узла `TSplineSurface.BuildFromLines`. Для входного параметра `creaseOuterVertices` задается значение False. Затем вершина с индексом 1 сгибается, и форма смещается вправо для удобства просмотра.
