## Подробности
WriteText позволяет записать строку в заданный файл. Если файл не существует, этот узел его создает. Для создания новых строк в выходном файле можно использовать escape-символ \r\n. В примере ниже приведен список строк, которые требуется записать в текстовый файл как три отдельные строки. Этот список объединяется в одну строку с использованием символа \r\n в качестве разделителя. Затем узел WriteText используется для записи этой строки в текстовый файл.
