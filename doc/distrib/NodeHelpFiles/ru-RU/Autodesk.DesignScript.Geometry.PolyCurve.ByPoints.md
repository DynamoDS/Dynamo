## Подробности
`PolyCurve.ByPoints` создает объединенные кривые из набора вершин. Этот узел также позволяет переключаться между замкнутой и незамкнутой формой с помощью входного значения `connectLastToFirst`.

В примере ниже точки создаются вдоль окружности и перерисовываются с помощью узла `PolyCurve.ByPoints`, создавая замкнутую многоугольную форму.
