## In-Depth
Этот узел возвращает объект TSplineUVNFrame, который может пригодиться для визуализации положения и ориентации вершины, а также для дальнейшего манипулирования Т-сплайновой поверхностью с помощью векторов U, V или N.

В приведенном ниже примере узел `TSplineVertex.UVNFrame` используется для получения кадра UVN выбранной вершины. Затем кадр UVN возвращает нормаль к вершине. Направление нормали используется для перемещения вершины с помощью узла `TSplineSurface.MoveVertices`.
