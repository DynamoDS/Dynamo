## Подробности
`Geometry.ExportToSAT` экспортирует геометрию из Dynamo в файл SAT. Этот узел не имеет единиц измерения, для задания единиц измерения используйте узел `ExportToSAT`.

В примере ниже объект PolySurface из пересечения трех сфер можно экспортировать в файл SAT, когда выбран путь к файлу.
