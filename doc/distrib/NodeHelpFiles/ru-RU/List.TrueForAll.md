## Подробности
`List.TrueForAll` возвращает логическое значение, которое указывает, выполняется ли условие во входном параметре `queryFunction` (True) для всех элементов в списке. Параметр `queryFunction` должен являться логическим значением.

В примере ниже показаны три функции, используемые для обработки входного списка. В первой используется узел «больше чем» (>) для определения того, являются ли все значения в списке больше 11. Возвращается значение False. Во второй используется узел «меньше чем» (<) для определения того, являются ли все значения в списке меньше 11. Возвращается значение True. В третьей используется узел «меньше чем» (<) для определения того, являются ли все значения в списке меньше 5. Возвращается значение False.
