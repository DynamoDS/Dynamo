## Подробности
Knots возвращает узлы NURBS-кривой в виде списка чисел. В этом примере сначала генерируется несколько случайных точек, а затем с помощью узла NurbsCurve.ByPoints создается кривая, интерполированная по точкам. С помощью узла Knots можно получить список узлов для этой кривой.
