## Подробности
`List.DiagonalRight` возвращает новый список, который содержит списки элементов из каждой диагонали в матрице списков от правого нижнего угла к левому верхнему.

В примере ниже создается сетка точек, через которые по диагонали проходят линии. `List.DiagonalRight` возвращает список точек по диагонали от нижнего правого угла сетки к верхнему.
