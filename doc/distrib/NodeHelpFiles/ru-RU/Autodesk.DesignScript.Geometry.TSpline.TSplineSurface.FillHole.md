## Подробности
В приведенном ниже примере зазоры в Т-сплайновой поверхности цилиндра заполняются с помощью узла `TSplineSurface.FillHole`, для которого необходимо задать следующие входные параметры:
— `edges`: количество ребер границы, выбранных из Т-сплайновой поверхности, которую требуется заполнить.
— `fillMethod`: числовое значение от 0 до 3, которое указывает на метод заполнения.
    * 0 означает, что для заполнения отверстия используется метод тесселяции.
    * 1 заполняет отверстие одной гранью NGon.
    * 2 создает точку в центре отверстия, из которой исходят треугольные грани в направлении ребер.
    * 3 аналогичен методу 2, только центральные вершины объединяются в одну вершину, а не просто накладываются друг на друга.
— `keepSubdCreases`: логическое значение, указывающее на то, сохраняются ли сгибы Subd.
