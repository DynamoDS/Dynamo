## Подробности

Функция «Тепловая карта» создает диаграмму, в которой точки данных представлены в виде прямоугольников, окрашенных разными оттенками цветов заданного диапазона.

Чтобы назначить метки для каждой строки и столбца, укажите список строковых меток в полях входных данных меток по осям X и Y соответственно. Количество меток по осям X и Y может быть разным.

Укажите значение для прямоугольников в поле входных значений. Количество вложенных списков должно соответствовать количеству строковых значений для меток по оси X, так как оно представляет количество столбцов. Значение в каждом вложенном списке представляет количество прямоугольников в каждом столбце. Например, если добавлено четыре списка, создается четыре столбца. Если в каждом списке пять значений, каждый столбец состоит из пяти прямоугольников.

Еще один пример: чтобы создать сетку из пяти строк и пяти столбцов, укажите пять строковых значений для меток по оси X и меток по оси Y. Метки по оси X будут отображаться под диаграммой, по оси X, а метки по оси Y будут отображаться слева от диаграммы, по оси Y.

В поле ввода значений введите список списков. В каждом вложенном списке должно быть пять значений. Значения отрисовываются в столбцах слева направо и снизу вверх, так что первое значение в первом вложенном списке представляет нижний прямоугольник в левом столбце, второе значение — прямоугольник над ним и т. д. Каждый вложенный список представляет столбец в графике.

Чтобы различать точки данных, можно задать для них диапазон цветов, указав список значений цветов в поле входных данных. Самое маленькое значение в таблице будет первым цветом, самое большое — последним, а остальные значения будут находиться между ними. Если диапазон цветов не назначен, точки данных будут окрашены случайным образом от самого светлого к самому темному оттенку.

Для получения оптимальных результатов рекомендуется использовать один или два цвета. В файле примера представлен классический пример с двумя цветами: синим и красным. Если в качестве цветов указаны эти два цвета, в окне «Тепловая карта» автоматически создается градиент между этими цветами, причем малые значения будут иметь оттенки синего, а большие — оттенки красного.
