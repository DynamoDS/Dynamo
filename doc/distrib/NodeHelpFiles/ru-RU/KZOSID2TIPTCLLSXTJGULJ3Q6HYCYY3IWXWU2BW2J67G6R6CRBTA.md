## Подробности
Узел «||» является оператором «Или». Он принимает два логических значения в качестве входных данных и возвращает значение «Истина», если одно или оба значения имеют значение «Истина». Если оба значения имеют значение «Ложь», возвращается значение «Ложь». В примере ниже три узла «||» используются для определения того, имеется ли во входных логических значениях хотя бы одно значение «Истина». В первом узле имеется и значение «Истина», и значение «Ложь», поэтому возвращается значение «Истина». Во втором узле имеется два значения «Истина», поэтому возвращается значение «Истина». В третьем узле имеется два значения «Ложь», поэтому возвращается значение «Ложь».
