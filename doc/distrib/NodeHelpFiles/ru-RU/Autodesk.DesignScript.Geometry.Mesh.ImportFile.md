## Подробности
`Mesh.ImportFile` извлекает имя файла из указанного пути к нему и использует его в качестве входных данных. Файл может быть следующих типов:
- .obj — файл Wavefront OBJ;
- .mix — файл Meshmixer;
- .stl — файл STL;
- .ply — файл формата Polygon.
