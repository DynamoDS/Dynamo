## Подробности
PerimeterCurves возвращает кривые на кромках незамкнутой поверхности в виде списка кривых. В примере ниже сначала создается поверхность с помощью узла BySweep2Rails. Затем с помощью узла PerimeterCurves извлекаются кромки периметра.
