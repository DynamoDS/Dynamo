## Подробности
IsParallel возвращает логическое значение, показывающее, параллелен ли вектор другому вектору. В примере ниже сравниваются два вектора, представленные в виде линий. Перемещайте регулятор, чтобы повернуть один из векторов параллельно другому и под углом к нему.
