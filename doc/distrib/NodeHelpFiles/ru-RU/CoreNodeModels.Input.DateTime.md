## Подробности
DateTime позволяет создать объекта даты и времени на основе указанных входных значений месяца, дня, года и времени. Формат должен быть следующим: January 01, 1900 12:00 AM. Месяц пишется с прописной буквы и не сокращается. День должен состоять из двух цифр, год — из четырех.
