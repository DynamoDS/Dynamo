## Подробности
ApproximateWithTolerance позволяет создать NURBS-поверхность, аппроксимирующую входную поверхность. Входное значение допуска определяет точность соответствия результата исходной поверхности. В примере ниже поверхность создается с помощью узла ByPatch, в качестве входного элемента которого используется ближайшая NURBS-кривая. Обратите внимание, что при использовании этой поверхности в качестве входного элемента для узла ApproximateWithTolerance результат представляет собой необрезанную NURBS-поверхность с четырьмя сторонами.
