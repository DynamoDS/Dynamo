## Подробности
В приведенном ниже примере две вершины Т-сплайновой поверхности рамки объединяются с помощью узла `TSplineSurface.WeldVertices`. Новое положение объединенных вершин используется в качестве значения входного параметра `newPosition`. Кроме того, входной параметр `keepSubdCreases` определяет, сгибаются ли ребра вдоль соединения.
