## Подробности
StartsWith возвращает логическое значение, показывающее, начинается ли заданная строка со второй входной строки. По умолчанию этот узел чувствителен к регистру. Для входного параметра ignoreCase можно использовать логическое значение, чтобы узел игнорировал регистр строк. В примере ниже в качестве исходной строки используется строка Hello World, а в качестве второй строки — «h». Так как строка Hello World начинается со строки «h», а для параметра ignoreCase задано значение True, узел StartsWith возвращает значение True.
