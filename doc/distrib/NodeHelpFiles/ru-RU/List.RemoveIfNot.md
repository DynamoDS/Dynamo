## Подробности
`List.RemoveIfNot` возвращает список, в котором сохраняются элементы, соответствующие заданному типу элементов, и удаляются все остальные элементы из исходного списка.

Чтобы удалить элементы, может потребоваться использовать полный путь к узлу, например, `Autodesk.DesignScript.Geometry.Surface`, в параметре `type`. Чтобы получить пути для элементов списка, укажите список в узле `Object.Type`.

В примере ниже `List.RemoveIfNot` возвращает список с одной строкой, удаляя элементы-точки из исходного списка, поскольку они не соответствуют указанному типу.
