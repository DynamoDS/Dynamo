## Подробности
`PolyCurve.Heal` принимает самопересекающийся объект PolyCurve и возвращает новый объект PolyCurve, который не пересекает сам себя. Входной объект PolyCurve не может пересекать себя более трех раз. Другими словами, если один сегмент объекта PolyCurve пересекает более двух других сегментов или встречается с ними, исправление не сработает. Введите для `trimLength` значение больше 0, и конечные сегменты, которые длиннее `trimLength`, не будут обрезаны.

В примере ниже самопересекающийся объект PolyCurve исправляется с помощью узла `PolyCurve.Heal`.
