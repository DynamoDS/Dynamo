## Подробности

Функция «Точечная диаграмма» создает диаграмму с точками, расположение которых зависит от значений по осям X и Y, а цвет — от группы.
Помечайте группы или меняйте количество групп, вставляя список строковых значений в поле ввода меток. Каждая метка создает соответствующую цветовую группу. Если ввести только одно строковое значение, все точки будут иметь один цвет и одну метку.

Чтобы определить размещение каждой точки, используйте список списков, где содержатся двойные значения для точек по осям X и Y. Для осей X и Y должно быть указано одинаковое количество значений. Количество вложенных списков должно также совпадать с количеством строковых значений в поле ввода меток.

Группам можно присвоить цвета, указав список цветов в соответствующем поле. Если пользователь задает свои цвета, количество цветов должно соответствовать количеству строковых значений в метках. Если не указать цвета, каждой группе будет назначен случайный цвет.
