## Подробности
HorizontalFrameAtParameter возвращает систему координат, совмещенную с входной кривой в заданном параметре. Параметризация кривой измеряется в диапазоне от 0 до 1, где 0 обозначает начало кривой, а 1 — ее конец. Ось Z результирующей системы координат сонаправлена с осью Z МСК, а ось Y направлена по касательной к кривой в указанном параметре. В примере ниже сначала с помощью узла ByControlPoints создается NURBS-кривая, где в качестве входных элементов используется набор случайных точек. Для управления входным элементом parameter узла HorizontalFrameAtParameter используется числовой регулятор, значение которого находится в диапазоне от 0 до 1.
