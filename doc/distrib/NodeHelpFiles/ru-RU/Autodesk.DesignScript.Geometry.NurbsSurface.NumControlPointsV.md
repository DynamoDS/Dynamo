## Подробности
NumControlPointsV позволяет определить количество контрольных точек в направлении V объекта NurbsSurface. Этот узел возвращает целое число. В примере ниже NurbsSurface возвращает для своих контрольных точек V целое число, равное 12. Контрольные точки возвращаются в виде объектов Point с помощью узла NurbsSurface.ControlPoints.
