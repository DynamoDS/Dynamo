## Подробности
TrimTrailingWhitespace позволяет удалить пробелы в конце входной строки. В примере ниже используется строка «     Hello World     », содержащая несколько начальных и конечных пробелов. Узел TrimTrailingWhitespace позволяет удалить пробелы в конце строки.
