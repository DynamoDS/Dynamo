## Подробности
Метод `Mesh.Edges` возвращает базовые ребра сети в виде линий Dynamo. Этот узел удобно использовать для визуализации структуры сети, запроса ее конструкции, а также в качестве этапа преобразования сети или редактирования ее формы. В приведенном ниже примере `Mesh.Edges` используется для импортированной сети и дает полезную информацию о ее разрешении.
