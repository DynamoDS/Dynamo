## Подробности
`CoordinateSystem.Scale (coordinateSystem, amount)` возвращает новый объект CoordinateSystem, масштабированный с использованием заданного коэффициента.

В приведенном ниже примере объект CoordinateSystem в начале координат масштабируется с использованием коэффициента 3,5. В результате длины векторов и масштабные коэффициенты объекта CoordinateSystem равны 3,5. Для задания масштабного коэффициента используется регулятор.
