## Подробности
IndexOf позволяет найти в заданной строке вторую входную строку. При обнаружении второй строки этот узел возвращает индекс первого символа первого вхождения строки. По умолчанию этот узел чувствителен к регистру. Для входного параметра ignoreCase можно использовать логическое значение, чтобы узел игнорировал регистр строк. В примере ниже выполняется поиск символа «o» в строке Hello World. Эта буква встречается в строке дважды, однако узел IndexOf возвращает только индекс первого вхождения.
