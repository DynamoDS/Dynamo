## Подробности
Surface.ByLoft с поперечными сечениями принимает в качестве входного элемента список кривых. Этот узел создает поверхность путем лофтинга между списком кривых в заданном порядке. В примере ниже создаются две кривые, линия и синусоида. Узел List.Create используется для объединения этих двух кривых в список, который используется в качестве входного элемента узла Surface.ByLoft. Результат представляет собой поверхность, созданную путем лофтинга между синусоидой с одной стороны и линией с другой.
