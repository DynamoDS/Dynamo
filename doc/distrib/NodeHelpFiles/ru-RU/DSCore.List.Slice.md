## Подробности
`List.Slice` возвращает вложенный список на основе входного списка по заданным началу, концу и шагу. Входной параметр `start` указывает начальный индекс, с учетом которого строится выходной список. Входной параметр `end` указывает конечный индекс. Элемент с этим индексом не включается в выходной список. Входной параметр `step` определяет расстояние в индексах, на которое отстоят друг от друга элементы результирующего списка. По умолчанию используется значение 1.

В примере ниже используется начальный индекс 2 и конечный индекс 5. В результате из исходного списка создается новый список из элементов с индексами 2, 3 и 4.
