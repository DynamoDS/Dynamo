## Подробности

Линейный график значений индексов создает линейную диаграмму с одной или несколькими линиями, проходящими через последовательность точек. Линиям присваиваются метки и цвета. Количество строковых значений в списке, добавленном в поле входных данных меток, определяет число линий в диаграмме и соответствующие им метки.

Список значений с двойной точностью, указанный в поле входных значений, определяет положение каждой точки, через которую проходят линии. Чтобы назначить цвет каждой линии, укажите список значений цветов в поле входных данных.
