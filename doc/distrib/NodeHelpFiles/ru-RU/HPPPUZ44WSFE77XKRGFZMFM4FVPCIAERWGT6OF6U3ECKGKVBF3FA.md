## Подробности
В приведенном ниже примере простая Т-сплайновая поверхность преобразуется в строковый формат TSM с помощью узла `TSplineSurface.SerializeAsTSM` и экспортируется в текстовый файл. Этот файл можно впоследствии воссоздать с помощью узла `TSplineSurface.DeserializeFromTSM`.
