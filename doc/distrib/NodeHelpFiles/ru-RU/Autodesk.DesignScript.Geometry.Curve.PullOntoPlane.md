## Подробности
PullOntoPlane позволяет создать новую кривую путем проецирования входной кривой на входную плоскость с использованием нормали плоскости в качестве направления проецирования. В примере ниже сначала с помощью узла ByControlPoints создается NURBS-кривая, где в качестве входных элементов используется набор случайных точек. В узле PullOntoPlane плоскость XY МСК используется в качестве плоскости, на которую выполняется проекция. Результат представляет собой плоскую кривую на плоскости XY.
