## Подробности
Remove принимает в качестве входного значения строку и удаляет из нее символы, начиная с входного значения startIndex. Количество удаляемых символов задается индексом count. В примере ниже используется строка Hello World. Значение startIndex и количество символов, которые требуется удалить с помощью узла Remove, задаются с помощью двух целочисленных регуляторов.
