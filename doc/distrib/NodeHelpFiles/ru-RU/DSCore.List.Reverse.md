## Подробности
`List.Reverse` извлекает элементы из списка и создает новый список с обратным порядком элементов.

В этом примере сначала создается список с использованием `Range`, который позволяет получить номера в порядке возрастания. Затем с помощью узла `List.Reverse` можно изменить порядок элементов в списке.
