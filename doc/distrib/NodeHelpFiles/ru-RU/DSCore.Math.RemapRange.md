## Подробности
RemapRange принимает список чисел и осуществляет линейное масштабирование этих чисел в новый заданный диапазон. Минимальное значение исходного списка сопоставляется с входным значением newMin, а максимальное — с входным значением newMax. Остальные числа масштабируются для сохранения коэффициента распределения. В примере ниже список чисел для повторного сопоставления создается с помощью узла RandomList. Новые минимальное и максимальное значения в результирующем списке создаются с помощью двух числовых регуляторов.
