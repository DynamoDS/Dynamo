## Подробности
Combine позволяет соединить несколько входных элементов в одну строку. В примере ниже для выбора каталога используется функция DirectoryPath. Имя файла задается в Code Block. Затем с помощью узла Combine путь к каталогу и имя файла объединяются в путь к файлу.
