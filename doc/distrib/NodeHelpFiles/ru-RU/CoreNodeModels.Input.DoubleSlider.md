## Подробности
Number Slider — это настраиваемый сенсорный узел, который позволяет взаимодействовать с данными, управляющими определениями Dynamo. В данном примере числовой регулятор используется в паре с блоком кода для создания динамической последовательности списков с переменным шагом.
