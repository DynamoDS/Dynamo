## Подробности
`List.Clean` возвращает список без значений null и пустых списков.

В примере ниже `List.Clean` удаляет два значения null из списка и возвращает только целые числа. С помощью логического значения можно включить или отключить входной параметр `preserveIndices`, чтобы сохранить индексы, которые содержали значения null.
