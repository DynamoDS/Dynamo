## Подробности
`Not` возвращает значение, противоположное входному логическому значению. В DesignScript узел Not также известен как `!` (восклицательный знак), и он преобразует значение True в False и наоборот.

В приведенном ниже примере узлы `Not` используются для замены входных логических значений на противоположные. В примере также показано использование блока кода с `!input` для преобразования логического выражения.
