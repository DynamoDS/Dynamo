## In-Depth
В приведенном ниже примере узел `TSplineFace.Valence` используется для запроса степеней всех граней Т-сплайнового конуса-примитива. Обратите внимание на то, что для всех граней, включая верхний ряд конуса, возвращается степень 4, даже если может показаться, что верхний ряд состоит из треугольников. Это особенность пары примитивных форм Т-сплайна, таких как конус и сфера, где для получения формы две из четырех вершин группируются в одну точку.
