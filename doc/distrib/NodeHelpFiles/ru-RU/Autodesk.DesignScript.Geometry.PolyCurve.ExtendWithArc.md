## Подробности
ExtendWithArc позволяет добавить дугу окружности в начало или в конец входного объекта PolyCurve. Этот узел возвращает один объединенный объект PolyCurve. Входное значение радиуса определяет радиус окружности, а входное значение длины — расстояние вдоль окружности для дуги. Общая длина должна быть меньше длины полной окружности с заданным радиусом или равна ей. Сгенерированная дуга будет касательной к концу входного объекта PolyCurve. Логическое входное значение для параметра endOrStart определяет, на каком конце объекта PolyCurve будет создана дуга. Если задано значение True, дуга будет создана в конце объекта PolyCurve. В противном случае дуга будет создана в начале объекта PolyCurve. В примере ниже сначала создается объект PolyCurve с помощью набора случайных точек и узла PolyCurve.ByPoints. Затем задаются параметры ExtendWithArc с помощью двух числовых регуляторов и логического переключателя.
