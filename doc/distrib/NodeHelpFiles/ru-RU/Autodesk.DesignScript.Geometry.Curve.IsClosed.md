## Подробности
IsClosed возвращает логическое значение, показывающее, замкнута ли входная кривая. В примере ниже сначала с помощью узла ByControlPoints создается NURBS-кривая, где в качестве входных элементов используется набор случайных точек. После этого с помощью узла IsClosed можно проверить, замкнута ли кривая.
