## Подробности
Round с параметром digits позволяет выбрать количество десятичных знаков, до которого требуется округлить входное число. 0 округляется до ближайшего целого числа. Значения меньше 0 не допускаются. В примере ниже для создания случайного числа используется узел Rand. Входное значение digits для узла Round задается с помощью целочисленного регулятора.
