## Подробности
Repair пытается восстановить поверхности или сложные поверхности с недопустимой геометрией, а также оптимизировать их. Узел Repair возвращает новый объект поверхности.
Этот узел используется для устранения ошибок при выполнении операций с импортированными или преобразованными объектами геометрии.

Например, если при импорте данных из контекста-основы, например из **Revit** или из файла **SAT**, происходит неожиданный сбой логической операции или обрезки, операция восстановления может очистить *недопустимые объекты геометрии*, являющиеся причиной сбоя.

В большинстве случаев использовать эту функцию для объектов геометрии, создаваемых в Dynamo, не нужно; она нужна только для объектов геометрии из внешних источников. В противном случае сообщите об ошибке на странице команды Dynamo на github.
