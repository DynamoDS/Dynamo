## Подробности
`Mesh.EdgesAsSixNumbers` определяет координаты X, Y и Z вершин каждого уникального ребра в сети. На каждое ребро приходится шесть чисел. Этот узел можно использовать для запроса или реконструкции сети или ее ребер.

В приведенном ниже примере `Mesh.Cuboid` используется для создания сети-кубоида, которая затем используется в качестве входных данных для извлечения списка ребер, выраженного в виде шести чисел, с помощью узла `Mesh.EdgesAsSixNumbers`. Список разбивается на отдельные списки из 6 элементов с помощью `List.Chop`, затем с помощью `List.GetItemAtIndex` и `Point.ByCoordinates` восстанавливаются списки начальных и конечных точек каждого ребра. Наконец, `List.ByStartPointEndPoint` реконструирует ребра сети.
