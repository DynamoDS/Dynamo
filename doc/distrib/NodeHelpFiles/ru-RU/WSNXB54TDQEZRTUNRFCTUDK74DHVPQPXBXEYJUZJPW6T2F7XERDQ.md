## Подробности
В приведенном ниже примере для отображения списка вершин на плоской Т-сплайновой поверхности используется узел `TSplineTopology.VertexByIndex`.
Поверхность представляет собой выдавленные вершины, некоторые из которых затем выравниваются с помощью узла `TSplineSurface.FlattenVertices(vertices)`.
Узлы `TSplineVertex.UVNFrame` и `TSplineUVNFrame.Position` используются для выделения выравниваемых вершин.
