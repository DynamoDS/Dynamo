## Подробности
Pixels возвращает список цветов из входного изображения в виде двумерного массива. Узел Pixels принимает два дополнительных входных значения для задания количества образцов по осям X и Y, из которых будут извлекаться пиксели. В примере ниже изображение считывается с помощью узла ReadFromFile. Затем это изображение используется в качестве входного элемента узла Pixels, который извлекает цвета пикселей.
