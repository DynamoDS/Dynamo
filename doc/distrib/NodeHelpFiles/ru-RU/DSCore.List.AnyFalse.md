## Подробности
`List.AnyFalse` возвращает значение True, если в заданном списке есть хотя бы одно логическое значение False.

В примере ниже с помощью `List.AnyFalse` определяется, есть ли в списке значения False. В нем содержится одно значение False, поэтому `List.AnyFalse` возвращает значение True.
