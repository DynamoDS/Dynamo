## Подробности
DirectoryPath позволяет выбрать каталог на компьютере. Чтобы задать путь к каталогу, нажмите кнопку «Обзор...» в узле DirectoryPath, а затем перейдите в нужный каталог. Если этот узел добавлен в ранее сохраненный файл DYN, по умолчанию будет открываться каталог с файлом DYN.
