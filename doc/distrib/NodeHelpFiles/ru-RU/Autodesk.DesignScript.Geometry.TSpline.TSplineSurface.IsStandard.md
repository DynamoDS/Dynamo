## Подробности
Т-сплайновая поверхность является стандартной, когда все Т-точки отделены от нулевых точек по крайней мере двумя изолиниями. Стандартизация необходима для преобразования Т-сплайновой поверхности в поверхность NURBS.

В приведенном ниже примере одна из граней Т-сплайновой поверхности, созданной с помощью узла `TSplineSurface.ByBoxLengths`, разделена. Узел `TSplineSurface.IsStandard` проверяет, является ли поверхность стандартной, однако возвращает отрицательный результат.
Затем для стандартизации поверхности используется узел `TSplineSurface.Standardize`. Новые управляющие точки вводятся без изменения формы поверхности. Итоговая поверхность проверяется с помощью узла `TSplineSurface.IsStandard`, который теперь подтверждает, что она действительно является стандартной.
Узлы `TSplineFace.UVNFrame` и `TSplineUVNFrame.Position` используются для выделения на поверхности разделенной грани.
