## In-Depth
Узел `TSplineReflection.Axis` возвращает значение вектора, представляющее ось, относительно которой отражен Т-сплайн.

В приведенном ниже примере Т-сплайновая поверхность создана с добавленными отражениями. Затем используется уел `TSplineSurface.Reflections` для запроса отражений, примененных к поверхности. После этого используется узел `TSplineReflection.Axis` для возврата вектора, представляющего ось отражения.
