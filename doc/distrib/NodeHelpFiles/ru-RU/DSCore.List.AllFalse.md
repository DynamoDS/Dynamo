## Подробности
`List.AllFalse` возвращает значение False, если какой-либо элемент в заданном списке имеет значение True или не является логическим значением. `List.AllFalse` возвращает значение True, только если каждый элемент в списке является логическим значением False.

В примере ниже с помощью `List.AllFalse` обрабатываются списки логических значений. В первом списке есть значение True, поэтому возвращается значение False. Второй список содержит только значения False, поэтому возвращается значение True. В третьем списке есть вложенный список, содержащий значение True, поэтому возвращается значение False. Конечный узел оценивает два вложенных списка и возвращает значение False для первого из них, так как в нем есть значение True, и возвращает значение True для второго, поскольку в нем содержатся только значения False.
