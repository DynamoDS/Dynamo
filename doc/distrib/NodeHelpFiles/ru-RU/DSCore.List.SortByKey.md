## Подробности
`List.SortByKey` принимает входной список для сортировки и список соответствующих сортируемых ключей, таких как числа или строки. `List.SortByKey` сортирует список ключей по возрастанию и переупорядочивает входной список в соответствии с новым порядком сортированных ключей. Этот узел выводит переупорядоченный список и сортированный список ключей.

В примере ниже в качестве входного списка используется диапазон от 0 до 9. Для списка ключей создается список из 10 случайных чисел от 0 до 4. Список ключей отсортирован в числовом виде, а диапазон от 0 до 9 переупорядочен в соответствии с порядком соответствующих элементов в списке ключей.
