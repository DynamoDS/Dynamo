## Подробности
ContainmentTest возвращает логическое значение, показывающее, находится ли данная точка внутри заданного многоугольника. Для этого многоугольник должен быть плоским и не являться самопересекающимся. В примере ниже многоугольник создается с использованием последовательности точек, созданной с помощью узла ByCylindricalCoordinates. Сохранение постоянной отметки и сортировка углов гарантируют, что многоугольник будет плоским и не будет самопересекающимся. Затем создается точка, которую требуется проверить, после чего с помощью узла ContainmentTest проверяется, находится ли она внутри многоугольника или вне его.
