## Подробности
Alpha возвращает значение альфа входного цвета в диапазоне от 0 до 255. В примере ниже с помощью узлов ByARGB создаются два цвета, которые затем объединяются. Значение альфа результирующего цвета можно найти с помощью узла Alpha.
