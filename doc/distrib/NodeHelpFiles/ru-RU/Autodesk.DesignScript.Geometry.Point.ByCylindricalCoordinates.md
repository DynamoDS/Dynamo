## Подробности
Узел Point.ByCylindricalCoordinates позволяет создать точку, расположенную в пределах цилиндрического пространства. Здесь в качестве параметра переплетения задается перекрестная ссылка, а для создания сетки точек вдоль цилиндра с динамическим радиусом используется диапазон углов и отметок.
