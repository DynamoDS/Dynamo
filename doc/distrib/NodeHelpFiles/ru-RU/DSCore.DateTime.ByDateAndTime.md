## Подробности
ByDateAndTime возвращает значение dateTime на основе входных значений года, месяца, дня, часов, минут, секунд и миллисекунд. В примере ниже на основе соответствующих входных значений создается новое значение dateTime, равное August 08, 2006 4:24PM.
