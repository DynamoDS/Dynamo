## Подробности
TangentAtVParameter возвращает касательный вектор в направлении V для заданного положения UV на поверхности. В примере ниже сначала создается поверхность с помощью узла BySweep2Rails. Затем с помощью двух регуляторов чисел задаются параметры U и V для определения касательного вектора V.
