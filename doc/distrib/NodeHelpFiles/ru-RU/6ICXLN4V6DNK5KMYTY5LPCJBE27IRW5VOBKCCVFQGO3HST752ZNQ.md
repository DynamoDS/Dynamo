## Подробности

В приведенном ниже примере Т-сплайновая поверхность сопоставляется с NURBS-кривой с помощью
узла `TSplineSurface.CreateMatch(tSplineSurface,tsEdges,curves)`. Минимальный ввод, необходимый
для узла, — базовая поверхность `tSplineSurface`, набор ребер поверхности, задаваемый входным параметром `tsEdges`, и кривая
или список кривых.
Параметры сопоставления определяются следующими входными параметрами:
— `continuity`: позволяет задать тип непрерывности для сопоставления. Для входного параметра необходимо указать значение 0, 1 или 2, соответствующее позиционной непрерывности G0, непрерывности по касательной G1 или по кривизне G2. Однако для сопоставления поверхности с кривой доступна только непрерывность G0 (входное значение 0).
— `useArcLength`: управляет параметрами типа выравнивания. Если задано значение True, используется тип выравнивания «Длина
дуги». Это выравнивание минимизирует физическое расстояние между каждой точкой Т-сплайновой поверхности
и соответствующей точкой на кривой. Если задано значение False, используется тип выравнивания «Параметрическое» —
каждая точка на Т-сплайновой поверхности сопоставляется с точкой на сравнимом параметрическом расстоянии вдоль
целевой кривой сопоставления.
— `useRefinement`: если задано значение True, добавляет на поверхность управляющие точки для сопоставления с целью
в пределах заданного допуска уточнения (параметр `refinementTolerance`).
— `numRefinementSteps`: максимальное число операций деления базовой Т-сплайновой поверхности
при попытке достичь значения `refinementTolerance`. Параметры `numRefinementSteps` и `refinementTolerance` игнорируются, если для параметра `useRefinement` задано значение False.
— `usePropagation`: определяет долю поверхности, которая будет затронута сопоставлением. Если задано значение False, поверхность затрагивается минимально. Если задано значение True, на поверхность влияет значение параметра `widthOfPropagation`.
— `scale`: масштаб соприкосновения, который влияет на результаты для непрерывности G1 и G2.
— `flipSourceTargetAlignment`: изменяет направление выравнивания на противоположное.
