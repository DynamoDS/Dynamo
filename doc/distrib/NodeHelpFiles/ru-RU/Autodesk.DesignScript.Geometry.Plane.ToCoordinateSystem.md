## Подробности
Plane.ToCoordinateSystem возвращает систему координат на основе входной плоскости с использованием ее начала координат, а также XAxis и YAxis. В примере ниже сначала с помощью набора случайных точек создается оптимально вписанная плоскость по точкам. Затем с помощью ToCoordinateSystem можно преобразовать плоскость в систему координат.
