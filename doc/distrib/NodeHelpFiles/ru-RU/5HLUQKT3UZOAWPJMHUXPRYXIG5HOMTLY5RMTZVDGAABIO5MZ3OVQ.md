## Подробности
`Surface.Thicken (surface, thickness, both_sides)` создает тело путем смещения поверхности в соответствии с входным значением `thickness` и путем ограничения концов для замыкания тела. У узла есть дополнительное входное значение, предназначенное для указания необходимости утолщения по обе стороны. Входной параметр `both_sides` принимает логическое значение. Значение True означает утолщение с обеих сторон, а False — утолщение с одной стороны. Параметр `thickness` определяет общую толщину конечного тела, поэтому если для параметра `both_sides` задано значение True, результат будет смещен от исходной поверхности на половину входного значения толщины с обеих сторон.

В примере ниже сначала создается поверхность с помощью узла `Surface.BySweep2Rails`. Затем создается тело с помощью регулятора чисел, определяющего входное значение `thickness` узла `Surface.Thicken`. Переключатель с логическими значениями указывает на утолщение с обеих сторон или только с одной.
