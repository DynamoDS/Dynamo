## Подробности
`Surface.ByLoft (crossSections, guideCurves)` принимает список поперечных сечений для лофтинга и отдельный список направляющих кривых для определения профиля лофт-объекта.

В примере ниже две прямые линии используются в качестве входных поперечных сечений. Для направляющих кривых создается одна синусоида и одна прямая линия. Регулятор чисел определяет расстояние между синусоидой и прямой линией. Полученный лофт-объект интерполируется между синусоидой и прямой линией.
