## Подробности
`Delaunay.ByParametersOnSurface` возвращает триангуляцию Делоне входной поверхности.

В примере ниже возвращается триангуляция Делоне для объекта NurbsSurface с числом UV, равным 400. Увеличение числа UV позволит создать лучшую аппроксимацию исходной поверхности, но с большим количеством треугольников.
