## Подробности
`List.IsHomogeneous` возвращает логическое значение, которое указывает, являются ли все элементы списка элементами одного типа.

В примере ниже простой список чисел сравнивается с этим же списком, в который вставлена строка. Исходный список является однородным (только числа), поэтому `List.IsHomogeneous` возвращает значение True. Новый список не является однородным (числа и строка), поэтому `List.IsHomogeneous` возвращает значение False.
