## Подробности
Corners возвращает список точек, которые являются углами заданного многоугольника. В примере ниже сначала создается набор случайных точек, отсортированных по углу, для использования в узле Polygon.ByPoints. С помощью узла Corners эти точки можно вновь получить в виде списка.
