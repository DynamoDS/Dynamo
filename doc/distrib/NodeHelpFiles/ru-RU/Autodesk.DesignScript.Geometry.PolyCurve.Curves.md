## Подробности
`PolyCurve.Curves` возвращает список отдельных кривых, составляющих входной объект PolyCurve.

В примере ниже создается набор случайных точек, а затем `PolyCurve.ByPoints` используется для создания незамкнутого объекта PolyCurve. После этого узел `PolyCurve.Curves` используется для создания списка отдельных кривых, составляющих объект PolyCurve.
