## Подробности
NormalAtPoint позволяет найти вектор нормали поверхности во входной точке на поверхности. Если входная точка не находится на поверхности, этот узел находит на поверхности точку, ближайшую к входной. В примере ниже сначала создается поверхность с помощью узла BySweep2Rails. Затем в Code Block задается точка, в которой требуется найти нормаль. Точка не находится на поверхности, поэтому узел использует ближайшую точку на поверхности в качестве точки, в которой нужно найти нормаль.
