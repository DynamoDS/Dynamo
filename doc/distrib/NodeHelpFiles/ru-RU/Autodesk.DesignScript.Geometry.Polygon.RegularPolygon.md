## Подробности
Polygon.RegularPolygon — это простой компонент, который вписывает правильный многоугольник с любым количеством сторон в окружность. В данном примере создается опорная окружность путем оптимального вписывания по случайным точкам, внутри которой строится шестисторонний многоугольник.
