## Подробности
`List.GetItemAtIndex` получает в качестве аргументов список и индекс (целое число) и возвращает элемент из списка по указанному индексу.

В примере ниже сначала с помощью блока кода создается диапазон чисел от 0 до 5 с шагом 1. Затем с помощью узла `List.GetItemAtIndex` извлекается один элемент с индексом 4.
