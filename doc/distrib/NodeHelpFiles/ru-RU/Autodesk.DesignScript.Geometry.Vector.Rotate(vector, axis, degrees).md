## Подробности
`Vector.Rotate (vector, axis, degrees)` возвращает новый вектор, повернутый вокруг входной оси на заданное количество градусов.

В приведенном ниже примере вектор, представленный линией, поворачивается вокруг вектора оси Z. Для управления степенью поворота используется узел Number Slider. Можно подключить другие векторы, чтобы получить разные оси вращения.
