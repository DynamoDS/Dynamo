## Подробности
ByARGB позволяет создать цвет с помощью входных значений Alpha, Red, Green и Blue. Входные значения находятся в диапазоне от 0 до 255. В примере ниже для задания входных значений узла ByARGB используются 4 числовых регулятора в диапазоне от 0 до 255. Для визуализации результирующего цвета используются узел Display.ByGeometryColor и стандартный кубоид.
