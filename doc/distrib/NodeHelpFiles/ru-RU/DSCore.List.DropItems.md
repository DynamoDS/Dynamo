## Подробности
`List.DropItems` удаляет из списка определенное количество элементов, которое указывается во входном параметре `amount`. Если `amount` является положительным числом, элементы удаляются с начала списка. Если `amount` является отрицательным числом, элементы удаляются с конца списка.

В примере ниже сначала с помощью `Range` создается список, а затем с помощью `List.DropItems` и параметра `amount`, равного 1, удаляется первый элемент списка. Поскольку значение входного параметра `amount` является положительным, элемент удаляется из начала списка.
