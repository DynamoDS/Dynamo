## Подробности
ToNumber позволяет преобразовать входную строку в числовой тип данных. Этот узел принимает строку в числовом формате. Из примера ниже видно, что строку Hello World преобразовать в число нельзя, а строку 5,2 можно преобразовать в числовое значение 5,2.
