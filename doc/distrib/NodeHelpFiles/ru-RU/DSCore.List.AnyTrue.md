## Подробности
`List.AnyTrue` возвращает значение True, если в заданном списке есть хотя бы одно логическое значение True.

В примере ниже с помощью `List.AnyTrue` определяется, есть ли в списке значения True. В нем содержится 2 значения True, поэтому AnyTrue возвращает значение True.
