## In-Depth
В приведенном ниже примере создается Т-сплайновая поверхность-примитив цилиндра. Положение и ориентация цилиндра определяются системой координат, заданной входным параметром `cs`. Размер цилиндра определяется параметрами `radius` и `height`. Вертикальные и радиальные пролеты задаются параметрами `radiusSpans` и `heightSpans`. Исходная симметрия формы определяется входным параметром `symmetry`. Если для симметричности по оси X или Y задано значение True, количество радиальных пролетов должно быть кратно 4. Входной параметр `inSmoothMode` используется для переключения между режимами сглаживания и рамки при предварительном просмотре Т-сплайновой поверхности.
