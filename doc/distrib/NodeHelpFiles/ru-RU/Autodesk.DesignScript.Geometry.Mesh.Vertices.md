## Подробности
`Mesh.Vertices` возвращает список уникальных вершин сети в виде списка точек. Структура выходного списка совместима с выходными индексами узла `Mesh.VertexIndicesByTri`, и их можно использовать вместе для реконструкции или редактирования сети.
