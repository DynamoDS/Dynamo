## Подробности
Узел FromPixels с входными значениями ширины и высоты создает изображение на основе входного плоского списка цветов, каждый цвет в котором становится одним пикселем. Произведение ширины на высоту должно быть равно общему количеству цветов. В примере ниже сначала с помощью узла ByARGB создается список цветов. В блоке кода создается диапазон значений от 0 до 255, который затем при подключении к входным значениям r и g образует последовательность цветов от черного до желтого. Создается изображение шириной 8. Для определения высоты изображения используются узлы Count и Division. Для предварительного просмотра созданного изображения можно использовать узел WatchImage.
