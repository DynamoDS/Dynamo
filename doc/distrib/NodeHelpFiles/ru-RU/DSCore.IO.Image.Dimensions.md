## Подробности
Dimensions возвращает ширину и высоту входного изображения в пикселях. В примере ниже сначала с помощью узлов FilePath и File.FromPath создается объект файла, указывающий на изображение. Затем для создания объекта изображения на основе файла используется узел Image.ReadFromFile. Узел Dimensions показывает, что ширина и высота этого изображения составляют 270 пикселей.
