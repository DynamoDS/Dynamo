## Подробности
Arc.ByCenterPointStartPointSweepAngle позволяет создать дугу вокруг центральной точки с заданным углом сдвига. По начальной точке программа Dynamo определяет, где начинать построение дуги. В данном примере построение дуги выполняется вокруг точки, созданной случайным образом.
