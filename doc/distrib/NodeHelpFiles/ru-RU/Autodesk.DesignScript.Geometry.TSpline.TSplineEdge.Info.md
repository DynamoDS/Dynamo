## In-Depth
Узел `TSplineEdge.Info` возвращает следующие свойства ребра Т-сплайновой поверхности:
— `uvnFrame`: точка на выпуклой оболочке, вектор U, вектор V и вектор нормали ребра Т-сплайна;
— `index`: индекс ребра;
— `isBorder`: определение того, является ли выбранное ребро границей Т-сплайновой поверхности;
— `isManifold`: определение того, является ли выбранное ребро однородным.

В приведенном ниже примере для получения списка всех ребер Т-сплайновой поверхности-примитива цилиндра используется узел `TSplineTopology.DecomposedEdges`, а для исследования их свойств используется узел `TSplineEdge.Info`.
