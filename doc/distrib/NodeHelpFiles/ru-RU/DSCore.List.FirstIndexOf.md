## Подробности
`List.FirstIndexOf` выполняет поиск элемента, указанного во входном параметре Item, по заданному списку и возвращает индекс первого вхождения этого элемента в списке. Если элемент в списке не найден, возвращается значение –1.

В примере ниже сначала создается список из 10 случайных чисел в диапазоне от 0 до 4. С помощью узла `List.FirstIndexOf` выполняется поиск определенного элемента и извлекается индекс первого вхождения этого элемента в списке.
