## Подробности
`List.TakeItems` возвращает заданное количество элементов от начала входного списка. Если входной параметр `amount` равен отрицательному числу, элементы берутся из конца списка в обратном порядке.

В примере ниже сначала создается диапазон чисел от 0 до 9 с шагом 1. С помощью регулятора целочисленных значений для параметра `amount` задается количество элементов, которые нужно взять из исходного списка. Указано значение 4, таким образом выходной список содержит первые 4 элемента из исходного списка.
