## Подробности
`List.DropEveryNthItem` удаляет элементы из входного списка через интервалы, указанные во входном параметре n. Начальную точку интервала можно изменить с помощью аргумента `offset`. Например, если для n указать значение 3 и оставить для offset значение по умолчанию, равное 0, будут удалены элементы с индексами 2, 5, 8 и т. д. Если offset равно 1, будут удалены элементы с индексами 0, 3, 6 и т. д. Обратите внимание, что при любом значении offset выполняется перебор всех элементов списка, в том числе с возвратом в его начало. Если требуется сохранить выбранные элементы, а не удалять их, см. раздел `List.TakeEveryNthItem`.

В примере ниже сначала с помощью `Range` создается список чисел, а затем, чтобы удалить все числа через одно, для параметра `n` задается значение 2.
