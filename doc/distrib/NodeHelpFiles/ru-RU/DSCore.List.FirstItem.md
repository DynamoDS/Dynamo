## Подробности
`List.FirstItem` возвращает первый элемент входного списка.

В примере ниже создается список с тремя вложенными списками из 10 случайных чисел от 1 до 10. Затем с помощью узла FirstItem возвращается первый элемент списка.

С помощью средства выбора уровня списка в узле `List.FirstItem` можно задать разные способы возврата первого элемента из вложенного списка. Когда задано значение L1, возвращаются все элементы из всех вложенных списков в виде единого неструктурированного списка. Когда задано значение L2, возвращается первый элемент каждого вложенного списка. Когда задано значение L3, возвращаются все элементы списка 0.
