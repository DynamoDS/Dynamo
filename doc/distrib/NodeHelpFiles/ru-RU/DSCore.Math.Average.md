## Подробности
Average возвращает среднее (среднее арифметическое) списка чисел. В примере ниже сначала с помощью трех числовых регуляторов создаются три числа, среднее значение которых требуется найти. Затем с помощью узла List.Create эти три числа объединяются в список, который затем можно использовать в качестве входного элемента для узла Average.
