## In-Depth
В приведенном ниже примере создается Т-сплайновая поверхность-примитив цилиндра. Верхняя и нижняя плоскости цилиндра определяются входными параметрами `startPoint` и `endPoint`. С помощью входного параметра `radius` задается размер. Входные параметры `radiusSpans` и `heightSpans` определяют пролеты в радиальном и вертикальном направлениях. Исходная симметрия формы задается входным параметром `symmetry`. Если для симметричности по оси X или Y задано значение True, количество радиальных пролетов должно быть кратно 4. Входной параметр `inSmoothMode` используется для переключения между режимами сглаживания и рамки при предварительном просмотре Т-сплайновой поверхности.
