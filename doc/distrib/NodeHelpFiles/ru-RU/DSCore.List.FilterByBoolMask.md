## Подробности
В качестве входных данных в `List.FilterByBoolMask` передаются два списка. Первый список разбивается на два отдельных списка в соответствии со списком логических значений (True или False). Элементы из входного списка `list`, соответствующие значению True на входе `mask`, попадают в выходной параметр In, а элементы, соответствующие значению False, попадают в выходной параметр `out`.

В примере ниже `List.FilterByBoolMask` выбирает из списка материалов дерево и ламинат. Сначала два списка сравниваются для поиска совпадающих элементов, затем с помощью оператора `Or` определяются элементы списка, соответствующие значению True. Затем элементы списка фильтруются в зависимости от того, являются ли они деревом, ламинатом или другим материалом.
