## Подробности
Узел `TSplineTopology.BorderFaces` возвращает список граней границ, содержащихся в Т-сплайновой поверхности.

В приведенном ниже примере создаются две Т-сплайновые поверхности с помощью узла `TSplineSurface.ByCylinderPointsRadius`. Одна из этих поверхностей является незамкнутой, а вторая утолщается с помощью узла `TSplineSurface.Thicken`, который превращает ее в замкнутую. Обе поверхности исследуются с помощью узла `TSplineTopology.BorderFaces`, после чего для первой из них возвращается список граней границ, а для второй — пустой список. Это связано с тем, что у второй поверхности нет граней границ, так как она замкнута.
