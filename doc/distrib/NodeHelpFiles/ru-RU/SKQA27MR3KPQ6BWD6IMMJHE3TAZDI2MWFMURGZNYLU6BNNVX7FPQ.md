## Подробности
Узел «>=» является оператором «Больше или равно». Если входное значение «x» больше входного значения «y» или если эти значения равны, Существует в словаре 1.x; измененное имя графа: GreaterThanOrEqualTo.dyn Y

этот оператор возвращает значение «Истина». Если значение «x» меньше значения «y», возвращается значение «Ложь». В примере ниже оператор «>=» используется для определения того, больше ли входное значение «x» чем входное значение «y» или равно ему. Для управления входными данными для оператора «>=» используются два регулятора числовых значений.
