## Подробности
`Mesh.Volume` возвращает объем непересекающейся сети твердого тела. В приведенном ниже примере узлы `Mesh.Cone` и `Number.Slider` используются для создания конуса, который затем используется в качестве входных данных для подсчета треугольников. С помощью `Mesh.Volume` также можно определить, замкнута ли поверхность сети. Если поверхность сети не замкнута, узел вернет значение `null`.
