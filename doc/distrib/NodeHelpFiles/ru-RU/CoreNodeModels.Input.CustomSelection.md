## Подробности
Узел CustomDropdown позволяет создавать раскрывающийся список с пользовательскими метками и значениями. Если все значения являются числами, результат представляет собой число двойной точности. Если все значения являются целыми числами, результат представляет собой целое число. В примере ниже в узле CustomDropdownMenu выбрано значение Two, вследствие чего на выходе узла получено целое число `2`.
