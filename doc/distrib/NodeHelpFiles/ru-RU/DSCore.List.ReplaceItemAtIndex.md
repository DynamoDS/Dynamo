## Подробности
`List.ReplaceItemAtIndex` заменяет элемент во входном списке по заданному индексу.

В примере ниже сначала создается диапазон чисел от 0 до 5. Затем используется узел `List.ReplaceItemAtIndex` для замены элемента с индексом 3 на новый элемент, в данном случае — целое число 10.
