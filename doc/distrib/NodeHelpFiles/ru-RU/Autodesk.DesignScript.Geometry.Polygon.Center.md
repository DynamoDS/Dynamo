## Подробности
Polygon.Center позволяет определить центр заданного многоугольника. Этот узел принимает среднее значение углов. В случае невыпуклых многоугольников центр может фактически находиться вне многоугольника. В примере ниже сначала генерируется список случайных углов и радиусов, которые будут использоваться в качестве входных элементов узла Point.ByCylindricalCoordinates. Выполнение сортировки углов гарантирует, что результирующий многоугольник будет соединен в порядке увеличения углов, а потому не окажется самопересекающимся. Затем с помощью узла Center можно взять среднее значение точек и найти центр многоугольника.
