## In-Depth
`TSplineSurface.ByBoxLengths(origin, width, length, height, xSpans, ySpans, zSpans, symmetry, inSmoothMode)` creates a T-Spline Box surface centered around a given `origin`, with dimensions defined by `width`, `length` and `height` inputs and number of X, Y, and Z spans in each direction. The symmetry options can be specified with the `symmetry` input. The `inSmoothMode` Boolean input toggles between smooth preview and box mode.
