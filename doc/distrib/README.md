# Licensing Information

The folder consists of various Licensing and Terms of Use agreements that are used in product at different places.

- ADPAnalyticsConsent - Consent agreement for ADP Analytics tool usage.
- Autodesk - Generic Autodesk Software License Agreement
- License - Dynamo License. The file has to be in sync with the `License.txt` file at the root level.
- MLNodeAutocompleteConsent - Consent agreement for ML Node Autocomplete feature usage.
- TermsOfUse - Consent agreement for Dynamo Package Manager feature usage. The file has to be in sync with the `TermsOfUse.txt` as it is being used to display terms of Package Manager website.
