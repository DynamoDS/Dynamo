Ничего не выбрано

Эта ошибка указывает на отсутствие выбранных объектов в узле выбора (например, Select Model Element в Dynamo for Revit).

Устранение проблемы

Нажмите «Выбрать» в используемом узле и выберите элемент в Revit. Для этого необходимо перейти из окна Dynamo в окно Revit (или другой основной среды).

Дополнительные сведения

Дополнительные сведения можно найти в публикациях на форуме Dynamobim с обсуждением ошибки nothing is selected.
