Метод не найден

Это сообщение об ошибке означает, что Dynamo не удалось найти определение используемого метода DesignScript.

Устранение проблемы

Если пользовательский метод определен в узле Code Block, убедитесь, что используется допустимый код DesignScript. Убедитесь, что имя метода, заданное в узле CodeBlock, совпадает с именем вызываемого метода. Подробности см. в Руководстве по Design Script.
