Ожидает аргументы типа, а вызов выполнен с использованием

Это сообщение об ошибке указывает, что узел предусматривает определенный тип входных данных, однако полученный тип данных не может быть им использован. Обычно за сообщением об ошибке следует дополнительная информация о типах входных данных, переданных в узел, и поддерживаемых типах. Наиболее часто эта ошибка бывает вызвана следующими операциями.

  • Element.SetParameterByName — попробуйте использовать узел Element.Parameters для проверки параметров элемента. Дополнительные сведения можно найти в публикациях на форуме Dynamobim с обсуждением ошибки Expects argument type setparameter(«Ожидает аргументы типа setparameter»).
  • Geometry.BoundingBox — вместо этого узла попробуйте использовать узел Element.BoundingBox, как описано здесь. Дополнительные сведения можно найти в публикациях на форуме Dynamobim с обсуждением ошибки Expects argument type geometry («Ожидает аргументы типа geometry»).

Устранение проблемы

В разделе выше указаны некоторые способы, которые помогут решить эту проблему. Как правило, необходимо проверить, какие типы входных данных поддерживает используемый узел. Если возможно, измените тип входных данных, передаваемых в узел, на допустимый или проверьте, можно ли использовать другие узлы в той же функции. Кроме того, типы входных данных можно проверить с помощью узла Object.Type.
