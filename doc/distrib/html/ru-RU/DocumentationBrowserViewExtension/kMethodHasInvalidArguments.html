Содержит недопустимые аргументы

Это сообщение об ошибке указывает, что узел предусматривает входные данные определенного типа, но полученный им тип данных невозможно использовать. Эта ошибка аналогична другой ошибке — «ожидает аргументы типа, а вызов выполнен с использованием» — и обычно сопровождается дополнительной информацией, которая поможет определить, какие типы аргументов необходимо передать в используемый узел.

Устранение проблемы

Как правило, необходимо проверить, какие типы входных данных используются узлом, и попытаться определить, (а) передается ли в узел допустимый тип входных данных и (б) можно ли заменить тип входных данных, чтобы он поддерживался данным узлом (как показано в этом примере).

Например, может появиться примерно следующее сообщение Предупреждение. Не удалось выполнить операцию Element.SetParameterByName. Наилучшее соответствие перегруженного метода Revit.Elements.InternalUtilities.ElementUtils.SetParameterValue(Autodesk.Revit.DB.Parameter, double) содержит недопустимые аргументы — дополнительная информация в скобках поможет понять, в чем проблема. В этом случае SetParameterValue(Autodesk.Revit.DB.Parameter, double) означает, что используемый узел предусматривает входные данные типа double и другие входные данные не поддерживаются.
