Список индексов должен быть числовым

Это сообщение об ошибке появляется при попытке получить доступ к элементу в списке с помощью индекса, значение которого не является целым числом.

После выхода версии Dynamo 2.0 списки представляют собой наборы значений, доступ к которым возможен только с помощью целочисленных значений индексов (с диапазоном значений индексов от 0 до размера списка минус 1).

Подробнее о различиях между Dynamo 1.X и Dynamo 2.X (включая сведения об изменении правил доступа к элементам в списке) можно узнать здесь.

Устранение проблемы

Убедитесь, что значение индекса, которое используется для доступа к элементам списка, является целым числом (например, 2, а не "2" или 2,0).
