Операция не выполнена

Операция не выполнена — это стандартное сообщение об ошибке, которое указывает, что в одном из узлов не удалось выполнить операцию. Обычно вслед за сообщением об ошибке приводится дополнительная информация о сбое при выполнении операции и его причинах. Ниже перечислены некоторые распространенные операции, вызывающие данную ошибку.

  • IronPythonEvaluator.EvaluateIronPythonScript — это сообщение отображается, если узлу с кодом python не удалось корректно выполнить операцию. Этот сбой может быть вызван множеством причин, однако чаще всего это происходит из-за ошибки в коде python. Это сообщение об ошибке часто сопровождается дополнительной информацией о строке кода python, которую не удалось выполнить, и причинах сбоя. Дополнительные сведения об этой ошибке можно узнать здесь.
  • Element.SetParameterByName — дополнительные сведения об этой ошибке можно узнать здесь.
  • Floor.ByOutlineTypeAndLevel — дополнительные сведения об этой ошибке можно узнать здесь.
  • Dictionary.ByKeysValues — дополнительные сведения об этой ошибке можно узнать здесь.
  • List.GetItemAtIndex — дополнительные сведения об этой ошибке можно узнать здесь.
