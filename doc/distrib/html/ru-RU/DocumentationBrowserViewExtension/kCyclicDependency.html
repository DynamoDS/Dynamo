Существует циклическая зависимость между двумя переменными

Это предупреждение указывает на попытку передать выходные данные узла в качестве входных данных того же узла. Это приведет к созданию циклической зависимости, которая недопустима в Dynamo.

Устранение проблемы

Убедитесь, что выходные данные узла не передаются обратно в качестве входных данных для этого узла. При необходимости для создания цикла можно использовать узел LoopWhile, язык DesignScript или Python, как показано в этих примерах.
