Errore durante l'apertura del file

Questo è un messaggio di errore generico che indica che il file che si sta utilizzando non può essere aperto dalla versione di Dynamo in uso.

Come risolvere questo problema

Verificare che il file che si sta tentando di aprire sia stato creato utilizzando la stessa versione o una versione precedente di Dynamo rispetto alla versione in esecuzione sul computer in uso (ad es., se il file è stato creato in Dynamo 1.X, è possibile aprirlo con Dynamo 1.X o 2.X, ma se il file è stato creato in Dynamo 2.X, è possibile aprirlo solo con Dynamo 2.X o versione successiva).
