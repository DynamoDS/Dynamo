Browser documentazione Dynamo

Welcome to the Documentation Browser! Here, you can find out more about nodes, settings, features, and errors—without having to leave Dynamo.

Cosa è possibile trovare nel Browser della documentazione?

Node information: Want to learn more about a node? Select the node in the workspace and hit F1. This panel will show you information about how the node works, its inputs and outputs, and more. For several nodes, you can also interact with a sample graph showcasing use of the node in context.

Node issue documentation: Did you come across an unfamiliar node error or warning? Click the Learn More link in the error/warning tooltip to access more information, instructions to fix the issue, and links to forum discussions. If you don’t see a Learn More link, documentation is not yet available for that issue.

Informazioni su funzionalità e impostazioni: diverse finestre di dialogo di Dynamo includono icone con il punto interrogativo su cui è possibile fare clic per ulteriori informazioni sulla funzionalità o sull'impostazione.
