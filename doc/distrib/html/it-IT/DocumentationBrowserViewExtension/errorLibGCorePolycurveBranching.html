Gli elementi PolyCurve potrebbero (o meno) essere ramificati

Questo messaggio di errore indica che il nodo della geometria in uso sta eliminando i valori dopo la posizione decimale nelle impostazioni di grandi dimensioni. Ad esempio, i punti che in precedenza avevano valori X di [0.1, 0.2, 0.3, 5, 10], ora presentano valori [0, 0, 0, 5, 10].

Come risolvere questo problema

È possibile provare ad utilizzare Geometry.Scale per modificare le impostazioni di scala della geometria. Medio è l'impostazione di default per la scala della geometria.
