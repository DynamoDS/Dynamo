A conversão da matriz causará a redução da classificação da matriz

Essa mensagem de erro indica que você está usando uma lista de listas que não está formatada corretamente. Por exemplo, usar uma lista como [1,2,3,[4,5,6],7] em que os elementos estão em diferentes níveis acionará esse erro, mas usar uma lista como [[1,2,3],[4,5,6,7]] não acionará, porque ela tem consistentemente dois níveis de profundidade.

Como corrigir

Para solucionar esse problema, assegure-se de que a lista que você esteja passando como uma entrada para o nó tem níveis consistentes (ou seja, cada elemento pode ser acessado usando o mesmo nível da lista). Outra opção é usar List.NormalizeDepth, o que pode tornar os níveis de lista consistentes, mas pode não formatar a lista exatamente como você deseja.
