Método ou nó obsoleto

A mensagem de aviso Obsoleto significa que os autores do Dynamo desejam remover esse método ou nó no futuro, e você deve tentar usar diferentes métodos ou nós para obter a mesma funcionalidade. Esse aviso é uma forma de notificar os usuários do Dynamo de que o método será removido no futuro sem removê-lo imediatamente para que eles tenham tempo de atualizar os gráficos e os modelos.

Como corrigir

Tente usar nós diferentes para obter a mesma funcionalidade. Essa mensagem de erro é normalmente seguida por sugestões dos métodos ainda disponíveis para uso. Observe que é uma prática recomendada não usar nós ou métodos obsoletos.
