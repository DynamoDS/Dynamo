Espera tipos de argumento, mas foi chamado com

Essa mensagem de erro indica que o nó que você está usando está esperando um determinado tipo de entrada, mas recebeu um tipo de entrada que não pode usar. É normalmente seguido por informações adicionais sobre os tipos que foram passados para o nó que você está usando e os tipos que o nó esperava receber. Algumas das operações mais comuns que acionam esse erro:

  • Element.SetParameterByName – tente usar o nó Element.Parameters para verificar os parâmetros no elemento. Leia mais publicações no fórum Dynamobim que abordam “expects argument type setparameter”.
  • Geometry.BoundingBox – tente usar Element.BoundingBox em vez disso, conforme discutido aqui. Leia mais publicações no fórum Dynamobim que abordam “expects argument type geometry”.

Como corrigir

A seção acima menciona algumas abordagens que você pode adotar para recuperar desse erro. Em geral, verifique quais tipos de entrada o nó que você está usando espera. Se possível, altere as entradas passadas para o nó para os tipos esperados ou verifique se outros nós podem ser usados para obter a funcionalidade desejada. Observe que os tipos de entrada podem ser verificados usando o nó Object.Type.
