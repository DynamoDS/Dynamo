Falha na operação

Falha na operação é uma mensagem de erro genérica que indica que não foi possível executar um dos nós. É normalmente seguida de informações adicionais sobre qual operação falhou e por que a falha ocorreu. Algumas das operações mais comuns que falham:

  • IronPythonEvaluator.EvaluateIronPythonScript – essa mensagem é exibida quando um nó que executa o código python não é executado corretamente. Há muitos motivos para isso, mas normalmente isso acontece porque o código python contém uma falha. Essa mensagem de erro é frequentemente exibida com informações adicionais sobre a linha do código python que não foi executada e por que isso ocorreu. Leia mais sobre essa mensagem de erro aqui.
  • Element.SetParameterByName – leia mais sobre essa mensagem de erro aqui.
  • Floor.ByOutlineTypeAndLevel – leia mais sobre essa mensagem de erro aqui.
  • Dictionary.ByKeysValues – leia mais sobre essa mensagem de erro aqui.
  • List.GetItemAtIndex – leia mais sobre essa mensagem de erro aqui.
