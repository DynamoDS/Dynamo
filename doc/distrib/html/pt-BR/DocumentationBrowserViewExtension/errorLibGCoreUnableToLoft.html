Não é possível elevar

Esse aviso significa que você está tentando fazer algo que a elevação não será capaz de fazer. Na maioria dos casos, esse aviso é seguido por uma mensagem mais específica que deve informar mais sobre o motivo do aviso.

Como corrigir

Alguns dos motivos gerais para esse aviso ocorrer:

  • Você está tentando criar uma forma elevada entre um único perfil e dois perfis.
  • Os perfis não são curvas fechadas.

Mais informações
