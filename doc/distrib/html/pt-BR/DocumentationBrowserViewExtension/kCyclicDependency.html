Existe uma dependência cíclica entre duas variáveis

Essa mensagem de aviso indica que você está tentando passar a saída de um nó para a entrada daquele mesmo nó. Isso criará uma dependência cíclica que não é permitida no Dynamo.

Como corrigir

Assegure-se de que não esteja passando a saída de um nó de volta para a entrada daquele mesmo nó. Se você precisar criar um Loop, poderá usar o nó LoopWhile, criar um Loop usando DesignScript ou criar um loop usando o Python, conforme mostrado nestes exemplos.

Mais informações

Para saber mais, consulte as publicações no fórum Dynamobim que abordam “cyclic dependency exists between two variables”.
