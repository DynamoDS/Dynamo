Não é possível converter o valor nulo em tipo

Esse aviso significa que o nó que você está tentando usar está esperando uma entrada de determinado tipo de dados e que esse tipo não pode ser nulo. Nulo significa simplesmente “Nenhum objeto”, portanto o nó produz um aviso dizendo que você está tentando inserir nenhum objeto no nó, o que não é permitido.

Como corrigir

O erro pode ser causado por outros nós no gráfico. Verifique os nós a montante para verificar se eles são executados corretamente; é comum que os nós a montante não sejam executados corretamente e passem valores nulos a jusante até um nó que, em seguida, exibe essa mensagem de erro. Comece no nó que está causando esse aviso e verifique a saída de cada nó na direção inversa até encontrar o nó que está causando esse problema.

Se a mensagem de erro indicar qual parâmetro é nulo, verifique os nós no caminho a montante do parâmetro para ver se eles são executados corretamente e passam valores não nulos a jusante.
