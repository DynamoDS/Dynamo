A união de curva produziu mais de uma fiação na PolyCurve

Essa mensagem de erro indica que a saída de linhas por um nó PolyCurve não forma uma curva fechada.

Como corrigir

Isso pode dever-se à saída do nó PolyCurve ter valores nulos. Verifique em cada saída da lista pelo nó PolyCurve se as curvas se conectam em uma linha contínua. Uma forma de fazer isso é ocultar a visualização da geometria de todos os nós (clique com o botão direito do mouse no nó ou na seleção de nós para ocultar a visualização da geometria) e, em seguida, List.GetItemAtIndex, uma lista de cada vez para visualizar.

Além disso, esse erro normalmente aparece com Limites de ambiente. Tente usar Room.CenterBoundary para extrair as curvas.
