Tem alguns argumentos inválidos

Essa mensagem de erro indica que o nó que você está usando está esperando um determinado tipo de dados de entrada, mas recebeu um tipo de entrada que não pode usar. É semelhante ao erro “Espera tipos de argumento, mas foi chamado com” e normalmente exibe informações adicionais que podem ajudar a identificar quais tipos de argumentos você precisa passar para o nó que está usando.

Como corrigir

Em geral, verifique quais tipos de entrada o nó que você está usando espera e tente determinar se (a) o tipo correto de entrada é passado para o nó ou (b) é possível alterar o tipo de entrada para algo que o nó pode usar (por exemplo, alterando a entrada para um tipo diferente, conforme mostrado neste exemplo).

Como exemplo, quando você vê algo como Aviso: Falha na operação do Element.SetParameterByName. A melhor correspondência do método sobrecarregado para “Revit.Elements.InternalUtilities.ElementUtils.SetParameterValue(Autodesk.Revit.DB.Parameter, double)” tem alguns argumentos inválidos as informações adicionais entre parênteses podem ajudá-lo a entender qual é o problema. Nesse caso, SetParameterValue(Autodesk.Revit.DB.Parameter, double) mostra que o nó que você está usando está esperando uma entrada de tipo double, de modo que qualquer outro tipo de entrada não funcionará.
