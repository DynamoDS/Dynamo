O índice está fora da faixa especificada

Essa mensagem de erro significa que a lista não contém um elemento na posição de índice fornecida. Por exemplo, se a lista for [a, b, c, d] – que tem elementos em índices 0, 1, 2 e 3 – e você tentar acessar o elemento no índice 5, será exibida uma mensagem de erro, pois somente os elementos nos índices 0 até 3 existem na lista.

Como corrigir

Assegure-se de que a variável de índice que você está usando para acessar os elementos na lista seja maior ou igual a 0 e estritamente menor que o tamanho da lista. Os índices de lista vão de 0 até o tamanho da lista menos 1. Isso significa que se você tiver 10 elementos na lista e desejar acessar o 10º elemento, esse elemento estará no índice 9.
