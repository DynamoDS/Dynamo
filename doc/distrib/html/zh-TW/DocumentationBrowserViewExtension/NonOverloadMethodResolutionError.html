預期引數類型，卻使用...呼叫

此錯誤訊息表示您使用的節點預期收到某種類型的輸入，卻收到無法使用的輸入類型。後面通常會接哪些類型傳遞至您使用的節點，以及節點預期收到哪些類型的相關資訊。最常觸發此錯誤的作業包括:

  • Element.SetParameterByName - 嘗試使用 Element.Parameters 節點檢查元素中的參數。 請閱讀更多 Dynamobim 論壇上討論「expects argument type setparameter」的貼文。
  • Geometry.BoundingBox - 請依照這裡的討論，嘗試改用 Element.BoundingBox。 請閱讀更多 Dynamobim 論壇上討論「expects argument type geometry」的貼文。

如何修正

上一段提及一些您可以採取而從此錯誤恢復的方法。一般而言，請查看您使用的節點預期收到哪些輸入類型。如果可以，請將傳入節點的輸入變更為其預期的類型，或查看是否可以使用其他節點來實現您想要的功能。請注意，您可以使用 Object.Type 節點檢查輸入類型。
