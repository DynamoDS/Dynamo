找到多個定義

此錯誤訊息表示 DynamoCore 和您為了圖表而載入的不同 Dynamo 套件中使用了相同的方法或類別名稱，但是 Dynamo 不知道要使用哪個方法或類別 (亦即從哪個套件)。

這個警告通常是在 CodeBlock 中使用 DesignScript 時發生。例如，如果您嘗試在 CodeBlock 中展開清單，要使用 List.Flatten()。如果您載入的其中一個協力廠商套件也有一個稱為 List 的類別，您就會看到這個警告。

如何修正

若要修正此錯誤，請為您嘗試使用的函數包括完整的名稱空間。例如，如果您執行 List.Flatten() 並看到此錯誤訊息，請嘗試改用 DSCore.List.Flattte() (DSCore 代表「Design Script Core」，是 Dynamo 隨附的一個預設套件)。
