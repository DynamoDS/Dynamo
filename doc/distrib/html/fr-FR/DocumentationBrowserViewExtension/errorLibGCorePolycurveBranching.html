Les polycourbes peuvent être branchées (ou non)

Ce message d'erreur indique que le noeud de géométrie que vous utilisez retire les valeurs après la virgule lorsque l'option Très grand est activée dans les paramètres. Par exemple, les points qui avaient précédemment des valeurs X égales à [0.1, 0.2, 0.3, 5, 10] ont désormais des valeurs égales à [0, 0, 0, 5, 10].

Comment corriger cette erreur?

Vous pouvez utiliser Geometry.Scale pour modifier les paramètres de mise à l'échelle de la géométrie. Moyen est le paramètre par défaut pour l'échelle de géométrie.

Plus d'informations

Pour en savoir plus, consultez les billets publiés sur le forum Dynamobim sur le sujet "polycurves may be branching" (les polycourbes peuvent être branchées).
