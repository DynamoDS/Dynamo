La jonction des courbes a produit plusieurs WIRE dans la polycourbe

Ce message d'erreur indique que les lignes générées par un noeud de polycourbe ne forment pas une courbe fermée.

Comment corriger cette erreur?

Cela peut être dû au fait que la sortie du noeud contient des valeurs NULL. Vérifiez dans chaque sortie de liste du noeud de polycourbe si les courbes se rejoignent en une ligne continue. Pour ce faire, vous pouvez masquer l'aperçu de la géométrie de tous les noeuds (cliquez avec le bouton droit de la souris sur le noeud ou la sélection de noeuds pour masquer l'aperçu de la géométrie), puis exécuter List.GetItemAtIndex sur une liste à la fois pour l'afficher.

En outre, cette erreur survient souvent avec les limites de pièce. Essayez plutôt d'utiliser Room.CenterBoundary pour extraire les courbes.

Plus d'informations

Pour en savoir plus, consultez les billets publiés sur le forum Dynamobim sur le sujet "curve join produced more than one wire in polycurve" (la jonction des courbes a produit plusieurs fils dans la polycourbe).
