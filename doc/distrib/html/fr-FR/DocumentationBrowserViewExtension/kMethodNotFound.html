Méthode introuvable

Ce message d'erreur signifie que Dynamo ne peut pas trouver de définition pour la méthode DesignScript que vous essayez d'utiliser.

Comment corriger cette erreur?

Si vous définissez une méthode personnalisée dans un Code Block, assurez-vous que votre code DesignScript est correct. Veillez à ce que le nom de la méthode défini dans votre CodeBlock corresponde au nom de la méthode que vous appelez. Pour plus de détails, consultez le guide de DesignScript.

Plus d'informations

Pour en savoir plus, consultez les billets publiés sur le forum Dynamobim sur le sujet "method not found" (méthode introuvable).
