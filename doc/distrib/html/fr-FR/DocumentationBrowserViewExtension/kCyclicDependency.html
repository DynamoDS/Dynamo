Une dépendance cyclique existe entre deux variables

Ce message d'avertissement indique que vous essayez de transmettre la sortie d'un noeud à l'entrée de ce même noeud. Cela créera une dépendance cyclique, ce qui n'est pas autorisé dans Dynamo.

Comment corriger cette erreur?

Veillez à ne pas transmettre la sortie d'un noeud à l'entrée de ce même noeud. Si vous devez créer une boucle, vous pouvez utiliser le noeud LoopWhile, créer une boucle à l'aide de DesignScript ou créer une boucle à l'aide de Python comme indiqué dans ces exemples.
