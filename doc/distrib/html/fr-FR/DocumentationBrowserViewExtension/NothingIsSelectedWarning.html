Aucun élément n'est sélectionné

Cette erreur indique que vous n'avez sélectionné aucun élément dans un noeud de sélection (par exemple, Select Model Element dans Dynamo for Revit).

Comment corriger cette erreur?

Cliquez sur le noeud que vous utilisez et sélectionnez un élément dans Revit. Pour ce faire, vous devez passer de la fenêtre Dynamo à la fenêtre Revit (ou celle de tout autre environnement hôte).

Plus d'informations

Pour en savoir plus, consultez les billets publiés sur le forum Dynamobim sur le sujet "nothing is selected" (aucun élément n'est sélectionné).
