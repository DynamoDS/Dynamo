Possède plusieurs arguments non valides

Ce message d'erreur indique que le noeud que vous utilisez attend un certain type de données d'entrée mais qu'il a reçu un type d'entrée qu'il ne peut pas utiliser. Il est semblable à l'erreur "attend le ou les types d'argument, mais a été appelé avec", et affiche généralement des informations supplémentaires qui peuvent vous aider à identifier les types d'argument à transmettre au noeud que vous utilisez.

Comment corriger cette erreur?

En général, vérifiez les types d'entrée attendus par le noeud que vous utilisez, et essayez de déterminer (a) si le type d'entrée correct est transmis au noeud ou (b) si vous pouvez modifier le type d'entrée pour qu'il puisse être utilisé par le noeud (par exemple, en changeant l'entrée en un type différent, comme indiqué dans cet exemple).

Par exemple, lorsque vous voyez un message tel que Warning: Element.SetParameterByName operation failed. The best overloaded method match for ‘Revit.Elements.InternalUtilities.ElementUtils.SetParameterValue(Autodesk.Revit.DB.Parameter, double)’ has some invalid arguments, les informations supplémentaires entre parenthèses peuvent vous aider à comprendre le problème. Dans ce cas, SetParameterValue(Autodesk.Revit.DB.Parameter, double) indique que le noeud que vous utilisez est une entrée de type double. Ainsi, tout autre type d'entrée ne fonctionnera pas.
