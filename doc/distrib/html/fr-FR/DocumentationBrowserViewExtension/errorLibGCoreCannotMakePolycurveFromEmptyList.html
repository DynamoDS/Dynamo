Impossible de créer une polycourbe depuis une liste vide

Cet avertissement se produit si vous entrez une liste vide lorsque vous essayez de créer une polycourbe.

Comment corriger cette erreur?

L'erreur peut être provoquée par d'autres noeuds dans le graphique. Vérifiez les noeuds en amont pour voir s'ils s'exécutent correctement. Il est courant que les noeuds en amont transmettent des listes vides à un noeud en aval qui affiche alors ce message d'erreur. Commencez à partir du noeud à l'origine de cet avertissement et remontez la chaîne, en vérifiant la sortie de chaque noeud jusqu'à ce que vous trouviez le noeud à l'origine de ce problème.

Plus d'informations

Pour en savoir plus, consultez les billets publiés sur le forum Dynamobim sur le sujet "cannot make a polycurve from an empty list" (impossible de créer une polycourbe à partir d'une liste vide).
