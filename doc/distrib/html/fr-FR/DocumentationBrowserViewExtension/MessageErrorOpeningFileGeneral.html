Erreur lors de l'ouverture du fichier

Ce message d'erreur générique indique que le fichier sur lequel vous travaillez ne peut pas être ouvert par votre version de Dynamo.

Comment corriger cette erreur?

Assurez-vous que le fichier que vous essayez d'ouvrir a été créé avec la même version ou une version antérieure de Dynamo que celle que vous exécutez sur votre ordinateur (par exemple, si le fichier a été créé dans Dynamo 1.X, vous pouvez l'ouvrir avec Dynamo 1.X ou 2.X, mais si le fichier a été créé dans Dynamo 2.X, vous pouvez uniquement l'ouvrir avec Dynamo 2.X ou version ultérieure).

Plus d'informations

Pour en savoir plus, consultez les billets publiés sur le forum Dynamobim sur le sujet "error opening file" (erreur lors de l'ouverture du fichier).
