Navigateur de documentation Dynamo

Bienvenue dans le navigateur de documentation ! Vous pouvez en savoir plus sur les noeuds, les paramètres, les fonctions et les erreurs, le tout sans quitter Dynamo.

Que puis-je trouver dans le navigateur de documentation ?

Informations sur un noeud : vous souhaitez en savoir plus sur un noeud ? Sélectionnez le noeud dans l'espace de travail et appuyez sur F1. Ce groupe de fonctions affiche des informations sur le fonctionnement du noeud, ses entrées et sorties, et bien plus encore.Pour plusieurs noeuds, vous pouvez également interagir avec un exemple de graphique illustrant l'utilisation du noeud en contexte.

Documentation sur les problèmes de noeud : vous avez rencontré une erreur de noeud inhabituelle ou un avertissement inconnu ? Cliquez sur le lien En savoir plus dans l'info-bulle d'erreur/avertissement pour accéder à plus d'informations, à des instructions permettant de résoudre le problème et à des liens vers les forums de discussion. S'il n'y a pas de lien En savoir plus, cela signifie que la documentation n’est pas encore disponible pour ce problème.

Informations sur les fonctionnalités et les paramètres : plusieurs boîtes de dialogue Dynamo incluent des icônes de point d'interrogation sur lesquelles vous pouvez cliquer pour en savoir plus sur la fonctionnalité ou le paramètre.
