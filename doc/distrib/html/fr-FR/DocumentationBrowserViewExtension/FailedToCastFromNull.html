Une valeur NULL ne peut pas être projetée sur le type

Cet avertissement signifie que le noeud que vous essayez d'utiliser attend une entrée d'un certain type de données et que ce type ne peut pas être NULL. NULL signifie simplement "Aucun objet". Par conséquent, le noeud déclenche un avertissement indiquant que vous essayez d'entrer aucun objet dans le noeud, ce qui n'est pas autorisé.

Comment corriger cette erreur?

L'erreur peut être provoquée par d'autres noeuds dans le graphique. Vérifiez les noeuds en amont pour voir s'ils s'exécutent correctement. Il est courant que ces noeuds ne s'exécutent pas correctement et transmettent des valeurs NULL à un noeud en aval qui affiche alors ce message d'erreur. Commencez à partir du noeud à l'origine de cet avertissement et remontez la chaîne, en vérifiant la sortie de chaque noeud jusqu'à ce que vous trouviez le noeud à l'origine de ce problème.

Si le message d'erreur indique le paramètre qui a la valeur NULL, vérifiez les noeuds sur le chemin en amont du paramètre pour voir s'ils s'exécutent correctement et transmettent des valeurs non NULL en aval.
