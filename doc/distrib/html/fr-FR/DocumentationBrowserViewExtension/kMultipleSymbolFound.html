Plusieurs définitions trouvées

Ce message d'erreur signifie que le même nom de méthode ou de classe est utilisé dans DynamoCore et différents packages Dynamo que vous chargez pour votre graphique, et que Dynamo ne sait pas quelle méthode ou classe (package d'origine) utiliser.

Cet avertissement est généralement généré lors de l'utilisation de DesignScript dans un code de bloc. Par exemple, si vous essayez d'aplanir une liste dans un Code Block, vous devez utiliser List.Flatten(). Si l'un des packages tiers que vous chargez comporte également une classe appelée List, vous verrez cet avertissement.

Comment corriger cette erreur?

Pour corriger cette erreur, incluez l'espace de nom complet de la fonction que vous essayez d'utiliser. Par exemple, si vous exécutez List.Flatten() et que ce message d'erreur s'affiche, essayez d'utiliser plutôt DSCore.List.Flatten() (DSCore signifie "Design Script Core" et est l'un des packages par défaut fournis avec Dynamo).
