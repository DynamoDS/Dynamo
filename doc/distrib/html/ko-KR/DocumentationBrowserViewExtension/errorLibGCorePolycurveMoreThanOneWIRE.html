곡선 결합에서 Polycurve에 두 개 이상의 WIRE를 생성함

이 오류 메시지는 Polycurve 노드에서 출력한 선이 닫힌 곡선을 형성하지 않음을 나타냅니다.

수정 방법

이는 Polycurve 노드의 출력에 null 값이 포함되어 있기 때문일 수 있습니다. 곡선이 하나의 연속 선으로 연결된 경우 Polycurve 노드를 사용해 각 리스트 출력 내부를 확인합니다. 이 작업을 수행하는 한 가지 방법은 모든 노드의 형상 미리보기를 숨긴(형상 미리보기를 숨기려면 노드를 마우스 오른쪽 버튼으로 클릭하거나 노드를 선택) 다음, 한 번에 하나의 리스트를 List.GetItemAtIndex하여 보는 것입니다.

또한 이 오류는 흔히 룸 경계에서 발생합니다. 대신 Room.CenterBoundary를 사용하여 곡선을 추출해 보십시오.
