두 변수 사이에 주기적 의존성이 존재함

이 경고 메시지는 노드의 출력을 동일한 해당 노드의 입력에 전달하려고 시도 중임을 나타냅니다. 그러면 Dynamo에서 허용되지 않는 주기적 의존성이 만들어집니다.

수정 방법

노드의 출력을 동일한 노드의 입력으로 다시 전달하지 않도록 하십시오. 루프를 만들어야 하는 경우 LoopWhile 노드를 사용하거나, DesignScript를 사용하여 루프를 만들거나, 이러한 예에 나와 있듯이 Python을 사용하여 루프를 만들 수 있습니다.
