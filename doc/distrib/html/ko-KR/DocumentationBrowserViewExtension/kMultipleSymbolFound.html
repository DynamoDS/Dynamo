여러 정의를 찾음

이 오류 메시지는 DynamoCore와 그래프를 위해 로드 중인 다른 Dynamo 패키지에서 동일한 메서드 또는 클래스 이름이 사용되어 Dynamo에서 어떤 메서드나 클래스(즉, 어떤 패키지)를 사용할지 알지 못한다는 의미입니다.

이 경고는 보통 CodeBlock에서 DesignScript를 사용할 때 발생합니다. 예를 들어, CodeBlock에서 리스트를 단순화하려는 경우 List.Flatten()을 사용합니다. 로드 중인 타사 패키지 중 하나에 List라는 클래스도 있으면 이 경고가 표시됩니다.

수정 방법

이 오류를 해결하려면 사용하려는 함수의 전체 네임스페이스를 포함합니다. 예를 들어, List.Flatten()을 실행 중일 때 이 오류 메시지가 표시되는 경우 DSCore.List.Flatten()(DSCore는 "Design Script Core"를 나타내며 Dynamo와 함께 제공되는 기본 패키지 중 하나임)을 대신 사용해 보십시오.
