メソッドが見つかりません

このエラー メッセージは、使用する DesignScript メソッドについて、Dynamo が定義を見つけることができないことを意味しています。

修正方法

Code Block 内でカスタム メソッドを定義する場合は、コードが有効な DesignScript であることを確認してください。 CodeBlock で定義されたメソッド名が、呼び出しているメソッド名と一致することを確認してください。 詳細については、Design Script のガイドを参照してください。

詳細情報

詳細については、DynamoBIM のフォーラムで「method not found」に関する投稿を参照してください。
