2 つの変数間に循環従属関係が存在します

この警告メッセージは、ノードの出力をその同じノードの入力に渡していることを示しています。 これにより、Dynamo では許可されていない循環従属が作成されます。

修正方法

ノードの出力を、その同じノードの入力に戻していないことを確認してください。 ループを作成する場合は、LoopWhile ノード、DesignScript、または次の例のように Python を使用して作成することができます。

詳細情報

詳細については、DynamoBIM のフォーラムで「cyclic dependency exists between two variables」に関する投稿を参照してください。
