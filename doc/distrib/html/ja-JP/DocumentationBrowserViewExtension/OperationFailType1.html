操作が失敗しました

「操作が失敗しました」は、いずれかのノードの実行が失敗したことを示す一般的なエラー メッセージです。 通常このメッセージの後には、失敗した操作とその理由に関する情報も表示されます。 失敗する最も一般的な操作には次のようなものがあります。

  • IronPythonEvaluator.EvaluateIronPythonScript - Python コードを実行しているノードが正しく実行されない場合、このメッセージが表示されます。 これには多くの理由がありますが、一般的には、Python コードにバグが含まれていることが原因です。 このエラー メッセージは頻繁に表示され、実行に失敗した Python コードの行とその原因に関する情報も表示されます。 このエラー メッセージの詳細は、こちらを参照してください。
  • Element.SetParameterByName - このエラー メッセージの詳細は、 こちらを参照してください。
  • Floor.ByOutlineTypeAndLevel - このエラー メッセージの詳細は、こちらを参照してください。
  • Dictionary.ByKeysValues - このエラー メッセージの詳細は、こちらを参照してください。
  • List.GetItemAtIndex - このエラー メッセージの詳細は、こちらを参照してください。

詳細情報

詳細については、DynamoBIM のフォーラムで「operation failed」に関する投稿を参照してください。
