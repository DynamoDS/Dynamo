複数の設定が見つかりました

このエラー メッセージは、グラフにロードしている DynamoCore などの各種の Dynamo パッケージで、同じメソッドまたはクラス名が使用されており、Dynamo でどのメソッドまたはクラスを使用するか(つまり、どのパッケージから使用するか)を判断できないことを意味しています。

通常この警告は、CodeBlock 内で DesignScript を使用しているときに発生します。 たとえば、CodeBlock 内のリストをフラット化する場合は、List.Flatten()を使用します。 ロードしているサード パーティ製パッケージのいずれかに List というクラスがあると、この警告が表示されます。

修正方法

このエラーを修正するには、使用する関数の完全な名前空間を含めます。 たとえば、List.Flatten()の実行中にこのエラー メッセージが表示される場合は、代わりに DSCore.List.Flatten()を使用します(DSCore は「Design Script Core」を表し、Dynamo に付属の既定のパッケージの 1 つです)。

詳細情報

詳細については、DynamoBIM のフォーラムで「multiple definitions found」に関する投稿を参照してください。
