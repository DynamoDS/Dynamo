引数タイプが必要ですが、次で呼び出されました

このエラー メッセージは、使用しているノードに特定のタイプの入力が必要であるが、使用できないタイプの入力を受け取ったことを示しています。 通常このメッセージの後には、使用しているノードに渡されたタイプと、ノードが受け取る必要があるタイプに関する情報も表示されます。 このエラーの原因となる最も一般的な操作には次のようなものがあります。

  • Element.SetParameterByName - 要素のパラメータの確認には、ノード Element.Parameters を使用してください。 詳細は、DynamoBIM のフォーラムで「expects argument type setparameter」に関する投稿を参照してください。
  • Geometry.BoundingBox - こちらで説明されているように、代わりに Element.BoundingBox を使用してください。 詳細は、DynamoBIM のフォーラムで「expects argument type geometry」に関する投稿を参照してください。

修正方法

上記のセクションでは、このエラーを修復するために必要な方法についていくつか説明しています。 一般的には、使用しているノードに必要な入力タイプを確認します。 可能な場合は、ノードに渡される入力を必要なタイプに変更するか、他のノードを使用して目的の機能を実現できるかどうかを確認します。 入力タイプは、Object.Type ノードを使用して確認できます。

詳細情報

詳細については、DynamoBIM のフォーラムで「expects argument type」に関する投稿を参照してください。
