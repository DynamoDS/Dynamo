NULL 値をタイプにキャストできません

この警告は、使用しているノードに特定のデータ タイプの入力が必要であり、そのタイプを NULL にできないことを意味しています。 NULL は単に「オブジェクトがない」ことを意味しています。したがって、オブジェクトなしでノードに入力しようとしているため入力できないという警告がノードに表示されます。

修正方法

このエラーは、グラフ内の他のノードが原因で発生する可能性があります。 上流のノードが正しく実行されるかどうかを確認してください。通常、上流のノードが正しく実行されず、下流のノードに NULL 値が渡されると、このエラー メッセージが表示されます。 この警告の原因となっているノードから上流方向にたどって、この問題の原因となっているノードが見つかるまで各ノードの出力を確認します。

エラー メッセージにどのパラメータが NULL となっているかが示されている場合は、そのパラメータの上流のパスにあるノードについて、正しく実行されて NULL 以外の値を下流に渡すかどうかを確認します。

詳細情報

詳細については、「NULL 値をタイプにキャストできません」を参照してください。
