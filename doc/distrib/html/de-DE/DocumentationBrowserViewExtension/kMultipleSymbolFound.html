Mehrere Definitionen gefunden

Diese Fehlermeldung weist darauf hin, dass dieselbe Methode oder derselbe Klassenname in DynamoCore und verschiedenen Dynamo-Paketen verwendet wird, die Sie für Ihr Diagramm laden. Dynamo kann nicht erkennen, welche Methode oder Klasse (d. h. von welchem Paket) Sie verwenden möchten.

Diese Warnung wird in der Regel angezeigt, wenn Sie DesignScript in einem CodeBlock verwenden. Wenn Sie beispielsweise versuchen, eine Liste in einem CodeBlock abzuflachen, verwenden Sie List.Flatten(). Wenn eines der von Ihnen geladenen Drittanbieterpakete ebenfalls eine Klasse namens List enthält, wird diese Warnung angezeigt.

So beheben Sie Fehler

Um diesen Fehler zu beheben, schließen Sie den vollständigen Namensbereich der Funktion ein, die Sie verwenden möchten. Wenn Sie beispielsweise List.Flatten() ausführen und diese Fehlermeldung angezeigt wird, versuchen Sie, stattdessen DSCore.List.Flatten() zu verwenden (DSCore steht für "Design Script Core" und ist eines der vorgegebenen Pakete, die mit Dynamo geliefert werden).
