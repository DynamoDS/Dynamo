Fehler beim Öffnen der Datei

Dies ist eine allgemeine Fehlermeldung, die darauf hinweist, dass die Datei, mit der Sie arbeiten, nicht in Ihrer Version von Dynamo geöffnet werden kann.

So beheben Sie Fehler

Stellen Sie sicher, dass die Datei, die Sie öffnen möchten, mit derselben oder einer früheren Version von Dynamo als die Version erstellt wurde, die auf Ihrem Computer ausgeführt wird. (Beispiel: Wenn die Datei in Dynamo 1.X erstellt wurde, können Sie sie mit Dynamo 1.X oder 2.X öffnen, aber wenn die Datei in Dynamo 2.X erstellt wurde, können Sie sie nur mit Dynamo 2.x oder höher öffnen).
