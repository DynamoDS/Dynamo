Konvertieren eines Arrays würde eine Reduktion des Array-Rangs bewirken

Diese Fehlermeldung weist darauf hin, dass Sie eine Liste von Listen verwenden, die nicht korrekt formatiert ist. Wenn Sie beispielsweise eine Liste wie [1,2,3,[4,5,6],7] verwenden, wobei die Elemente auf verschiedenen Ebenen sind, wird dieser Fehler ausgelöst. Eine Liste wie [[1,2,3],[4,5,6,7]] hingegen löst den Fehler nicht aus, da sie einheitlich zwei Ebenen tief ist.

So beheben Sie Fehler

Um dieses Problem zu beheben, stellen Sie sicher, dass die Liste, die Sie als Eingabe an Ihren Block weitergeben, über einheitliche Ebenen verfügt (d. h. auf jedes Element kann mit derselben Ebene in der Liste zugegriffen werden). Eine andere Möglichkeit ist die Verwendung von List.NormalizeDepth, wodurch Ihre Listenebenen einheitlich gemacht werden können. Die Liste wird jedoch möglicherweise nicht genau wie gewünscht formatiert.
