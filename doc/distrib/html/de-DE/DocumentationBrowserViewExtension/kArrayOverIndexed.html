Der Index liegt außerhalb des angegebenen Bereichs

Diese Fehlermeldung weist darauf hin, dass Ihre Liste kein Element an der angegebenen Indexposition enthält. Wenn Ihre Liste beispielsweise [a, b, c, d] ist, wobei die Liste Elemente bei den Indizes 0, 1, 2 und 3 hat, und Sie versuchen, auf das Element unter Index 5 zuzugreifen, wird eine Fehlermeldung angezeigt, da in der Liste nur Elemente an den Indizes 0 bis 3 vorhanden sind.

So beheben Sie Fehler

Stellen Sie sicher, dass die Indexvariable, die Sie zum Zugriff auf Elemente in der Liste verwenden, größer als oder gleich 0 und ausschließlich kleiner als die Listengröße ist. Listenindizes reichen von 0 bis zur Listengröße minus 1. Dies bedeutet, dass wenn Sie über 10 Elemente in der Liste verfügen und auf das 10. Element zugreifen möchten, sich das Element unter Index 9 befindet.
