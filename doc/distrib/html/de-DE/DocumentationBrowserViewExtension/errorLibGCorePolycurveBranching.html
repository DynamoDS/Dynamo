Polykurven sind möglicherweise (nicht) verzweigt

Diese Fehlermeldung weist darauf hin, dass der verwendete Geometrieblock Werte nach der Dezimalstelle in extra großen Einstellungen entfernt. Beispiel: Punkte, die bisher X-Werte von [0.1, 0.2, 0.3, 5, 10] hatten, haben nun [0, 0, 0, 5, 10] als Werte.

So beheben Sie Fehler

Sie können versuchen, die Geometrieskalierungseinstellungen mit Geometry.Scale zu ändern. Mittel ist die Vorgabeeinstellung für die Geometrieskalierung.
