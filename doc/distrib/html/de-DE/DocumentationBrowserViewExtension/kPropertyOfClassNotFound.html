Keine Eigenschaft gefunden

Diese Fehlermeldung weist darauf hin, dass Sie im verwendeten Block versuchen, auf eine Eigenschaft eines Objekts zuzugreifen, das diese Eigenschaft nicht besitzt.

Eigenschaften sind Werte, die in einem Objekt gespeichert sind. In Dynamo verfügt z. B. ein Punktobjekt über die Eigenschaften X, Y und Z, die die Koordinatenwerte dieses bestimmten Punkts darstellen. Ein Linienobjekt weist unterschiedliche Eigenschaften auf, wie z. B. Länge, die die Länge der gegebenen Linie darstellt. Das Linienobjekt verfügt nicht über die Eigenschaft X, Y und Z, die das Punktobjekt hat, und das Punktobjekt verfügt nicht über die Eigenschaft Länge, die das Linienobjekt hat.

Dieser Fehler tritt daher auf, wenn Sie versuchen, auf eine bestimmte Eigenschaft eines Objekts zuzugreifen, das diese Eigenschaft nicht besitzt. Wenn Sie beispielsweise versuchen, auf die Eigenschaft Länge des Punktobjekts zuzugreifen, wird "No property called Length on Point could be found" angezeigt, da das Punktobjekt die Eigenschaft Länge nicht enthält.

So beheben Sie Fehler

Stellen Sie sicher, dass das Objekt, das Sie als Eingabe für den Block verwenden, über die Eigenschaft verfügt, auf die Sie zugreifen möchten.
