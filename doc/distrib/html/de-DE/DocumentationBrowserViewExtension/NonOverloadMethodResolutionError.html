Erwartet Argumenttyp(en), wurde jedoch aufgerufen mit

Diese Fehlermeldung weist darauf hin, dass der verwendete Block einen bestimmten Eingabetyp erwartet, aber einen Eingabetyp erhalten hat, den er nicht verwenden kann. In der Regel folgen zusätzliche Informationen dazu, welche Typen an den verwendeten Block übergeben wurden und welche Typen der Block erwartet hat. Einige der häufigsten Vorgänge, die diesen Fehler auslösen, sind:

  • Element.SetParameterByName: Versuchen Sie, mithilfe des Blocks Element.Parameters die Parameter im Element zu überprüfen. Lesen Sie weitere Beiträge im DynamoBIM-Forum zum Thema "expects argument type setparameter".
  • Geometry.BoundingBox: Versuchen Sie stattdessen, Element.BoundingBox zu verwenden, wie hier beschrieben. Lesen Sie weitere Beiträge im DynamoBIM-Forum zum Thema "expects argument type geometry".

So beheben Sie Fehler

Im obigen Abschnitt werden einige Maßnahmen erwähnt, die Sie ergreifen können, um diesen Fehler zu beheben. Überprüfen Sie generell, welche Eingabetypen für den verwendeten Block erwartet werden. Ändern Sie nach Möglichkeit die Eingaben, die an den Block weitergegeben werden, in die erwarteten Typen, oder überprüfen Sie, ob andere Blöcke verwendet werden können, um die gewünschte Funktionalität zu erzielen. Beachten Sie, dass Eingabetypen mit dem Object.Type-Block überprüft werden können.
