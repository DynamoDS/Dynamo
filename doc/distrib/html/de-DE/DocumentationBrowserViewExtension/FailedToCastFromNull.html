Nullwert kann nicht in Typ umgewandelt werden

Diese Warnung bedeutet, dass der Block, den Sie verwenden möchten, eine Eingabe eines bestimmten Datentyps erwartet und dieser Typ nicht Null sein darf. Null bedeutet einfach "Kein Objekt". Daher gibt der Block eine Warnung aus, die besagt, dass Sie versuchen, kein Objekt in den Block einzugeben, was nicht zulässig ist.

So beheben Sie Fehler

Der Fehler kann durch andere Blöcke in Ihrem Diagramm verursacht werden. Überprüfen Sie die vorgelagerten Blöcke, um zu sehen, ob sie korrekt ausgeführt werden. Es kommt oft vor, dass die vorgelagerten Blöcke nicht korrekt ausgeführt werden und in der Folge Nullwerte an einen Block übergeben werden, der dann diese Fehlermeldung anzeigt. Beginnen Sie mit dem Block, der diese Warnung verursacht, und arbeiten Sie rückwärts, indem Sie die Ausgabe jedes Blocks überprüfen, bis Sie den Block finden, der dieses Problem verursacht.

Wenn die Fehlermeldung angibt, welcher Parameter Null ist, überprüfen Sie die Blöcke auf dem vorgelagert liegenden Pfad dieses Parameters, um zu ermitteln, ob sie korrekt ausgeführt werden und Werte, die nicht Null sind, an untergeordnete Blöcke übergeben werden.
