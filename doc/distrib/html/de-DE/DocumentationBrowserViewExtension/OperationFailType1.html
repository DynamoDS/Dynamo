Vorgang fehlgeschlagen

Operation failed ist eine allgemeine Fehlermeldung, die darauf hinweist, dass einer der Blöcke nicht ausgeführt werden konnte. In der Regel folgen zusätzliche Informationen darüber, welcher Vorgang fehlgeschlagen und warum der Fehler aufgetreten ist. Einige der häufigsten Vorgänge, die fehlschlagen, sind:

  • IronPythonEvaluator.EvaluateIronPythonScript: Diese Meldung wird angezeigt, wenn ein Block, der Ihren python-Code ausführt, nicht korrekt ausgeführt werden kann. Es gibt viele Gründe dafür, aber in der Regel geschieht dies, da der python-Code einen Fehler enthält. Diese Fehlermeldung wird häufig mit zusätzlichen Informationen zur Zeile des python-Codes angezeigt, der nicht ausgeführt werden konnte, und warum dies der Fall ist. Weitere Informationen zu dieser Fehlermeldung finden Sie hier.
  • Element.SetParameterByName: Weitere Informationen zu dieser Fehlermeldung finden Sie hier.
  • Floor.ByOutlineTypeAndLevel: Weitere Informationen zu dieser Fehlermeldung finden Sie hier.
  • Dictionary.ByKeysValues: Weitere Informationen zu dieser Fehlermeldung finden Sie hier.
  • List.GetItemAtIndex: Weitere Informationen zu dieser Fehlermeldung finden Sie hier.
