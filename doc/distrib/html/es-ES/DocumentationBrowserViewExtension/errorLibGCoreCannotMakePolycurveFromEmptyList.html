No se puede crear una PolyCurve a partir de una lista vacía

Esta advertencia se produce cuando se introduce una lista vacía al intentar crear una PolyCurve.

Cómo solucionar este error

El error puede deberse a otros nodos del gráfico. Compruebe los nodos ascendentes para ver si se ejecutan correctamente; es habitual que los nodos ascendentes transfieran listas vacías de forma descendente a un nodo que muestre a continuación este mensaje de error. Comience desde el nodo que provoca esta advertencia y retroceda en el proceso, comprobando la salida de cada nodo hasta que encuentre el nodo que provoca este problema.
