La conversión de la matriz provocaría una reducción de clasificación de matrices

Este mensaje de error indica que se está utilizando una lista de listas que no tiene el formato correcto. Por ejemplo, si se utiliza una lista como [1,2,3,[4,5,6],7] donde los elementos se encuentran en distintos niveles, se desencadenará este error, pero si se utiliza una lista como [[1,2,3],[4,5,6,7]], no se generará porque tiene dos niveles de profundidad.

Cómo solucionar este error

Para resolver este problema, asegúrese de que la lista que transfiere como entrada al nodo tiene niveles coherentes (es decir, se puede acceder a cada elemento mediante el mismo nivel de la lista). Otra opción es utilizar List.NormalizeDepth, lo que puede conseguir que los niveles de la lista sean coherentes, aunque es posible que la lista no presente exactamente el formato que desea.
