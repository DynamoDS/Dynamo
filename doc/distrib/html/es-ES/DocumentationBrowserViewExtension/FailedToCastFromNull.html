El valor nulo no se puede convertir en tipo

Esta advertencia indica que el nodo que está intentando utilizar espera una entrada de un determinado tipo de datos y que ese tipo no puede ser nulo. Un valor nulo simplemente indica "sin objeto", por lo que el nodo emite una advertencia que indica que no se está intentando introducir ningún objeto en el nodo, lo que no está permitido.

Cómo solucionar este error

El error puede deberse a otros nodos del gráfico. Compruebe los nodos ascendentes para ver si se ejecutan correctamente; es habitual que los nodos ascendentes no se ejecuten correctamente y transfieran los valores nulos de forma descendente a un nodo que muestre a continuación este mensaje de error. Comience desde el nodo que provoca esta advertencia y retroceda en el proceso, comprobando la salida de cada nodo hasta que encuentre el nodo que provoca este problema.

Si el mensaje de error indica el parámetro que es nulo, compruebe los nodos de la ruta ascendente de ese parámetro para ver si se ejecutan correctamente y transfieren los valores que no son nulos de forma descendente.
