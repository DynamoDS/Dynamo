Espera los tipos de argumentos, pero se ha llamado con

Este mensaje de error indica que el nodo que está utilizando espera un determinado tipo de entradas, pero ha recibido un tipo de entrada que no puede utilizar. Normalmente, después de este mensaje, se proporciona información adicional sobre los tipos que se han transferido al nodo que se está utilizando y los tipos que el nodo esperaba recibir. Algunas de las operaciones más habituales que generan este error son:

  • Element.SetParameterByName: pruebe a utilizar el nodo Element.Parameters para comprobar los parámetros del elemento. Lea más publicaciones del foro de Dynamobim sobre "expects argument type setparameter".
  • Geometry.BoundingBox: pruebe a utilizar Element.BoundingBox en su lugar, como se comenta aquí. Lea más publicaciones del foro de Dynamobim sobre "expects argument type geometry".

Cómo solucionar este error

En la sección anterior, se mencionan algunos métodos que puede utilizar para recuperarse de este error. En general, compruebe los tipos de entrada que espera el nodo que está utilizando. Si es posible, cambie las entradas transferidas al nodo a los tipos esperados o compruebe si se pueden utilizar otros nodos para obtener la función que desea. Tenga en cuenta que los tipos de entrada se pueden comprobar mediante el nodo Object.Type.
