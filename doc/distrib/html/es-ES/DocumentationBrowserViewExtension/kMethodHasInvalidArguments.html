Hay algunos argumentos no válidos

Este mensaje de error indica que el nodo que está utilizando espera un determinado tipo de datos de entrada, pero ha recibido un tipo de entrada que no puede utilizar. Es similar al error "Espera los tipos de argumentos, pero se ha llamado con" y, por lo general, muestra información adicional que puede ayudarle a identificar los tipos de argumentos que debe transferir al nodo que está utilizando.

Cómo solucionar este error

En general, compruebe los tipos de entrada que espera el nodo que está utilizando e intente determinar si (a) se ha transferido el tipo de entrada correcto al nodo o (b) puede cambiar el tipo de entrada a alguna que pueda usar el nodo (por ejemplo, mediante el cambio de la entrada a otro tipo, como se muestra en este ejemplo).

Por ejemplo, cuando aparece un mensaje como Warning: Element.SetParameterByName operation failed. The best overloaded method match for ‘Revit.Elements.InternalUtilities.ElementUtils.SetParameterValue(Autodesk.Revit.DB.Parameter, double)’ has some invalid arguments, la información adicional incluida entre paréntesis puede ayudarle a entender cuál es el problema. En este caso, SetParameterValue(Autodesk.Revit.DB.Parameter, double) muestra que el nodo que está utilizando espera una entrada de tipo double, por lo que cualquier otro tipo de entrada no funcionará.
