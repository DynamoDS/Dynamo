Los índices de lista deben ser numéricos

Este mensaje de error indica que está intentando acceder a un elemento de una lista mediante un valor de índice que no es un entero (es decir, un número entero).

A partir de Dynamo 2.0, las listas son colecciones de valores de una lista a los que solo se puede acceder individualmente mediante valores de índice de enteros (los valores de índice oscilan entre 0 y el tamaño de lista menos 1).

Aquí puede obtener más información sobre las diferencias entre Dynamo 1.X y Dynamo 2.X (incluido cómo ha cambiado el acceso a los elementos de una lista).

Cómo solucionar este error

Asegúrese de que el valor de índice que utiliza para acceder a los elementos de la lista sea únicamente un entero (por ejemplo, 2 y no "2" o 2.0).
