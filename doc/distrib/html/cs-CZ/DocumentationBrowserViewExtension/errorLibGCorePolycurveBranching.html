Objekty PolyCurve se mohou větvit

Tato chybová zpráva naznačuje, že používaný uzel geometrie v rozsáhlém nastavení zahazuje hodnoty za desetinnou tečkou. Například body, které dříve obsahovaly X hodnot [0.1, 0.2, 0.3, 5, 10], nyní obsahují hodnoty [0, 0, 0, 5, 10 ].

Jak provést opravu

Můžete zkusit změnit nastavení měřítka geometrie pomocí uzlu Geometry.Scale. Výchozí nastavení měřítka geometrie je střední.
