##Execution Interval
###Description
Forces an Execution after every interval

###Inputs
  * **ms** *(double)* - Delay in milliseconds

###Output
  * None


##Pause
###Description
Pauses execution for a given amount of time.

###Inputs
  * **ms** *(double)* - Delay in milliseconds

###Output
  * None
