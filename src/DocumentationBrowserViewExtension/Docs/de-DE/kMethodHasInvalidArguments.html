Enthält einige ungültige Argumente

Diese Fehlermeldung weist darauf hin, dass der verwendete Block einen bestimmten Eingabedatentyp erwartet, aber einen Eingabetyp erhalten hat, den er nicht verwenden kann. Er ähnelt dem Fehler "Erwartet Argumenttyp(en), wurde jedoch abgerufen mit". Er enthält in der Regel zusätzliche Informationen, mit deren Hilfe Sie ermitteln können, welche Argumenttypen Sie in den verwendeten Block übertragen müssen.

So beheben Sie Fehler

Überprüfen Sie generell, welche Eingabetypen für den verwendeten Block erwartet werden, und versuchen Sie zu ermitteln, ob (a) der richtige Eingabetyp an den Block übergeben wird, oder (b) Sie den Eingabetyp in einen Wert ändern können, den der Block verwenden kann (z. B. indem Sie die Eingabe in einen anderen Typ ändern, wie in diesem Beispiel gezeigt wird).

Wenn Sie beispielsweise einen ähnlichen Vorgang sehen wie Warning: Element.SetParameterByName operation failed. The best overloaded method match for ‘Revit.Elements.InternalUtilities.ElementUtils.SetParameterValue(Autodesk.Revit.DB.Parameter, double)’ has some invalid arguments, können Ihnen die zusätzlichen Informationen in Klammern dabei helfen, das Problem zu verstehen. In diesem Fall zeigt SetParameterValue(Autodesk.Revit.DB.Parameter, double) an, dass der verwendete Block eine Eingabe des Typs double erwartet. Damit ist klar, dass kein anderer Eingabetyp funktioniert.
