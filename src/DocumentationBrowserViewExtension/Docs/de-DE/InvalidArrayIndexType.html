Listenindizes müssen numerisch sein

Diese Fehlermeldung weist darauf hin, dass Sie versuchen, auf ein Element in einer Liste zuzugreifen, indem Sie einen Indexwert verwenden, der keine Ganzzahl ist.

Beginnend mit Dynamo 2.0 sind Listen Sammlungen von Werten, bei denen auf jeden Wert in der Liste nur mit ganzzahligen Indexwerten zugegriffen werden kann (Indexwerte von 0 bis Listengröße minus 1).

Weitere Informationen zu den Unterschieden zwischen Dynamo 1.X und Dynamo 2.X (einschließlich der Unterschiede beim Zugriff auf Elemente in einer Liste) finden Sie hier.

So beheben Sie Fehler

Stellen Sie sicher, dass der Indexwert, den Sie zum Zugriff auf Listenelemente verwenden, ausschließlich eine Ganzzahl ist (z. B. 2 und nicht "2" oder 2,0).

Weitere Informationen

Weitere Informationen finden Sie in den Beiträgen im DynamoBIM-Forum zum Thema list indices must be numeric.
