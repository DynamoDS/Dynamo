Zwischen zwei Variablen besteht eine zyklische Abhängigkeit

Diese Warnmeldungen gibt an, dass Sie versuchen, die Ausgabe eines Blocks an die Eingabe desselben Blocks zu übergeben. Dadurch wird eine zyklische Abhängigkeit erstellt, die in Dynamo nicht zulässig ist.

So beheben Sie Fehler

Stellen Sie sicher, dass Sie die Ausgabe eines Blocks nicht wieder in die Eingabe desselben Blocks übergeben. Wenn Sie eine Schleife erstellen müssen, können Sie den LoopWhile-Block verwenden, eine Schleife mithilfe von DesignScript erstellen oder eine Schleife mithilfe von Python erstellen, wie in den folgenden Beispielen dargestellt.
