Methode nicht gefunden

Diese Fehlermeldung weist darauf hin, dass Dynamo keine Definition für die DesignScript-Methode finden kann, die Sie verwenden möchten.

So beheben Sie Fehler

Wenn Sie eine benutzerdefinierte Methode in einem Codeblock definieren, stellen Sie sicher, dass Ihr Code ein gültiges DesignScript ist. Stellen Sie sicher, dass der in Ihrem CodeBlock definierte Methodenname mit dem Namen der Methode übereinstimmt, die Sie aufrufen. Weitere Informationen finden Sie in der DesignScript-Anleitung.
