Es ist nichts ausgewählt

Dieser Fehler weist darauf hin, dass Sie in einem Auswahlblock keine Auswahl getroffen haben (z. B. Select Model Element aus Dynamo for Revit).

So beheben Sie Fehler

Klicken Sie auf den Block, den Sie verwenden, und nehmen Sie in Revit eine Elementauswahl vor. Dazu müssen Sie vom Dynamo-Fenster in das Revit-Fenster (oder ein anderes Fenster der Hostumgebung) wechseln.

Weitere Informationen

Weitere Informationen finden Sie in den Beiträgen im DynamoBIM-Forum zum Thema nothing is selected.
