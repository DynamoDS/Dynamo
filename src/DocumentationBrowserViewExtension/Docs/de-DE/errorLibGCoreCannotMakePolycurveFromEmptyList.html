PolyCurve kann nicht aus einer leeren Liste erstellt werden

Diese Warnung wird angezeigt, wenn Sie beim Erstellen von PolyCurve eine leere Liste eingeben.

So beheben Sie Fehler

Der Fehler kann durch andere Blöcke in Ihrem Diagramm verursacht werden. Überprüfen Sie die vorgelagerten Blöcke, um zu sehen, ob sie korrekt ausgeführt werden. Es kommt oft vor, dass die vorgelagerten Blöcke leere Listen an einen Block übergeben, der dann diese Fehlermeldung anzeigt. Beginnen Sie mit dem Block, der diese Warnung verursacht, und arbeiten Sie rückwärts, indem Sie die Ausgabe jedes Blocks überprüfen, bis Sie den Block finden, der dieses Problem verursacht.
