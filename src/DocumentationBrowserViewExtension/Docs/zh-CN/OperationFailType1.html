操作失败

Operation failed 是一条常规错误消息，该消息指示其中一个节点无法执行。 它通常后跟附加信息，该信息涉及哪个操作失败以及失败的原因。 失败的一些最常见操作包括:

  • IronPythonEvaluator.EvaluateIronPythonScript - 当运行 python 代码的节点无法正确执行时，将显示此消息。 出现此情况有许多原因，但导致发生此情况的最常见原因是 python 代码包含错误。 此错误消息经常显示，其中包含有关 python 代码行无法执行以及出现此情况的原因的附加信息。 在此处阅读有关此错误消息的更多信息。
  • Element.SetParameterByName - 在此处阅读有关此错误消息的更多信息。
  • Floor.ByOutlineTypeAndLevel - 在此处阅读有关此错误消息的更多信息。
  • Dictionary.ByKeysValues - 在此处阅读有关此错误消息的更多信息。
  • List.GetItemAtIndex - 在此处阅读有关此错误消息的更多信息。
