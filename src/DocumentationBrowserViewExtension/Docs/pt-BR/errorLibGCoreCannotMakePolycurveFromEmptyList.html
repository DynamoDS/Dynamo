Não é possível criar uma PolyCurve com base em uma lista vazia

Esse aviso será exibido se você estiver inserindo uma lista vazia ao tentar criar uma PolyCurve.

Como corrigir

O erro pode ser causado por outros nós no gráfico. Verifique os nós a montante para ver se eles são executados corretamente – é comum que os nós a montante passem listas vazias a jusante até um nó que, em seguida, exibe essa mensagem de erro. Comece no nó que está causando esse aviso e verifique a saída de cada nó na direção inversa até encontrar o nó que está causando esse problema.

Mais informações
