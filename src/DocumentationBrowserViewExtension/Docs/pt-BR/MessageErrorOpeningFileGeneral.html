Erro ao abrir o arquivo

Essa é uma mensagem de erro genérica que indica que o arquivo com o qual você está trabalhando não pode ser aberto pela sua versão do Dynamo.

Como corrigir

O arquivo que você está tentando abrir foi criado usando a mesma versão ou uma versão anterior do Dynamo que a versão que está sendo executada no computador (por exemplo, se o arquivo tiver sido criado no Dynamo 1.X, será possível abri-lo com o Dynamo 1.X ou 2.X, mas se o arquivo tiver sidoi criado no Dynamo 2.X, só será possível abri-lo com o Dynamo  2.X ou superior).

Mais informações
