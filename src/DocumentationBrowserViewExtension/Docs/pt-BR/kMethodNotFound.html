Método não encontrado

Essa mensagem de erro significa que o Dynamo não pode localizar uma definição para o método DesignScript que você está tentando usar.

Como corrigir

Se você estiver definindo um método personalizado em um bloco de código, assegure-se de que o código seja DesignScript válido. Assegure-se de que o nome do método definido no CodeBlock coincide com o nome do método que você está chamando. Para obter mais detalhes, verifique o Guia do DesignScript.

Mais informações

Para saber mais, consulte as publicações no fórum Dynamobim que abordam “method not found”.
