Os índices da lista devem ser numéricos

Essa mensagem de erro indica que você está tentando acessar um elemento em uma lista usando um valor de índice que não é um número inteiro.

A partir do Dynamo 2.0, as listas são coleções de valores nas quais cada valor na lista pode ser acessado usando somente valores de índice inteiros (os valores de índice variam entre 0 e o tamanho da lista menos 1).

Leia mais sobre as diferenças entre o Dynamo 1.X e o Dynamo 2.X (incluindo como acessar elementos em uma lista foi alterado) aqui.

Como corrigir

Assegure-se de que o valor de índice usado para acessar os elementos da lista seja estritamente um inteiro (por exemplo, 2 e não “2” ou 2,0).

Mais informações

Para saber mais, consulte as publicações no fórum Dynamobim que abordam “list indices must be numeric”.
