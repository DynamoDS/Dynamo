Лофтинг не удался

Это предупреждение означает попытку выполнить действие, недопустимое для лофтинга. В большинстве случаев за этим предупреждением следует более конкретное сообщение с разъяснением причин проблемы.

Устранение проблемы

Приведем некоторые из основных причин появления этого предупреждения.

  • Вы используете лофтинг для создания формы между одним и двумя профилями.
  • Профили не являются замкнутыми кривыми.
