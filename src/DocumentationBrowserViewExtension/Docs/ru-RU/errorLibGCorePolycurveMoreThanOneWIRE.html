Соединение кривых дало в результате более одного провода в PolyCurve

Это сообщение об ошибке указывает, что линии, выводимые узлом Polycurve, не образуют замкнутую кривую.

Устранение проблемы

Это может быть связано с тем, что из узла Polycurve выводятся пустые значения. Проверьте в каждом списке, который выводится узлом Polycurve, соединяются ли кривые в одну непрерывную линию. Один из способов сделать это — скрыть предварительное изображение геометрии всех узлов (для этого щелкните узел или набор узлов правой кнопкой мыши), а затем запустить узел List.GetItemAtIndex отдельно для каждого списка.

Кроме того, эта ошибка часто возникает при работе с границами помещений. В этом случае для извлечения кривых попробуйте использовать узел Room.CenterBoundary.
