Найдено несколько определений

Это сообщение об ошибке означает, что в DynamoCore и других пакетах Dynamo, загружаемых для графика, используется одинаковое имя метода или класса, в результате чего Dynamo не может определить, какой метод или класс (т. е. из какого пакета) следует использовать.

Это предупреждение обычно появляется при использовании DesignScript в узле CodeBlock. Например, при попытке выровнять список в блоке кода используется команда List.Flatten(). Если в загружаемом пакете стороннего разработчика также имеется класс с именем List, появится это предупреждение.

Устранение проблемы

Для устранения этой ошибки используйте полное пространство имен используемой функции. Например, если это сообщение об ошибке появляется при попытке запустить функцию List.Flatten(), попробуйте вместо нее использовать функцию DSCore.List.Flatten() (DSCore расшифровывается как Design Script Core — пакет, поставляемый с Dynamo по умолчанию).
