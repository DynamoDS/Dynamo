自訂節點未載入

此錯誤訊息表示您的自訂節點尚未載入。自訂節點通常來自特定協力廠商套件。如果未在電腦上安裝套件，Dynamo 就無法找出並載入來自該套件的節點。請在這裡瞭解更多關於自訂節點的資訊。

如何修正

若要修正此問題，請確定在電腦上安裝包含您嘗試使用之自訂節點的套件。您可以從 Package Manager 安裝協力廠商套件。
