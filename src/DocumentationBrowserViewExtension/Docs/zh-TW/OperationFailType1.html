作業失敗

Operation failed 是一般錯誤訊息，表示您的其中一個節點無法執行。 後面通常會接哪個作業失敗，以及失敗發生原因的相關資訊。最常失敗的作業包括:

  • IronPythonEvaluator.EvaluateIronPythonScript - 執行 python 程式碼的節點無法正確執行時，會顯示此訊息。 原因很多，但最常發生的原因是 python 程式碼有錯誤。此錯誤訊息常會顯示無法執行的 python 程式碼行，以及發生原因的相關資訊。請在這裡閱讀更多關於此錯誤訊息的資訊。
  • Element.SetParameterByName - 在這裡閱讀更多關於此錯誤訊息的資訊。
  • Floor.ByOutlineTypeAndLevel - 在這裡閱讀更多關於此錯誤訊息的資訊。
  • Dictionary.ByKeysValues - 在這裡閱讀更多關於此錯誤訊息的資訊。
  • List.GetItemAtIndex - 在這裡閱讀更多關於此錯誤訊息的資訊。
