Hay una dependencia cíclica entre dos variables

Este mensaje de advertencia indica que está intentando transferir la salida de un nodo a la entrada de ese mismo nodo. Esto creará una dependencia cíclica que no se permite en Dynamo.

Cómo solucionar este error

Asegúrese de que no está transfiriendo la salida de un nodo a la entrada de ese mismo nodo. Si necesita crear un bucle, puede utilizar el nodo LoopWhile, o bien puede crear un bucle mediante DesignScript o Python, como se muestra en estos ejemplos.
