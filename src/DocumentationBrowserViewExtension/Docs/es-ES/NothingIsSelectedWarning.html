No se ha seleccionado nada

Este error indica que no se ha seleccionado nada en un nodo de selección (por ejemplo, Select Model Element de Dynamo for Revit).

Cómo solucionar este error

Haga clic en Seleccionar en el nodo que está utilizando y realice una selección de elementos en Revit. Para ello, debe pasar de la ventana de Dynamo a la de Revit (u otro entorno de anfitrión).

Más información

Para obtener más información, consulte las publicaciones del foro de Dynamobim sobre "nothing is selected".
