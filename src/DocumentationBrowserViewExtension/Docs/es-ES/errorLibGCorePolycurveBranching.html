Es posible que se bifurquen (o no) las PolyCurves

Este mensaje de error indica que el nodo de geometría que se está utilizando elimina los valores situados después del decimal en una configuración muy grande. Por ejemplo, los puntos que anteriormente tenían los valores X [0.1, 0.2, 0.3, 5, 10] ahora presentan los valores [0, 0, 0, 5, 10].

Cómo solucionar este error

Puede probar a utilizar Geometry.Scale para cambiar los parámetros de escala de la geometría. "Medium" (Medio) es el parámetro por defecto para la escala de la geometría.
