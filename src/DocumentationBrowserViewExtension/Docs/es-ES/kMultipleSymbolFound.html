Se han encontrado varias definiciones

Este mensaje de error indica que se utiliza el mismo método o nombre de clase en DynamoCore y se han cargado diferentes paquetes de Dynamo para el gráfico, y Dynamo no sabe qué método o clase (es decir, desde qué paquete) se debe utilizar.

Esta advertencia suele producirse cuando se utiliza DesignScript en un bloque de código. Por ejemplo, si intenta aplanar una lista en un bloque de código, puede utilizar List.Flatten(). Si uno de los paquetes de terceros que está cargando también tiene una clase denominada List, aparecerá esta advertencia.

Cómo solucionar este error

Para solucionar este error, incluya el espacio de nombres completo de la función que está intentando utilizar. Por ejemplo, si está ejecutando List.Flatten() y aparece este mensaje de error, pruebe a utilizar DSCore.List.Flatten() en su lugar (DSCore equivale a "DesignScript Core" y es uno de los paquetes por defecto que se suministra con Dynamo).
