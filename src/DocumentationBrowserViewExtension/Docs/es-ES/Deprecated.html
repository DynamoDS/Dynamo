Método o nodo obsoletos

El mensaje de advertencia de elemento obsoleto significa que los autores de Dynamo desean eliminar este método o nodo en el futuro y debe intentar utilizar métodos o nodos diferentes para obtener la misma función. Esta advertencia permite notificar a los usuarios de Dynamo que el método se eliminará en el futuro sin suprimirlo al instante para que tengan tiempo para actualizar los gráficos y los modelos.

Cómo solucionar este error

Pruebe a utilizar diferentes nodos para obtener la misma función. Este mensaje de error suele ir seguido de sugerencias de los métodos que aún están disponibles para usarlos en su lugar. Tenga en cuenta que es recomendable no utilizar nodos o métodos obsoletos.
