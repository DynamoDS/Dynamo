Error al abrir el archivo

Este es un mensaje de error genérico que indica que la versión de Dynamo no puede abrir el archivo con el que está trabajando.

Cómo solucionar este error

Asegúrese de que el archivo que intenta abrir se haya creado con la misma versión de Dynamo (o una anterior) que la versión que está ejecutando en el equipo (por ejemplo, si el archivo se ha creado en Dynamo 1.X, puede abrirlo con Dynamo 1.X o 2.X, pero si el archivo se ha creado en Dynamo 2.X, solo puede abrirlo con Dynamo 2.X o superior).
