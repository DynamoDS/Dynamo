Error en la operación

Operation failed es un mensaje de error genérico que indica que uno de los nodos no se ha podido ejecutar. Por lo general, después de este mensaje, se proporciona información adicional sobre la operación que ha presentado errores y el motivo del fallo. Algunas de las operaciones más frecuentes que presentan errores son:

  • IronPythonEvaluator.EvaluateIronPythonScript: este mensaje se muestra cuando un nodo que ejecuta el código de Python no se ejecuta correctamente. Puede deberse a muchos motivos, pero el más habitual es que el código de Python contiene un error. Este mensaje de error se muestra con frecuencia con información adicional sobre la línea del código de Python que no se pudo ejecutar y el motivo. Obtenga más información acerca de este mensaje de error aquí.
  • Element.SetParameterByName: obtenga más información acerca de este mensaje de error aquí.
  • Floor.ByOutlineTypeAndLevel: obtenga más información sobre este mensaje de error aquí.
  • Dictionary.ByKeysValues: obtenga más información sobre este mensaje de error aquí.
  • List.GetItemAtIndex: obtenga más información sobre este mensaje de error aquí.
