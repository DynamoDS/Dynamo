Exception non gérée

Ce message d'erreur indique généralement la présence d'un bogue dans le code Dynamo qui provoque le blocage de Dynamo. Ce n'est pas nécessairement lié à votre graphique (ou à l'un des packages que vous utilisez). Signaler un bogue peut aider les développeurs Dynamo à résoudre ces problèmes dans une version ultérieure.

Comment corriger cette erreur?

L'enregistrement de votre travail, le redémarrage de Dynamo et le nouveau chargement de votre fichier peuvent vous permettre de récupérer le système suite à ces erreurs. En outre, vous pouvez essayer de mettre à jour Dynamo et les packages Dynamo que vous utilisez sur votre ordinateur vers leurs dernières versions, qui peuvent déjà inclure des corrections pour les problèmes auxquels vous êtes confrontés. Si vous pouvez identifier le package à l'origine de ces problèmes, vous pouvez également contacter directement l'auteur du package pour plus d'informations.

Plus d'informations

Pour en savoir plus, consultez les billets publiés sur le forum Dynamobim sur le sujet "unhandled exception" (exception non gérée).
