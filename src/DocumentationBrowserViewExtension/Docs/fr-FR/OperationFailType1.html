Échec de l'opération

Échec de l'opération est un message d'erreur générique qui indique que l'un de vos noeuds n'a pas pu être exécuté. Il est généralement suivi d'informations supplémentaires identifiant l'opération qui a échoué et la raison de l'échec. Voici quelques-unes des opérations les plus courantes qui échouent:

  • IronPythonEvaluator.EvaluateIronPythonScript: ce message s'affiche lorsqu'un noeud exécutant votre code python ne s'est pas exécuté correctement. Il existe de nombreuses raisons à cela, mais cela se produit le plus souvent lorsque le code python contient un bogue. Ce message d'erreur s'affiche fréquemment avec des informations supplémentaires concernant la ligne du code python qui n'a pas pu être exécutée et la raison pour laquelle cela s'est produit. En savoir plus sur ce message d'erreur ici.
  • Element.SetParameterByName: en savoir plus sur ce message d'erreur ici.
  • Floor.ByOutlineTypeAndLevel: en savoir plus sur ce message d'erreur ici.
  • Dictionary.ByKeysValues: en savoir plus sur ce message d'erreur ici.
  • List.GetItemAtIndex: en savoir plus sur ce message d'erreur ici.
