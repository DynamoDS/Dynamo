이 오류 메시지는 사용 중인 노드가 특정 유형의 입력을 예상하지만 사용할 수 없는 입력 유형을 받았음을 나타냅니다. 일반적으로 이 뒤에는 사용 중인 노드로 전달된 유형 및 노드에서 받을 것으로 예상되는 유형에 관한 추가 정보가 표시됩니다. 이 오류를 트리거하는 가장 일반적인 작업 몇 가지는 다음과 같습니다.

  • Element.SetParameterByName - Element.Parameters 노드를 사용하여 요소의 매개변수를 확인하려고 합니다. "expects argument type setparameter"에 대해 논의하는 Dynamobim 포럼의 게시물을 더 읽어보십시오.
  • Geometry.BoundingBox - 여기에서 논의된 것처럼 Element.BoundingBox를 대신 사용하려고 합니다. "expects argument type geometry"에 대해 논의하는 Dynamobim 포럼의 게시물을 더 읽어보십시오.
