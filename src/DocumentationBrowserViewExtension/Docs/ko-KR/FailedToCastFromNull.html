null 값은 유형에 투영할 수 없음

이 경고는 사용하려는 노드가 특정 데이터 유형의 입력을 기대하고 있으며 해당 유형이 null일 수 없음을 의미합니다. null은 단순히 "객체 없음"을 의미하므로 노드에서 노드에 객체를 입력하지 않는다는 경고를 표시합니다(이는 허용되지 않음).

수정 방법

이 오류는 그래프의 다른 노드로 인해 발생할 수 있습니다. 업스트림 노드가 제대로 실행되는지 확인하십시오. 업스트림 노드가 제대로 실행되지 않고 null 값을 노드에 다운스트림으로 전달하여 이 오류 메시지가 표시되는 경우가 자주 있습니다. 이 경고를 유발하는 노드부터 시작하여 거슬러 올라가며 이 문제를 일으키는 노드를 찾을 때까지 각 노드의 출력을 확인하십시오.

오류 메시지에 null인 매개변수가 표시되는 경우 해당 매개변수의 업스트림 경로에서 노드가 제대로 실행되고 null이 아닌 값을 다운스트림으로 전달하는지 여부를 확인하십시오.
