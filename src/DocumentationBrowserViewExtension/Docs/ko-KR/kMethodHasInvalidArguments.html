일부 인수가 잘못됨

이 오류 메시지는 사용 중인 노드가 특정 유형의 입력 데이터를 예상하고 있지만 사용할 수 없는 입력 유형을 받았음을 나타냅니다. 이는 "인수 유형을 예상하지만 다음으로 호출됨" 오류와 유사하며, 일반적으로 사용 중인 노드로 전달해야 하는 인수 유형을 식별하는 데 도움이 될 수 있는 추가 정보를 표시합니다.

수정 방법

일반적으로, 사용 중인 노드에서 예상하는 입력 유형을 확인하고 (a) 올바른 유형의 입력이 노드에 전달되는지 또는 (b) 입력 유형을 노드에서 사용할 수 있는 유형으로 변경(예: 이 예에 나와 있듯이 다른 유형으로 입력 변경)할 수 있는지 확인하십시오.

예를 들어, Warning: Element.SetParameterByName operation failed. The best overloaded method match for ‘Revit.Elements.InternalUtilities.ElementUtils.SetParameterValue(Autodesk.Revit.DB.Parameter, double)’ has some invalid arguments와 같은 메시지가 표시되는 경우 괄호 안의 추가 정보는 문제가 무엇인지 이해하는 데 도움이 될 수 있습니다. 이 경우 SetParameterValue(Autodesk.Revit.DB.Parameter, double)는 사용 중인 노드가 double 유형의 입력을 예상함을 나타내므로 다른 모든 유형의 입력이 작동하지 않습니다.
