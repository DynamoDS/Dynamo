Polycurves는 분기할 수 있음(없음)

이 오류 메시지는 사용 중인 형상 노드가 아주 큰 설정에서 소수점 이하의 값을 없애고 있음을 나타냅니다. 예를 들어 이전에는 [0.1, 0.2, 0.3, 5, 10]의 X 값이 있었던 점에 [0, 0, 0, 5, 10]의 값이 있습니다.

수정 방법

Geometry.Scale을 사용하여 형상 축척 설정을 변경할 수 있습니다. 중간은 형상 축척의 기본 설정입니다.
