리스트 색인은 숫자여야 함

이 오류 메시지는 정수(예: 범자연수)가 아닌 색인 값을 사용하여 리스트의 요소에 액세스하려고 함을 나타냅니다.

Dynamo 2.0부터 리스트는 값의 집합으로, 각 값은 정수 색인 값(색인 값의 범위는 0부터 리스트 크기-1까지의 값)을 사용해서만 액세스할 수 있습니다.

Dynamo 1.X와 Dynamo 2.X 간의 차이점(리스트 요소에 액세스하는 방식이 어떻게 변경되었는지 포함)에 대해 더 읽어 보십시오.

수정 방법

리스트 요소에 액세스하는 데 사용하는 색인 값이 정확히 정수(예: 2, "2" 또는 2.0 아님)여야 합니다.
