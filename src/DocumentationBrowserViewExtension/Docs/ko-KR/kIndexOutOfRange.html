색인이 범위를 벗어남

이 오류 메시지는 사용자가 제공한 색인 위치의 요소가 리스트에 포함되어 있지 않음을 의미합니다. 예를 들어, 리스트가 [a, b, c, d](색인 0, 1, 2, 3에 요소가 있음)인데 색인 5의 요소에 액세스하려고 하면 0부터 3까지 색인에 있는 요소만 리스트에 있으므로 오류 메시지가 표시됩니다.

수정 방법

리스트의 요소에 액세스하는 데 사용 중인 색인 변수가 0보다 크거나 같고 정확히 리스트 크기보다 작은지 확인하십시오. 리스트 색인의 범위는 0에서 리스트 크기-1까지의 값입니다. 즉, 리스트에 10개의 요소가 있는데 10번째 요소에 액세스하려는 경우 해당 요소는 색인 9에 있게 됩니다.
