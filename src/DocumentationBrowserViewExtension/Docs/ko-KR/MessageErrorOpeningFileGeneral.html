파일을 여는 중에 오류가 발생함

이는 사용 중인 파일을 Dynamo 버전에서 열 수 없음을 나타내는 일반 오류 메시지입니다.

수정 방법

열려는 파일이 컴퓨터에서 실행 중인 버전과 같거나 이전 버전의 Dynamo를 사용하여 작성되었는지 확인합니다(예: 파일이 Dynamo 1.X에서 작성된 경우 Dynamo 1.X 또는 2.X에서 해당 파일을 열 수 있지만, Dynamo 2.X에서 파일을 작성한 경우에는 Dynamo 2.X 이상 버전에서만 파일을 열 수 있음).
