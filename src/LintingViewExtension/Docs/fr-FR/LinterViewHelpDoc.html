État du graphique

Présentation

L'extension État du graphique permet de trouver des informations sur les problèmes que Dynamo a détectés dans votre graphique. Les problèmes qui sont affichés dans le groupe de fonctions État du graphique sont distincts des avertissements et des erreurs qui sont affichés sur les noeuds Dynamo individuels dans l'espace de travail. Ceux-ci sont dérivés de règles personnalisées qui peuvent être définies par des extensions Dynamo.

Type de graphique

Lorsqu'une extension Dynamo utilise l'API Linting de Dynamo pour enregistrer un jeu de règles personnalisé, le nom de cette extension apparaît sous la forme d'une option dans la liste déroulante intitulée Type de graphique. Dynamo vérifie si ces règles ont été enfreintes lorsque des modifications sont apportées au graphique.

Par exemple, l'extension Generative Design pour Dynamo utilise cette fonction pour s'assurer que les graphiques Dynamo conçus pour être utilisés avec Generative Design possèdent les propriétés nécessaires pour que le graphique fonctionne correctement dans l'application Generative Design. Generative Design nécessite que les graphiques disposent d'au moins un noeud d'entrée et un noeud de sortie, et cette opération est appliquée à l'aide des règles enregistrées par l'extension Dynamo.

Problèmes de noeud et de graphique

Les règles définies par une extension sont enregistrées sous l'un des deux types suivants: règles de graphique ou règles de noeud. Si Dynamo détecte que l'une de ces règles a été endommagée dans le graphique en cours, un avertissement contenant des informations sur la manière de résoudre le problème et un lien vers l'emplacement où le problème a été détecté s'affichent dans l'une de ces deux sections.

Autres informations

Lorsqu'un graphique est enregistré, l'extension État du graphique stocke certaines métadonnées dans le fichier Dynamo, telles que le type de graphique sélectionné, et indique si des problèmes sont survenus lors du dernier enregistrement du graphique.

Lorsqu'un nouveau graphique Dynamo est créé, l'extension État du graphique ne vérifie pas automatiquement la présence de règles spécifiques. La sélection par défaut pour le type de graphique est Aucun. Actuellement, un seul type de graphique peut être sélectionné à la fois dans la vue État du graphique.
