Статус графика

Обзор

В расширении «Статус графика» представлены сведения о проблемах, обнаруженных в Dynamo на графике. Проблемы, отображаемые на панели «Статус графика», отделены от предупреждений и ошибок, отображаемых в отдельных узлах Dynamo в рабочем пространстве. Они в свою очередь создаются на основе пользовательских правил, которые можно определить с помощью расширений Dynamo.

Тип графика

Если в расширениях Dynamo используется API-интерфейс Linting Dynamo для регистрации пользовательского набора правил, имя этого расширения будет отображаться в раскрывающемся списке с меткой Тип графика. В Dynamo выполняется проверка, не нарушены ли эти правила при внесении изменений в график.

Например, в расширении «Генеративное проектирование» для Dynamo эта функция используется, чтобы обеспечить в графиках Dynamo, разработанных для использования с модулем генеративного проектирования, свойств, необходимых для правильной работы графика в приложении «Генеративное проектирование». В модуле «Генеративное проектирование» требуется, чтобы в графиках имелся хотя бы один входной и один выходной узел; это условие выполняется с помощью правил, зарегистрированных в расширении Dynamo.

Проблемы с узлами и графиками

Правила, определяемые расширением, регистрируются как два типа: правила для графиков и правила для узлов. Если в Dynamo обнаруживается, что одно из этих правил было нарушено на текущем графике, в одном из этих разделов выводится предупреждение с информацией о том, как устранить проблему, и ссылкой на место, где была обнаружена проблема.

Прочие сведения

Каждый раз при сохранении графика расширение «Статус графика» сохраняет в файл Dynamo некоторые метаданные, например выбранный тип графика, а также сведения о наличии или отсутствии проблем при последнем сохранении графика.

При создании нового графика Dynamo расширение «Статус графика» не выполняет автоматическую проверку конкретных правил. По умолчанию для параметра «Тип графика» выбрано значение Нет. В настоящее время в виде «Статус графика» одновременно можно выбрать только один тип графика.
