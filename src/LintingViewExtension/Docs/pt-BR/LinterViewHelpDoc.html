Status do gráfico

Visão geral

A extensão Status do gráfico é onde você pode encontrar informações sobre problemas que o Dynamo detectou em seu gráfico. Os problemas que são mostrados no painel Status do gráfico são separados dos avisos e dos erros que são exibidos nos nós individuais do Dynamo no espaço de trabalho. Esses são decorrentes das regras personalizadas que podem ser definidas por extensões do Dynamo.

Tipo de gráfico

Quando as extensões do Dynamo usam a API de lint do Dynamo para registrar um conjunto de regras personalizadas, o nome dessa extensão aparece como uma opção no menu suspenso denominado Tipo de gráfico. Sempre que são feitas alterações no gráfico, o Dynamo verifica se essas regras foram cumpridas.

Por exemplo, a extensão do Projeto generativo do Dynamo usa esse recurso para garantir que os gráficos do Dynamo projetados para uso com o Projeto generativo tenham as propriedades necessárias exigidas para que o gráfico funcione corretamente no aplicativo do Projeto generativo. O Projeto generativo requer que os gráficos tenham pelo menos um nó de entrada e um nó de saída, e isso é aplicado usando as regras registradas pela extensão do Dynamo.

Problemas de nó e de gráfico

As regras definidas por uma extensão são registradas como pertencentes a um de dois tipos: regras de gráfico ou regras de nó. Se o Dynamo detectar que um desses tipos de regras não foi cumprido no gráfico atual, um aviso com informações sobre como corrigir o problema e um link que encaminha para o local onde problema foi detectado serão mostrados em uma dessas duas seções.

Outras informações

Sempre que um gráfico é salvo, a extensão Status do gráfico armazena alguns metadados no arquivo do Dynamo, como o tipo de gráfico selecionado e se os problemas estavam ou não presentes quando o gráfico foi salvo pela última vez.

Quando um novo gráfico do Dynamo é criado, a extensão Status do gráfico não verifica automaticamente se existem regras específicas. A seleção padrão para o Tipo de gráfico é Nenhum. No momento, somente um tipo de gráfico pode ser selecionado de cada vez na vista Status do gráfico.
