How to access Node Autocomplete

Node autocomplete can be enabled from the Experimental section of the Preference menu.
It can be accessed on any input or output port by Alt + clicking it, and a popup window will appear next to the target port.

How the ranking mechanism works

Nodes that are closest to the type (Have the smallest type distance) and match the input/output port type appear higher in the ranking than those nodes that have a are further away from the type (Have a greater type distance). After this initial ranking, nodes are then sorted by Group (Such as Curve, List or Math) and then by their returned value (Output parameters) before being ranked by their names alphabetically.

How the auto-layout mechanism works

The node auto-layout mechanism takes into consideration the entire node graph, paying particular attention to the local context, the parent node and the margins between them before placing the new node. You may notice that your graph changes minimally with every new node placed through the Node Autocomplete feature.

How the auto-wire mechanism works

The auto-wiring mechanism will automatically attach the input/output port of the new node to the target output/input port and place it on teh workspace.

What nodes are hardcoded in and out of results

For nodes that do not match any criteria, such as some primitive type input/output ports, Node Autocomplete will show a fixed list of suggestions that contain: String, Number Slider, Integer Slider, Number and Boolean.
